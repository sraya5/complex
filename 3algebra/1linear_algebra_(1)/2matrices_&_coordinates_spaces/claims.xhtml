    1.2 כפל מטריצה בווקטור

    טענה 1.1. לכל A ∈ M m × n ( F ) מתקיים A ⋅ 0 → = 0 → .
    טענה 1.2. תהא A ∈ M m × n ( F ) ויהי v ∈ F n , אם השורה ה- i של A היא שורת אפסים אז A ⋅ v הוא וקטור שבקואורדינטה ה- i שלו יופיע 0 (עבור m ≥ i ∈ N ).
    טענה 1.3. לכל A ∈ M m × n ( F ) העמודה ה- j של A היא A ⋅ e j (עבור n ≥ j ∈ N ).
    1. ♣ כלומר אנחנו יודעים בדיוק לאן מעתיקה T A כל אחד מהווקטורים בבסיס הסטנדרטי, תכף נראה מדוע זה חשוב.
    ‫מסקנה 1.4. תהיינה A , B ∈ M m × n ( F ) , מתקיים A = B אם”ם לכל v ∈ F n מתקיים T A ( v ) = A ⋅ v = B ⋅ v = T B ( v ) (כלומר אם”ם T A = T B ).
    משפט 1.5: תכונות של כפל מטריצה בווקטור
    יהיו A , B ∈ M m × n ( F ) , x , y ∈ F n ו- c ∈ F ; מתקיימים שלושת הפסוקים הבאים:
    • פילוג (דיסטריבוטיביות) ביחס לחיבור וקטורי - T A ( x + y ) = A ⋅ ( x + y ) = A ⋅ x + A ⋅ y = T A ( x ) + T A ( y ) .
    • פילוג (דיסטריבוטיביות) ביחס לחיבור מטריצות - T A + B ( x ) = ( A + B ) ⋅ x = A ⋅ x + B ⋅ x = T A ( x ) + T B ( x ) .
    • קיבוץ וחילוף ביחס לכפל בסקלר
      1
      לא כתבנו את הביטוי ( A ⋅ c ) ⋅ x מפני שהכפל בסקלר מוגדר משמאל בלבד.
      - T A ( c ⋅ x ) = A ⋅ ( c ⋅ x ) = ( c ⋅ A ) ⋅ x = c ⋅ ( A ⋅ x ) = c ⋅ T A ( x ) .
    1. ♣ שתי התכונות הללו אומרות שכפל מטריצה בווקטור שומר על המבנה של F n כמרחב קואורדינטות עם פעולות החיבור הווקטורי והכפל בסקלר, מכאן שכפל מטריצה בווקטור מעתיק תתי-מרחבים של F n לתתי-מרחבים של F m .
    2. ♣ בגלל התכונה השנייה לא נטרח לכתוב סוגריים בביטויים כגון c ⋅ A ⋅ x או A ⋅ c ⋅ x .

    1. ♣ בפרט לכל x ∈ F n מתקיים: T A ( x ) = T A ( x 1 x 2 ⋮ x n ) = T A ( ∑ i = 1 n x i ⋅ e i ) = ∑ i = 1 n x i ⋅ T A ( e i ) = ∑ i = 1 n x i ⋅ c i כאשר c i היא העמודה ה- i של A . כלומר אם אנחנו יודעים כיצד T A פועלת על איברי הבסיס הסטנדרטי (וכפי שראינו אנחנו אכן יודעים זאת עבור A נתונה) אז אנחנו יודעים בדיוק כיצד פועלת הפונקציה T A על מרחב הקואורדינטות F n .

    1.3 כפל מטריצות

    יהי l ∈ N .
    טענה 1.7. תהיינה A ∈ M m × n ( F ) ו- B ∈ M l × m ( F ) , מתקיים:
    • אם העמודה ה- j של מטריצה A היא עמודת אפסים אז העמודה ה- j של B ⋅ A היא עמודת אפסים (לכל n ≥ j ∈ N ).
    • אם השורה ה- i של מטריצה B היא שורת אפסים אז השורה ה- i של B ⋅ A היא שורת אפסים (לכל l ≥ i ∈ N ).
    משפט 1.8: תהיינה A ∈ M m × n ( F ) ו- B ∈ M l × m ( F ) , לכל x ∈ F n מתקיים T B ( T A ( x ) ) = B ⋅ ( A ⋅ x ) = ( B ⋅ A ) ⋅ x = T B ⋅ A ( x ) .
    1. ♣ כלומר כפל מטריצות שקול להרכבת ההעתקות המוגדרות על ידן - מתקיים T B ○ T A = T B ⋅ A .
    משפט 1.9: תכונות של כפל מטריצות
    תהיינה A , A 0 ∈ M m × n ( F ) , B , B 0 ∈ M l × m ( F ) ו- C ∈ M k × l ( F ) ו- x ∈ F , מתקיים:
    • קיבוץ (אסוציאטיביות) - ( C ⋅ B ) ⋅ A = C ⋅ ( B ⋅ A ) .
    • פילוג (דיסטריבוטיביות) ביחס לחיבור מטריצות - B ⋅ ( A + A 0 ) = B ⋅ A + B ⋅ A 0 ו- ( B + B 0 ) ⋅ A = B ⋅ A + B 0 ⋅ A .
    • קיבוץ וחילוף ביחס לכפל בסקלר - B ⋅ ( x ⋅ A ) = ( x ⋅ B ) ⋅ A = x ⋅ ( B ⋅ A ) .
    1. ♣ בגלל התכונה הראשונה לא נטרח לכתוב סוגריים בביטוי כגון C ⋅ B ⋅ A .
    2. ♣ מכיוון שראינו את השקילות בין חיבור/כפל מטריצות לחיבור/הרכבת הפונקציות המוגדרות על ידן נוכל לנסח את המשפט בשפה של פונקציות:
      • קיבוץ (אסוציאטיביות) - בתכונה זו הכיוון הוא הפוך: אנחנו יודעים שכפל מטריצות מקיים חוק הקיבוץ מפני שהוא שקול להרכבת פונקציות וכל הרכבת פונקציות מקיימת קיבוץ.
      • פילוג (דיסטריבוטיביות) ביחס לחיבור פונקציות - T A ○ ( T B + T C ) = T A ○ T B + T A ○ T C .
      • קיבוץ וחילוף ביחס לכפל בסקלר - T A ○ ( x ⋅ T B ) = ( x ⋅ T A ) ○ T B = x ⋅ ( T A ○ T B ) .
    3. ♣ כפל מטריצות אינו מקיים חילוף, לדוגמה: [ 1 5 2 1 ] ⋅ [ 1 - 3 0 4 ] = [ 1 17 2 - 2 ] ≠ [ - 5 2 8 4 ] = [ 1 - 3 0 4 ] ⋅ [ 1 5 2 1 ]

    1. ♣ מסקנה זו היא הסיבה לכך שמטריצת היחידה נקראת גם מטריצת הזהות.
    טענה 2.4. תהא P ∈ M n ( F ) מטריצה הפיכה, גם T P הפיכה.
    ‫מסקנה 2.5. תהא P ∈ M n ( F ) מטריצה הפיכה, קיימת Q ∈ M n ( F ) יחידה כך ש- P ⋅ Q = I n = Q ⋅ P .

    ‫מסקנה 2.8. תהא A ∈ M n ( F ) , אם קיימת B ∈ M n ( F ) כך ש- A ⋅ B = I n ו/או B ⋅ A = I n אז A הפיכה ו- A - 1 = B (כלומר A ⋅ B = I n וגם B ⋅ A = I n ).
    טענה 2.9. תהיינה P , Q ∈ M n ( F ) מטריצות הפיכות, P ⋅ Q היא מטריצה הפיכה וההופכית שלה היא Q - 1 ⋅ P - 1 .
    1. ♣ ניתן להסיק מכאן שכל מכפלה של מטריצות הפיכות היא מטריצה הפיכה וההופכית שלה היא מכפלת המטריצות ההופכיות בסדר הפוך.

    5.3 דרגה של מטריצה

    טענה 5.12. דרגת השורות שווה לדרגת העמודות בכל מטריצה.
    טענה 5.13. תהיינה A , B ∈ M m × n ( F ) מטריצות שקולות שורה, מתקיים rk A = rk B .
    1. ♣ טענה זו מאפשרת לנו למצוא את הדרגה של מטריצה נתונה ע”י דירוג המטריצה וספירת האיברים המובילים במטריצה המדורגת-מצומצמת.
    ‫מסקנה 5.14. תהא A ∈ M n ( F ) מטריצה ריבועית, A הפיכה אם”ם rk A = n .
    ‫מסקנה 5.15. תהא P ∈ M n ( F ) מטריצה הפיכה, סדרת העמודות של P היא בסיס של F n וכמוה גם סדרת השורות של P היא בסיס של F n .
    טענה 5.16. יהי l ∈ N ותהיינה A ∈ M m × n ( F ) ו- B ∈ M l × m ( F ) , מתקיים rk ( B ⋅ A ) ≤ min { rk A , rk B } .
    ‫מסקנה 5.17. תהיינה A ∈ M m × n ( F ) ו- B ∈ M n × m ( F ) , אם n > m אז B ⋅ A ≠ I n .
    ‫מסקנה 5.18. תהיינה A , B ∈ M n ( F ) , אם A אינה הפיכה אז גם A ⋅ B ו- B ⋅ A אינן הפיכות.
    משפט 5.19: תהא A ∈ M m × n ( F ) מטריצה ותהיינה P ∈ M n ( F ) ו- Q ∈ M m ( F ) מטריצות הפיכות, מתקיים: rk ( A ⋅ P ) = rk A = rk ( Q ⋅ A ) כלומר כפל במטריצה הפיכה אינו משנה את דרגת המטריצה.
    משפט 5.20: הדרגה של מטריצה שווה לסדר הגדול ביותר של תת-מטריצה הפיכה שלה.
