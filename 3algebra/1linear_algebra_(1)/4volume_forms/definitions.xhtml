    1. ♣ תהא A ∈ M n ( F ) , הלמה שלעיל (לפני כלל קרמר) מעלה בנו את הרעיון להגדיר את הפונקציה הבאה - תהא f A : F n → F n פונקציה המוגדרת ע”י (לעכל b ∈ F n ): f A ( b ) = [ det A b ( 1 ) det A b ( 2 ) ⋮ det A b ( n ) ] כאשר A b ( i ) היא המטריצה המתקבלת מ- A ע”י החלפת העמודה ה- i ב- b (לכל b ∈ F n ולכל n ≥ i ∈ N ).
      ע”פ הלמה f A מעתיקה כל וקטור אל אחד המקורות שלו (אם קיים כזה) ביחס לפונקציה T A עד כדי כפל בקבוע det A , ולכן במובן מסוים היא הופכית של T A גם אם T A אינה הפיכה.
      מתכונות הדטרמיננטה נובע ש- f הנ”ל היא העתקה ליניארית ולכן קיימת מטריצה יחידה B ∈ M n ( F ) כך ש- f A = T B , אבל איזו מטריצה זו?

    2. ♣ לעיל ראינו ש- f A מחזירה כל וקטור למקורו (אם יש לו כזה) ביחס ל- T A כשהוא מוכפל בקבוע det A , מכאן שמתקיים ( adj A ) ⋅ A = ( det A ) ⋅ I n ולכן אם A הפיכה אז A - 1 = 1 det A ⋅ adj A .

    3. ♣ מהעובדה שהדטרמיננטה של מטריצה ושל המשוחלפת שלה שוות נקבל שמתקיים (לכל n ≥ i , j ): [ adj A ] = ( - 1 ) i + j ⋅ | A | = ( - 1 ) i + j ⋅ | ( A ) | בדרך זו קל יותר לחשב כל קואורדינטה במטריצה המצורפת.
