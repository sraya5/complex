1 דינמיקה של אופרטור

    נקרא להעתקה ליניארית ממ”ו V לעצמו ( f : V → V ) בשם אופרטור
    1
    במקומות אחרים קוראים לכל העתקה ליניארית בשם “אופרטור”, גם אם היא בין שני מרחבים וקטוריים שונים.
    , נסמן ב- End ( V ) את מרחב האופרטורים מ- V לעצמו
    2
    ראינו בקורס הקודם שמדובר במרחב וקטורי.
    .
    ‫הגדרה 1.1. מערכת ליניארית
    יהיו V מ”ו ו- f : V → V אופרטור על V , הזוג ( V , f ) נקרא מערכת ליניארית.
    תהא ( V , f ) מערכת ליניארית מעל לשדה F .
    סימון:
    נסמן f 0 : = Id V ולכל n ∈ N נסמן f n : = f ○ f n - 1 .
    1. ♣ בכל מקום אחר בפירוש של f n ( x ) הוא ( f ( x ) ) n , כלומר העלאה בחזקה ולא הרכבה, תכף נראה למה בקורס זה נוח לנו להגדיר אחרת.
    ‫הגדרה 1.2. הצבה של אופרטור/מטריצה בפולינום
    יהיו A ∈ M n ( F ) ו- P ( x ) : = ∑ i = 0 m a i ⋅ x i ∈ F [ x ] , נגדיר P ( f ) : = ∑ i = 0 m a i ⋅ f i ונקרא ל- P ( f ) ההצבה של f בפולינום P , כמו כן נגדיר P ( A ) : = ∑ i = 0 m a i ⋅ A i ונקרא ל- P ( A ) ההצבה של המטריצה A בפולינום P .
    1. ♣ נזכור שהוכחנו שהרכבה וסכום של העתקות ליניאריות וכן כפל של ה”ל בסקלר הם העתקות ליניאריות ולכן הצבה של אופרטור בפולינום היא אופרטור על אותו מרחב (ודאי שגם הצבה של מטריצה בפולינום היא מטריצה מאותו סדר גודל).
    2. ♣ בקורס הקודם סימנו מטריצה מייצגת של ה”ל בצורה [ f ] C B עבור בסיסים B של התחום ו- C של הטווח, בקורס הזה נתעסק בעיקר באופרטורים (תחום וטווח זהים) ובד”כ גם נרצה לייצג את המרחב באמצעות בסיס מסוים, לכן נסמן את [ f ] B B ע”י [ f ] B .
    3. ♣ הצבה של אופרטור בפולינום מדרגה 0 הוא תמיד כפולה של העתקת הזהות, פעמים רבות נסמן את λ ( f ) = λ ⋅ f 0 = λ ⋅ Id V (כאשר λ הוא פולינום) ע”י λ בלבד ועל הקורא יהיה מוטל להבין מן ההקשר שבזה מדובר.
    ‫הגדרה 1.3. מסלול
    יהי v ∈ V , נגדיר O f ( v ) : = { f k ( v ) ∣ k ∈ N 0 } ונקרא ל- O f ( v ) המסלול של v תחת f .
    ‫הגדרה 1.4. מסילה
    יהי v ∈ V , הסדרה ( f k ( v ) ) k = 0 ∞ תקרא המסילה של v תחת f .
    ‫הגדרה 1.5. מרחב ציקלי
    יהי v ∈ V , נסמן Z f ( v ) : = span ( O f ( v ) ) ונקרא ל- Z f ( v ) מרחב ציקלי של v תחת f .
    ‫הגדרה 1.6. אם קיים v ∈ V כך ש- Z f ( v ) = V אז נאמר ש- v הוא וקטור ציקלי ביחס ל- f וש- V הוא ציקלי ביחס ל- f באמצעות v .
    ‫הגדרה 1.7. קבוצה שמורה
    תהא S ⊆ V , נאמר ש- S שמורה/אינווריאנטית תחת f
    3
    במהלך הקורס נתקלנו גם בביטוי “ f -שמורה” שאינו נכון מבחינה לשונית.
    אם מתקיים f ( S ) ⊆ S , כלומר Im ( f ∣ S ) ⊆ S .
    1. ♣ הגדרה זו תופסת גם עבור תתי-מרחבים וקטוריים וכמובן שבד”כ יעניינו אותנו תמ”וים שמורים ולא סתם קבוצות.
    2. ♣ V ו- { 0 V } מקיימים תכונות רבות באופן טריוויאלי (הם שמורים תחת כל אופרטור לדוגמה) ולכן במקרים רבים (בעיקר כשנרצה למעט אותם) נקרא להם תתי-מרחבים טריוויאליים.
    3. ♣ אם V נ”ס וניתן להציג אותו כסכום ישר של תמ”וים שמורים תחת f אז ההצגה המטריציאלית של f בשרשור בסיסים שלהם היא מטריצה אלכסונית לפי בלוקים.

2 פולינום מינימלי של וקטור ושל קבוצת וקטורים

    יהי V מ”ו נ”ס
    4
    למעשה הסיבה היחידה לדרישה ש- V נ”ס היא כדי שיהיה ברור שהפולינום המינימלי קיים, ניתן להחליף את הדרישה הזו בדרישה שהמסלול של של הווקטור יהיה תלוי ליניארית (כשמדובר בפולינום מינימלי של וקטור) או בדרישה זו על כל הווקטורים בקבוצה/הבסיס שלה (כשמדובר בפולינום מינימלי של קבוצה).
    מעל לשדה F ויהי f אופרטור על V .
    ‫למה 2.1. יהי 0 V ≠ v ∈ V , א”כ קיים n ∈ N מינימלי כך ש- ( v , f ( v ) , f 2 ( v ) , … , f n ( v ) ) תלויה ליניארית ועבור אותו n קיימים a 0 , a 1 , … , a n - 1 ∈ F יחידים כך ש- f n ( v ) = ∑ k = 0 n - 1 a k ⋅ f k ( v ) .
    ‫הגדרה 2.2. בסימוני הלמה שלעיל: נשים לב ש- ( v , f ( v ) , f 2 ( v ) , … , f n - 1 ( v ) ) הוא בסיס של Z f ( v ) , בסיס זה נקרא הבסיס הציקלי של Z f ( v ) ונסמן אותו ב- C f ( v ) .
    ‫הגדרה 2.3. פולינום מינימלי
    בסימוני הלמה שלעיל: נסמן min v ( x ) : = x n + ∑ k = 0 n - 1 - a k ⋅ x k
    5
    ראינו גם את הסימון min v f , שהרי הפולינום המינימלי מוגדר ע”י האופרטור.
    ונקרא ל- min v הפולינום המינימלי של v תחת f .
    1. ♣ מהגדרה הפולינום המינימלי של וקטור האפס הוא 1 , ובנוסף, וקטור האפס הוא הווקטור היחיד ש- 1 הוא הפולינום המינימלי שלו.
    ‫למה 2.4. בסימוניי הלמה שלעיל, נשים לב לכך שמתקיים: [ f ∣ Z f ( v ) ] C f ( v ) = [ 0 0 0 ⋯ 0 a 0 1 0 0 ⋯ 0 a 1 0 1 0 ⋱ 0 a 2 0 0 1 ⋱ ⋮ ⋮ ⋮ ⋮ ⋱ ⋱ 0 a n - 2 0 0 ⋯ ⋯ 1 a n - 1 ]
    ‫הגדרה 2.5. יהי P ( x ) : = ∑ k = 0 n b k ⋅ x k ∈ F [ x ] פולינום מתוקן ( b n = 1 ) המטריצה הבאה תקרא המטריצה המלווה של P : C P : = [ 0 0 0 ⋯ 0 - b 0 1 0 0 ⋯ 0 - b 1 0 1 0 ⋱ 0 - b 2 0 0 1 ⋱ ⋮ ⋮ ⋮ ⋮ ⋱ ⋱ 0 - b n - 2 0 0 ⋯ ⋯ 1 - b n - 1 ]
    1. ♣ המטריצה המלווה של min v היא [ f ∣ Z f ( v ) ] C f ( v ) .
    ‫הגדרה 2.6. נאמר שפולינום P ∈ F [ x ] מאפס וקטור v תחת f אם מתקיים P ( f ) v = 0 V , כמו כן נאמר ש- P מאפס קבוצה S ⊆ V תחת f אם לכל s ∈ S מתקיים [ P ( f ) ] ( s ) = 0 V .
    טענה. נניח ש- V נ”ס ותהא S ⊆ V קבוצת וקטורים, קיים פולינום מתוקן P ∈ F [ x ] יחיד המאפס את S תחת f ומקיים שלכל פולינום G ∈ F [ x ] המאפס את S תחת f מתקיים P ∣ G .
    ‫הגדרה 2.7. לאותו P שבטענה האחרונה נקרא הפולינום המינימלי של S ונסמן אותו ב- min S
    6
    ראינו גם את הסימון min S f , שהרי הפולינום המינימלי מוגדר ע”י האופרטור.
    .

3 פולינום מינימלי של אופרטור, ערכים עצמיים ואופרטורים הניתנים ללכסון

    תהא ( V , f ) מערכת ליניארית מעל לשדה F .
    ‫הגדרה 3.1. נאמר שפולינום P ∈ F [ x ] מאפס את האופרטור f אם לכל v ∈ V מתקיים P ( f ) v = 0 V .
    ‫למה 3.2. אם V נ”ס אז קיים פולינום מתוקן P ∈ F [ x ] יחיד המאפס את f כך שלכל G ∈ F [ x ] המאפס את f מתקיים P ∣ G .
    ‫הגדרה 3.3. לאותו P נקרא הפולינום המינימלי של f ונסמן אותו ב- μ f , בנוסף, לכל A ∈ M n ( F ) נגדיר את הפולינום המינימלי של A להיות הפולינום המינימלי של T A
    7
    כזכור, בקורס הקודם הגדרנו לכל A ∈ M m × n ( F ) את T A להיות ההעתקה הליניארית מ- F n ל- F m המוגדרת ע”י T A ( v ) : = A ⋅ v (לכל v ∈ V ).
    ונסמן אותו ב- μ A .
    1. ♣ נשים לב שמתקיים min V f = μ f ולכן הרבה ממה שנאמר על μ f נכון גם על הפולינום המינימלי של כל תמ”ו של V (עם ההתאמות הנדרשות).
    ‫הגדרה 3.4. נאמר שסקלר λ ∈ F הוא ערך עצמי של f אם קיים 0 V ≠ v ∈ V כך ש- f ( v ) = λ ⋅ v , ל- v כזה נקרא וקטור עצמי עם ערך עצמי λ .
    1. ♣ הדרך הכי פשוטה ליצור אופרטור שאין לו ערכים עצמיים היא סיבוב של המישור ( R 2 ) בזווית שאינה מתחלקת כפולה שלמה של π , ואילו הדרך הכי פשוטה ליצור אופרטור בעל ערכים עצמיים שאינו כפולה של אופרטור הזהות היא שיקוף של המישור סביב ציר סימטריה כלשהו.
    ‫הגדרה 3.5. קבוצת הערכים העצמיים של f תקרא הספקטרום של f ותסומן ע”י σ ( f ) .
    ‫למה 3.6. לכל λ ∈ σ ( f ) נסמן V λ : = { v ∈ V ∣ f ( v ) = λ ⋅ v } ו- V λ : = { v ∈ V ∃ h ∈ N : [ ) f - λ ( h ] ) v ( = 0 V } ; V λ ו- V λ הם תמ”ו של V .
    ‫הגדרה 3.7. לכל λ ∈ σ ( f ) נקרא ל- V λ מרחב עצמי עם ערך עצמי λ ביחס ל- f ול- V λ נקרא מרחב עצמי מוכלל עם ערך עצמי λ ביחס ל- f .
    1. ♣ נשים לב שלכל λ ∈ σ ( f ) מתקיים V λ = ker ( f - λ ) .
    ‫הגדרה 3.8. וקטור השייך למרחב עצמי מוכלל עם ערך עצמי λ ∈ F יקרא וקטור עצמי מוכלל השייך לערך עצמי λ .
    ‫הגדרה 3.9. נניח ש- V נ”ס, נאמר ש- f הוא אופרטור ניתן ללכסון או לכסין אם קיים בסיס B של V כך ש- [ f ] B היא מטריצה אלכסונית, כמו כן נאמר שמטריצה A ∈ M n ( F ) היא מטריצה לכסינה אם קיימת מטריצה אלכסונית הדומה לה.
    1. ♣ מהגדרה אופרטור ניתן ללכסון אם”ם יש לו בסיס של וקטורים עצמיים.
    2. ♣ היותה של מטריצה לכסינה מקלה עלינו להעלות אותה בחזקה: תהיינה A , P , D ∈ M n ( F ) כך ש- A לכסינה, P הפיכה ו- D = P - 1 A P אלכסונית, א”כ לכל m ∈ N מתקיים: A m = ( P D P - 1 ) m = ( P D P - 1 ) ⋅ ( P D P - 1 ) ⋅ … ⋅ ( P D P - 1 ) = P ⋅ D m ⋅ P - 1 והעלאתה של מטריצה אלכסונית בחזקה היא קלה ונוחה.

4 צורת ז’ורדן

    תהא ( V , g ) מערכת ליניארית מעל לשדה F .
    ‫הגדרה 4.1. נאמר ש- g הוא אופרטור נילפוטנטי אם קיים n ∈ N כך ש- g n = 0 ( g n היא העתקת האפס).
    ‫הגדרה 4.2. אם g נילפוטנטי אז קיים h ∈ N מינימלי המקיים g h = 0 , h כזה יקרא הגובה של g ויסומן ב- height ( g ) ; כמו כן אם g נילפוטנטי אז לכל v ∈ V קיים h ∈ N מינימלי כך ש- g h ( v ) = 0 V , h כזה יקרא הגובה של v ויסומן ע”י height ( v ) .
    1. ♣ מהגדרה לכל v ∈ V מתקיים height ( v ) ≤ height ( g ) ושאם h = height ( v ) אז C g ( v ) = ( v , g ( v ) , g 2 ( v ) , … , g h ( v ) ) , מכאן שאם V נ”ס אז מתקיים גם height ( g ) ≤ dim V .
    2. ♣ נשים לב שאם h = height ( v ) אז min v ( x ) = x h ושאם h = height ( g ) אז μ g ( x ) = x h .
    3. ♣ נשים לב שאם h = height ( v ) ו- w : = ∑ k = j h - 1 a k ⋅ g k ( v ) (כאשר a j ≠ 0 ) אז height ( w ) = h - j .
    ‫הגדרה 4.3. נניח ש- g הוא אופרטור נילפוטנטי ויהי v ∈ V ונסמן h : = height ( v ) , הבסיס הציקלי C g ( v ) נקרא שרשרת באורך h
    8
    ראינו גם את השם “ h -שרשרת” שאינו נכון מבחינה לשונית.
    .
    1. ♣ בסיס המהווה שרשור של בסיסים ציקליים נקרא בסיס שרשראות.
    ‫הגדרה 4.4. בלוק ז’ורדן
    9
    ערך בוויקיפדיה: ז'ורדן קאמי.
    אלמנטרי עם ערך עצמי λ ∈ F מסדר n הוא מטריצה J n ( λ ) ∈ M n ( F ) מהצורה: J n ( λ ) = [ λ 0 0 ⋯ 0 0 1 λ 0 ⋯ 0 0 0 1 λ ⋱ ⋮ ⋮ 0 0 1 ⋱ 0 0 ⋮ ⋮ ⋱ ⋱ λ 0 0 0 ⋯ ⋯ 1 λ ]
    1. ♣ יש המגדירים את בלוקי ז’ורדן בתור המטריצה המשוחלפת לזו שהגדרנו (כך למשל הגדירו בוויקיפדיה), אני מנחש שהסיבה לכך היא שהם מסדרים את הבסיס הציקלי בסדר ההפוך מזה שבו הגדרנו אנחנו כך שבאופרטור נילפוטנטי האיבר האחרון בבסיס הציקלי מועתק לזה שלפניו וכן הלאה עד שהראשון מועתק ל- 0 .
    ‫הגדרה 4.5. מטריצה J ∈ M n ( F ) תקרא מטריצת ז’ורדן אם היא מטריצה אלכסונית לפי בלוקים וכל בלוק ז’ורדן שלה הוא בלוק ז’ורדן אלמנטרי.
    תהא A ∈ M n ( F ) מטריצה, נאמר שמטריצת ז’ורדן J ∈ M n ( F ) היא צורת ז’ורדן של A אם J דומה ל- A , כמו כן, בהינתן אופרטור f ש- A היא מטריצה מייצגת שלו בבסיס B נאמר ש- J היא צורת ז’ורדן של f וש- B הוא בסיס מז’רדן של f .
    1. ♣ בקובץ הטענות אנחנו נראה ששתי מטריצות ז’ורדן הן דומות אם”ם הן זהות עד כדי שינוי סדר הבלוקים, א”כ לכל מטריצה יש לכל היותר מטריצת ז’ורדן אחת (עד כדי שינוי סדר הבלוקים) וכך ניתן לקבוע באופן חד משמעי אם שתי מטריצות נתונות דומות זו לזו.

5 הפולינום האופייני, הריבוי הגאומטרי והריבוי האלגברי

    יהי V מ”ו נ”ס מממד n מעל לשדה F ויהא f אופרטור על V .
    טענה. לכל שתי מטריצות דומות A , B ∈ M n ( F ) ולכל λ ∈ F מתקיים: det ( λ ⋅ I n - A ) = det ( λ ⋅ I n - B )
    ‫הגדרה 5.1. יהי B בסיס של V , הפולינום det ( x ⋅ I n - [ f ] B )
    10
    x ⋅ I n - [ f ] B היא מטריצה ב- M n ( F [ x ] ) , כלומר היא מטריצה מעל חוג הפולינומים שמעל F .
    נקרא הפולינום האופייני של f ומסומן ע”י χ f , כמו כן לכל A ∈ M n ( F ) נגדיר χ A : = χ T A
    11
    כזכור T A היא ההעתקה הליניארית המוגדרת ע”י הכפלת המטריצה בוקטור.
    ונקרא ל- χ A הפולינום האופייני של A .
    ‫הגדרה 5.2. אם λ ∈ F הוא ערך עצמי של f אז הריבוי הגאומטרי של λ מוגדר להיות dim V λ = dim ( ker ( f - λ ) ) .
    ‫הגדרה 5.3. אם λ ∈ F הוא ערך עצמי של f אז הריבוי האלגברי של λ (ביחס ל- f ) הוא הריבוי האלגברי של λ ב- χ f , כלומר הריבוי האלגברי של λ הוא החזקה של x - λ בפירוק של χ f לגורמים.
