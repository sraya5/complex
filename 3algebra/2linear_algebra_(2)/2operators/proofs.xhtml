    5.1 ניתוח אופרטור ע”פ צורת ז’ורדן, הפולינום המינימלי והפולינום האופייני

    תהא J צורת ז’ורדן של אופרטור f ויהי λ ∈ σ ( f ) ונניח שהמטריצה הבאה היא הבלוק המתאים ל- λ ב- J : J ( λ ) = [ J h 1 ( λ ) 0 ⋯ 0 0 J h 2 ( λ ) ⋯ 0 0 0 ⋱ 0 0 0 ⋯ J h s ( λ ) ]
    מאחורי צורת ז’ורדן מסתתר בסיס שרשראות המורכב מבסיסי שרשראות של המרחבים העצמיים המוכללים, כל בלוק ז’ורדן אלמנטרי מייצג שרשרת אחת ולכן:
    • מספר השרשראות של λ שווה למספר הבלוקים האלמנטריים בעלי ע”ע λ המופיעים בצורת ז’ורדן, במקרה שלנו זה אומר שמספר השרשראות הוא s .
    • מכיוון שכל שרשרת מכילה בסופה וקטור עצמי ואלו פורשים את V λ נדע ש- dim V λ = s (וזהו גם הריבוי הגאומטרי של λ ).
    • מכיוון שכל שרשרת היא בגודל של הבלוק המתאים לה ואיחוד השרשראות של λ הוא בסיס של V λ נדע שמתקיים dim V λ = h 1 + h 2 + … + h s , כלומר הממד של המרחב העצמי המוכלל שווה לסכום אורכי הבלוקים האלמנטריים (ששווה לגודל של J ( λ ) ).
    • מהגדרה, החזקה של x - λ בפירוק של μ f לגורמים היא אורך השרשרת הארוכה ביותר בבסיס השרשראות של V λ ולכן זהו גם הגודל הגדול ביותר של בלוק אלמנטרי בעל ערך עצמי λ .
    • ראינו לעיל שהחזקה של x - λ בפירוק של χ f לגורמים (הריבוי האלגברי של λ ) היא dim V λ ולכן היא שווה לסכום אורכי הבלוקים האלמנטריים (ששווה לגודל של J ( λ ) ).
    ובכיוון ההפוך:
