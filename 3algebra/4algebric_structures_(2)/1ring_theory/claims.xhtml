    יהי R חוג.

    1.1 התחלה

    משפט 1.1: לכל a ∈ R מתקיים a ⋅ 0 = 0 ⋅ a = 0 .
    ‫מסקנה 1.2. אם יש ב- R שני איברים שונים אז 1 ≠ 0 (כמובן שגם הכיוון ההפוך נכון).
    משפט 1.3: יחידות האיבר האדיש לחיבור
    יהיו a , b ∈ R , אם a + b = a אז b = 0 .
    ‫מסקנה 1.4. יחידות הנגדי
    יהיו a , b , c ∈ R , אם a + b = 0 וגם a + c = 0 אז b = c .
    1. ♣ בגלל מסקנה זו יש משמעות לסימון - a עבור a ∈ R .
    2. ♣ מתקיים - 0 = 0 .
    משפט 1.5: לכל a ∈ R מתקיים a ⋅ 1 = 1 ⋅ a = a .
    טענה 1.6. לכל a , b ∈ F מתקיימים כל הפסוקים הבאים:
    1. - ( - a ) = a .
    2. ( - 1 ) ⋅ a = - a .
    3. a ≠ 0 אם”ם - a ≠ 0 .
    4. a = b אם”ם a - b = 0 .
    5. - ( a + b ) = - a - b .
    6. - ( a - b ) = b - a .
    7. ( - a ) ⋅ b = a ⋅ ( - b ) = - ( a ⋅ b ) .
    8. ( - a ) ⋅ ( - b ) = a ⋅ b .
