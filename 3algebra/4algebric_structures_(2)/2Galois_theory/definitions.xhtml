1 התחלה

    תהא E F הרחבת שדות.
    ‫הגדרה 1.1. E F תיקרא הרחבה רדיקלית פשוטה אם קיימים α ∈ E ו- n ∈ N כך ש- E = F ( α ) ו- α n ∈ F .
    כמו כן, E F תיקרא הרחבה רדיקלית אם קיימים α 1 , α 2 , … , α r ∈ E ו- n 1 , n 2 , … , n r ∈ N כך ש- E = F ( α 1 , α 2 , … , α r ) ו- ( α i ) n i ∈ F ( α 1 , α 2 , … , α i - 1 ) לכל r ≥ i ∈ N .
    1. ♣ מטרתנו היא לקבוע מתי ניתן לפתור פולינום נתון באמצעות חיבור, חיסור, כפל, חילוק והוצאת שורש; הצורה הפורמלית לומר זאת היא שהפולינום מתפצל בשדה הרחבה רדיקלית.
    2. ♣ מטרה נוספת היא לקבוע מתי יש נוסחה קבועה לפתרון כל הפולינומים ממעלה כלשהי מעל שדה נתון.
    ‫הגדרה 1.2. נאמר שפולינום f ∈ F [ x ] ניתן לפתרון באמצעות רדיקלים אם קיימת הרחבות שדות רדיקלית K F כך ש- f מתפצל ב- K .
    ‫הגדרה 1.3. איבר ζ ∈ F ייקרא שורש יחידה מסדר n ∈ N , אם ζ n = 1 , כמו כן ייקרא שורש יחידה פרימיטיבי מסדר n אם (בנוסף) ζ k ≠ 1 לכל n > k ∈ N .
    ‫הגדרה 1.4. חבורת גלואה
    1
    ערך בוויקיפדיה: אווריסט גלואה.

    חבורת גלואה של ההרחבה E F היא Gal ( E F ) : = { φ ∈ Aut ( E ) : φ ∣ F = Id F } (פעולת החבורה היא הרכבה כמובן).
    1. הערה: במקומות אחרים קוראים לקבוצה { φ ∈ Aut ( E ) : φ ∣ F = Id F } חבורת האוטומורפיזמים של ההרחבה E F ומסמנים אותה ב- Aut ( E F ) , את השם "חבורת גלואה" ואת הסימון Gal ( E F ) הם שומרים למקרה שבו E F היא הרחבת גלואה (אנחנו נגדיר מהי הרחבת גלואה בהמשך).
    ‫למה 1.5. לכל תת-חבורה H ⩽ Gal ( E F ) , הקבוצה { x ∈ E ∣ ∀ σ ∈ H σ ( x ) = x } היא שדה המכיל את F .
    ‫הגדרה 1.6. לכל תת-חבורה H ⩽ Gal ( E F ) נסמן E H : = { x ∈ E ∣ ∀ σ ∈ H σ ( x ) = x } , E H ייקרא שדה השבת של H .
    ‫הגדרה 1.7. התאמות גלואה
    התאמות גלואה של ההרחבה E F הן שתי הפונקציות הבאות:
    • F : { H ∣ H ⩽ Gal ( E F ) } → { K ∣ E K הוא שדה ביניים של K } המוגדרת ע"י F ( H ) : = E H = { x ∈ E ∣ ∀ σ ∈ H σ ( x ) = x } לכל H ⩽ Gal ( E F ) .
    • G : { K ∣ E K הוא שדה ביניים של K } → { H ∣ H ⩽ Gal ( E F ) } המוגדרת ע"י G ( K ) : = Gal ( E K ) = { σ ∈ Aut ( E ) ∣ ∀ x ∈ K σ ( x ) = x } לכל שדה ביניים K של E F .
    1. ♣ כלומר F ו- G הן פונקציות בין תתי-החבורות של חבורת גלואה של ההרחבה, לבין קבוצת שדות הביניים של ההרחבה: F מתאימה לכל תת-חבורה את שדה הביניים הגדול ביותר שנשמר תחת פעולתה, ו- G מתאימה לכל שדה ביניים את תת-החבורה הגדולה ביותר שהשדה נשמר תחתיה. השאלה שנעסוק בה תהיה מתי F ו- G הופכיות זו לזו, כלומר מתי יש התאמה חח"ע ועל בין שדות הביניים של הרחבת שדות לבין תתי-החבורות של חבורת גלואה. או אז נוכל להשתמש בכלים החזקים שפיתחנו בקורס הקודם כדי לחקור את שדות הביניים, ולקבוע מתי ניתן לפתור את הפולינום שיצר את ההרחבה.
    ּצריך להסביר שהאוטומורפיזמים מאפשרים לנו לחקור את הקשרים האלגבריים בין שורשי הפולינום היוצר את ההרחבה מבלי "ללכלך את הידיים".
    ‫מסקנה 1.8. מתקיים F ( { Id } ) = E , G ( E ) = { Id } ו- G ( F ) = Gal ( E F ) .
    1. ♣ אין זה מוכרח שיתקיים F ( Gal ( E F ) ) = F , כך לדוגמה לכל σ ∈ Gal ( Q ( 2 3 ) Q ) מתקיים σ ( 2 3 ) = 2 3 ולכן F ( Gal ( Q ( 2 3 ) Q ) ) = Q ( 2 3 ) .
    ‫הגדרה 1.9. נאמר ש- E F היא הרחבת גלואה אם F ו- G הופכיות זו לזו.
    כמו כן, אם E F היא הרחבת גלואה נאמר ש- E F היא הרחבה ציקלית/אבלית/פתירה אם Gal ( E F ) היא חבורה ציקלית/אבלית/פתירה.
    בכיתה הגדרנו שהרחבת גלואה היא הרחבה נורמלית וספרבילית, אנחנו נראה בהמשך שאלה הן הגדרות שקולות.
    ‫מסקנה 1.10. E F היא הרחבת גלואה אם"ם לכל שדה ביניים K (של E F ), ההרחבה E K היא הרחבת גלואה.

2 הרחבות ספרביליות והרחבות נורמליות

    יהי F שדה.

    2.1 הרחבות ספרביליות

    ‫הגדרה 2.1. נאמר שפולינום f ∈ F [ x ] הוא ספרבילי אם בשדה הפיצול שלו הוא מתפרק לגורמים ליניאריים שונים, כלומר הריבוי של כל אחד משורשיו הוא 1 .
    בהינתן הרחבת שדות E F נאמר שאיבר α ∈ F הוא ספרבילי אם m α ספרבילי, וכמו כן נאמר ש- E F היא הרחבה ספרבילית אם כל α ∈ E ספרבילי.
    1. סימון: יהיו Ω שדה סגור אלגברית ו-\varphi:\MKfield\hookrightarrow\Omega שיכון, לכל הרחבה סופית E F נסמן I_{\varphi,\Omega}\left(\nicefrac{\MKbbe}{\MKfield}\right):=\left\{ \hat{\varphi}:\MKbbe\hookrightarrow\Omega\mid\left.\hat{\varphi}\right|_{\MKfield}=\varphi\right\} ו- i φ , Ω ( E F ) : = | I φ , Ω ( E F ) | , כלומר i φ , Ω ( E F ) הוא מספר השיכונים של E ב- Ω המרחיבים את φ .
    2. סימון: לכל פולינום f : = ∑ i = 0 n a i ⋅ x i ∈ Ω נסמן φ ( f ) : = ∑ i = 0 n φ ( a i ) ⋅ x i ∈ Ω .
    למה. יהיו Ω שדה סגור אלגברית ו-\varphi:\MKfield\hookrightarrow\Omega שיכון; לכל α ∈ Ω , i φ , Ω ( F ( α ) F ) שווה למספר השורשים השונים של φ ( m α ) ב- Ω .
    למה. יהיו Ω שדה סגור אלגברית ו-\varphi_{1},\varphi_{2}:\MKfield\hookrightarrow\Omega שיכונים; לכל פולינום f ∈ F [ x ] , מספר השורשים השונים של φ 1 ( f ) שווה למספר השורשים השונים של φ 2 ( f ) .
    משפט. תהיינה K F ו- E K הרחבות שדות, יהיו Ω 1 ו- Ω 2 שדות סגורים אלגברית, ויהיו \varphi_{1}:\MKfield\hookrightarrow\Omega_{1} ו-\varphi_{2}:\MKfield\hookrightarrow\Omega_{2} שיכונים.
    אם E F היא הרחבה סופית אז מתקיימים שלושת הפסוקים הבאים:
    1. i φ 1 , Ω 1 ( E F ) = i φ 2 , Ω 2 ( E F )
    2. i φ 1 , Ω 1 ( E F ) ≥ 1
    3. i φ 1 , Ω 1 ( E F ) = i φ 1 , Ω 1 ( E K ) ⋅ i φ 1 , Ω 1 ( K F )
    1. סימון: לכל הרחבה סופית E F נסמן i ( E F ) : = i φ , Ω ( E F ) עבור שדה סגור אלגברית Ω ושיכון \varphi:\MKfield\hookrightarrow\Omega , ונקרא ל- i ( E F ) דרגת הספרביליות של ההרחבה E F .
    2. ♣ דרגת הספרביליות נקראת כך משום שהיא מודדת עד כמה כל הרחבה פשוטה ב"מגדל" ההרחבות שיוצר את E F היא ספרבילית (כמה שורשים שונים יש לפולינום המינימלי של יוצר ההרחבה).

    2.2 הרחבות נורמליות

    ‫הגדרה 2.2. תהא E F הרחבת שדות, נאמר ש- E F היא הרחבה נורמלית אם לכל פולינום אי-פריק f ∈ F [ x ] , כך שיש ל- f שורש ב- E , f מתפצל ב- E .
    ‫מסקנה 2.3. תהא E F הרחבת שדות, E F היא הרחבה נורמלית אם"ם לכל α ∈ E הפולינום המינימלי של α מעל F מתפצל ב- E .

    ‫הגדרה 3.1. נאמר שהרחבת שדות E F היא הרחבת גלואה אם היא נורמלית וספרבילית.

    3.2 מתי פולינום נתון הוא ספרבילי?

    ‫הגדרה 3.2. נאמר ש- F הוא שדה משוכלל אם כל הרחבה אלגברית שלו היא ספרבילית.
    1. הערה: במקומות אחרים אומרים על שדה כזה שהוא מושלם.
    ‫הגדרה 3.3. יהי f ( x ) : = ∑ i = 0 n a i ⋅ x i ∈ F [ x ] פולינום, פולינום הנגזרת של f הוא הפולינום f ' ( x ) : = ∑ i = 0 n - 1 ( i + 1 ) ⋅ a i + 1 ⋅ x i .
    1. ♣ נשים לב: כש- i מופיע בחזקה ( x i ) הוא איבר ב- N 0 , ואילו כאשר הוא מופיע במקדמי הפולינום ( ( i + 1 ) ⋅ a i + 1 ) הוא איבר ב- F , בפרט ייתכן שמקדמים יתאפסו (אם char ( F ) ראשוני) וכתוצאה מכך יתקיים deg f ' < deg f - 1 .
    2. סימון: יהי E שדה הרחבה של F , הסגור הספרבילי של F בתוך E הוא הקבוצה F E sep : = { α ∈ E ∣ F ספרבילי מעל α } .
    טענה. לכל שדה הרחבה E של F , גם F E sep הוא שדה הרחבה של F .
