1 יחס החלוקה

    טענה 1.1. יהיו a , b , c ∈ Z , מתקיימים כל הפסוקים הבאים:
    1. אם a ∣ b אז גם - a ∣ b וגם a ∣ - b (ולכן גם - a ∣ - b ).
    2. אם a ∣ b ו- b ≠ 0 אז | a | ≤ | b | .
    3. אם a ∣ b אז a ∣ b q לכל q ∈ Z .
    4. אם a ∣ b וגם a ∣ c אז a ∣ b + c
    5. אם a ∣ b וגם a ∣ c אז a ∣ b x + c y לכל x , y ∈ Z .
    6. יחס החלוקה הוא טרנזיטיבי, כלומר אם a ∣ b וגם b ∣ c אז a ∣ c .
    7. מתקיים a ∣ b וגם b ∣ a אם”ם מתקיים a = ± b (או | a | = | b | ).
    8. לכל 0 ≠ m ∈ Z מתקיים a ∣ b אם”ם m a ∣ m b .
    1. ♣ בגלל סעיף 1 משפטים רבים שננסח עבור הטבעיים יהיו נכונים על השלמים בשינויים הקלים המתבקשים.
    טענה 1.2. יהיו a , b ∈ Z , a מחלק את b אם”ם ( b ) ⊆ ( a ) ומתקיים שוויון בין ( a ) ל- ( b ) אם”ם a = ± b .
    משפט 1.3: חילוק עם שארית
    יהיו a , b ∈ Z כך ש- 0 < a (כלומר a ∈ N ), קיימים ויחידים q , r ∈ Z כך ש- 0 ≤ r < a וגם b = q ⋅ a + r ; בנוסף a ∣ b אם”ם r = 0 .
    1. ♣ r זה נקרא השארית של חלוקת b ב- a ו- q נקרא המנה של חלוקה זו.
    2. ♣ יש לשים לב לכך ש- r אי-שלילי ולכן השארית של חלוקת - 8 ב- 3 היא 1 ולא - 2 כפי שניתן היה לחשוב.
    3. ♣ לא בכל חוג קיימת חלוקה עם שארית, זהו הבדל מהותי בין חוג השלמים לחוגים אחרים, כך למשל השקילות בין אי-פריקות לראשוניות (שנגיע אליה בהמשך) נובעת ממשפט זה.
    משפט 1.4: יהיו a 1 , a 2 , … , a n ∈ Z , מתקיימים שני הפסוקים הבאים:

    משפט 1.6: לכל a , b ∈ Z כך שלפחות אחד מהם שונה מאפס היוצר של האידיאל { x ⋅ a + y ⋅ b ∣ x , y ∈ Z } הוא gcd ( a , b ) .
    1. ♣ כלומר לכל d ∈ Z מתקיים d = gcd ( a , b ) אם”ם ( d ) = { x ⋅ a + y ⋅ b ∣ x , y ∈ Z } .
    2. ♣ מכאן נובע שניתן להציג את ה- gcd כצר”ל של a ו- b ושהוא המספר החיובי הקטן ביותר שניתן להציגו כך.
    3. ♣ לאפשרות להציג את ה- gcd כצר”ל של a ו- b ישנה חשיבות רבה בחשבון מודולרי: אם a ו- b זרים אז קיימים x , y ∈ Z כך ש- 1 = x ⋅ a + y ⋅ b ומכאן שלכל b > c , d ∈ N 0 קיים x ' ∈ Z כך ש- c ≡ d + x ' ⋅ a mod b שהרי מתקיים x ⋅ a = 1 - y ⋅ b ומכאן שגם: d + ( c - d ) ⋅ x ⋅ a ≡ d + ( c - d ) ⋅ ( 1 - y ⋅ b ) ≡ d + c - d - ( c - d ) ⋅ y ⋅ b ≡ c mod b
    טענה 1.7. יהיו a , b ∈ Z .
    1. gcd ( a , b ) = gcd ( b , a ) .
    2. לכל m ∈ N מתקיים m ⋅ gcd ( a , b ) = gcd ( m a , m b ) .
    3. אם קיים d ∈ Z כך ש- d ∣ a , b אז אותו d מקיים gcd ( a d , b d ) = 1 d ⋅ gcd ( a , b ) .
    4. לכל x ∈ Z מתקיים gcd ( a , b ) = gcd ( a , b + a x ) .
    5. אם קיים c ∈ Z כך ש- c ∣ a b וגם gcd ( b , c ) = 1 אז c ∣ a .
