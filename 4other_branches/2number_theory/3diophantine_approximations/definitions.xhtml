1 התחלה

    1. ♣ “בתורת המספרים, קירוב דיופנטי של מספר ממשי נתון הוא מספר רציונלי קרוב אל המספר המבוקש. האנליזה הדיופנטית עוסקת, בין השאר, בקיומם של קירובים דיופנטיים, בטיב הקירוב האפשרי, ובהכללות של הבעיה היסודית. התחום נקרא על שמו של דיופנטוס שהציג בעיות שהפתרונות שלהן דווקא במספרים שלמים.” (ציטוט מהערך “דיופנטי קירוב” בוויקיפדיה העברית)
    ‫הגדרה 1.1. מספר z ∈ C יקרא מספר אלגברי אם קיים פולינום P ∈ Z [ x ] כך ש- z הוא שורש של P , אחרת יקרא מספר טרנסצנדנטי (או טרנסצנדנטלי).
    1. ♣ יש המחליפים את Z [ x ] ב- Q [ x ] אך זה שקול מפני שניתן להכפיל במכנה המשותף של כל המקדמים ולקבל פולינום מתאים ב- Z [ x ] .
    2. ♣ קבוצת המספרים האלגבריים היא בת-מנייה שכן Z [ x ] בת-מנייה, ולכן העוצמה של קבוצת המספרים הטרנסצנדנטיים היא עוצמת הרצף, כלומר במונחים של עוצמה ישנם הרבה יותר מספרים טרנסצנדנטיים מאשר מספרים אלגבריים.
    3. ♣ קשה מאד להוכיח שמספרים כמו e ו- π הם אכן טרנסצנדנטיים, לעומתם נראה בהמשך דרך לבנות מספרים טרנסצנדנטיים שאינם מועילים במיוחד (בטח בהשוואה ל- e ו- π ).
    ‫הגדרה 1.2. יהי α ∈ C מספר אלגברי, הדרגה הקטנה ביותר של פולינום ב- Z [ x ] ש- α הוא שורש שלו ( min { deg P ∣ P ∈ Z [ x ] , P ( α ) = 0 } ) תקרא הדרגה של α ואז נאמר ש- α הוא מספר אלגברי מדרגה/מעלה זו.

2 סדרות פרי (Farey)

    ‫הגדרה 2.1. סדרות פרי (Farey)
    1
    ערך בוויקיפדיה האנגלית: John Farey.
    לכל n ∈ N סדרת פרי ה- n -ית (מסומנת ב- F n ) היא סדרה סופית שבה כל הרציונליים בקטע [ 0 , 1 ] שהמכנה שלהם (בהצגה מצומצמת) קטן או שווה ל- n כשהם מסודרים בסדר עולה; כלומר: F 1 : = ( 0 1 , 1 1 ) F 2 : = ( 0 1 , 1 2 , 1 1 ) F 3 : = ( 0 1 , 1 3 , 1 2 , 2 3 , 1 1 ) F 4 : = ( 0 1 , 1 4 , 1 3 , 1 2 , 2 3 , 3 4 , 1 1 ) F 5 : = ( 0 1 , 1 5 , 1 4 , 1 3 , 2 5 , 1 2 , 3 5 , 2 3 , 3 4 , 4 5 , 1 1 ) ⋮

3 שברים משולבים

    אינטואיציה:
    יהי x ∈ R ונסמן r 0 : = ⌊ x ⌋ ו- a 0 : = x - r 0 , א”כ 0 ≤ r 0 < 1 וגם x = a 0 + r 0 ; אם נסמן x 1 : = 1 r 0 > 1 נקבל: x = a 0 + r 0 = a 0 + 1 x 1 כעת ניתן לחזור על התהליך ולסמן r 1 : = ⌊ x 1 ⌋ , a 1 : = x 1 - r 1 ו- x 2 : = 1 r 1 ולקבל: x = a 0 + 1 a 1 + 1 x 2 וכך ניתן לחזור על התהליך כמה פעמים שנרצה: x = a 0 + 1 a 1 + 1 a 2 + 1 x 3 x = a 0 + 1 a 1 + 1 a 2 + 1 … a n - 1 + 1 x n טוב, האמת היא שלא תמיד אפשר להמשיך את התהליך לנצח, אם היה i שעבורו r i = 0 התהליך היה נעצר משום שאז לא היה ניתן להגדיר x i + 1 : = 1 r i ; אנחנו נראה שכצפוי זה קורה אם”ם x ∈ Q .
    ‫הגדרה 3.1. שבר משולב סופי
    שבר משולב סופי הוא ביטוי מהצורה: a 0 + 1 a 1 + 1 a 2 + 1 … 1 a n כאשר 0 ≤ a 0 ∈ R ו- 0 < a i ∈ R לכל n ≥ i ∈ N .
    מכיוון שהביטוי הנ”ל אינו נוח לכתיבה מקובלים שני כתיבים נוספים עבור אותו ביטוי
    2
    יש המשתמשים בסוגריים מרובעים במקום המשולשים.
    : ⟨ a 0 , a 1 , … , a n ⟩ a 0 + 1 a 1 + 1 a 2 + 1 a 3 + … 1 a n - 1 + 1 a n
    ‫הגדרה 3.2. שבר משולב אינסופי
    שבר משולב אינסופי הוא ביטוי מהצורה: a 0 + 1 a 1 + 1 a 2 + 1 a 3 + 1 ⋯ כאשר 0 ≤ a 0 ∈ R ו- 0 < a i ∈ R לכל i ∈ N והכוונה היא לגבול
    3
    אם הוא קיים, אחרת מדובר בביטוי פורמלי בלבד.
    של הסדרה ( b n ) n = 0 ∞ המוגדרת ע”י (לכל n ∈ N ): b 0 : = ⟨ a 0 ⟩ , b n : = ⟨ a 0 , a 1 , a 2 , … , a n - 1 + 1 a n ⟩ אם זו מתכנסת אומרים גם שהשבר המשולב מתכנס.

4 משוואות פל (Pell)

    1. ♣ הסיבה היחידה לכך שפרק זה מופיע בקבצים העוסקים בקירובים דיופנטיים היא שההוכחה ללמה 4.5 (ראו בקובצי הטענות וההוכחות) משתמשת בטענה שלמספר ממשי יש אינסוף קירובים שונים מסדר שני (ראו טענה 2.6 בקובצי הטענות וההוכחות).
    יהי n ∈ N שאינו ריבוע ונסמן R : = Z [ n ] : = { x + n y ∣ x , y ∈ Z } ו- F : = Q [ n ] : = { r + n ⋅ s ∣ r , s ∈ Q } .
    ‫למה 4.1. R הוא חוג ו- F הוא שדה (עם פעולות החיבור והכפל המושרות מ- R ).
    ‫הגדרה 4.2. נגדיר את הנורמה ב- R וב- F ע”י N ( a + n b ) : = ( a + n b ) ( a - n b ) לכל a + n b ב- R ו/או ב- F .
    ‫הגדרה 4.3. משוואות פל
    4
    ערך בוויקיפדיה: פל ג'ון.
    הן משוואות דיופנטיות מהצורה x 2 - D y 2 = N כאשר D ∈ N אינו ריבוע ו- 0 ≠ N ∈ Z .
    1. ♣ כלומר משוואת פל שואלת אם קיים איבר בחוג R : = { x + D y ∣ x , y ∈ Z } כך ש- N הוא הנורמה של אותו איבר.
    2. ♣ קיימות משוואות פל שאין להן פתרון, כך למשל למשוואה x 2 - 3 y 2 = 2 אין פתרון מודולו 3 ולכן אין לה גם פתרון בשלמים.
    למה. הנורמה (ב- R וב- F ) כפלית.
    מסקנה. לכל a , b ∈ Z מתקיים: a + D ⋅ b הפיך ב- R אם”ם a 2 - D ⋅ b 2 = ± 1 .
    1. ♣ מסקנה זו היא הסיבה העיקרית לעניין במשוואות פל.
