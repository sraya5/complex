    1.1 משפטים בסיסיים

    ‫למה 1.1. יהי f ∈ Z [ x ] , לכל x , y ∈ Z המקיימים x ≡ y mod N מתקיים f ( x ) ≡ f ( y ) mod N .
    טענה 1.2. יהי f ∈ Z [ x ] אם קיים x ∈ Z כך ש- f ( x ) = 0 אז אותו x מקיים f ( x ) ¯ = 0 ¯ .
    1. ♣ השימוש המרכזי של טענה זו הוא הוכחה שלפולינום נתון אין שורשים בכך שנראה שאין לו שורשים מודולו N (כאשר את N נוכל לבחור בעצמנו).
    2. ♣ למעשה ניתן להרחיב את הטענה: לכל משוואה שיש לה פתרון בשלמים יש לה גם פתרון בכל חוג שלמים מודולו N , לכן אם ברצוננו להראות שלמשוואה מסוימת בשלמים אין פתרון נוכל לבחור מודולוס כאוות נפשנו (כמובן שיש לבחור אותו בצורה אסטרטגית וזה החלק הכי מסובך בעניין) ולהראות שבחוג המודולרי שלו אין פתרון למשוואה.
    טענה 1.3. לכל x , y ∈ Z ולכל n ∈ Z , אם x ≡ y mod N וגם n ∣ N אז x ≡ y mod n .
    טענה 1.4. לכל x , y ∈ Z ולכל 0 ≠ a ∈ Z מתקיים: a x ≡ a y mod N ⟺ x ≡ y mod N gcd ( a , N )
    ‫מסקנה 1.5. כלל הצמצום
    לכל x , y ∈ Z ולכל 0 ≠ a ∈ Z הזר ל- N מתקיים: a x ≡ a y mod N ⟺ x ≡ y mod N
    משפט 1.6: למשוואה מהצורה a x ≡ b mod N יש פתרון אם”ם gcd ( a , N ) ∣ b , במקרה כזה ניתן להמיר אותה (ע”פ טענה 1.4) למשוואה (נגדיר d : = gcd ( a , N ) ): a d ⋅ x ≡ b d mod N d ואז a d זר ל- N d ולכן יש לו הופכי מודולרי והפתרונות מוכרחים לקיים: x ≡ ( a d ) - 1 ⋅ b d mod N d
    1. ♣ נשים לב לכך שקיום פתרון יחיד מודולו N d אומר שישנם d פתרונות מודולו N .
    2. ♣ ניתן למצוא ההופכי המודולרי ע”י אלגוריתם אוקלידס המורחב: יהיו s , t ∈ Z מספרים זרים, האלגוריתם נותן לנו n , m ∈ Z כך ש- 1 = n ⋅ s + m ⋅ t ומכאן ש- n ⋅ s ≡ 1 mod t ו- m ⋅ t ≡ 1 mod s , כלומר n הוא ההופכי של s מודולו t ו- m הוא ההופכי של t מודולו s (היחידות היא עד כדי שקילות מודולרית כמובן).
    טענה 1.7. הקבוצה ( Z / N Z ) * סגורה תחת כפל, כלומר לכל a ¯ , b ¯ ∈ ( Z / N Z ) * מתקיים a ¯ ⋅ b ¯ ∈ ( Z / N Z ) * .

    1.2 המשפט הקטן של פרמה, פונקציית אוילר ומשפט השאריות הסיני

    משפט 1.8: המשפט הקטן של פרמה
    יהי p ∈ N מספר ראשוני, לכל a ∈ Z כך ש- a / ≡ 0 mod p מתקיים a p - 1 ≡ 1 mod p .
    ‫מסקנה 1.9. יהי p ∈ N ראשוני, לכל a ∈ Z כך ש- a / ≡ 0 mod p ולכל i , j ∈ N , מתקיים a i ≡ a j mod N אם”ם i ≡ j mod p - 1 .
    1. ♣ המסקנה מראה לנו שמהמשפט הקטן של פרמה נובע שכדי לחשב חזקות בחשבון מודולו p ניתן לבצע חשבון מודולו p - 1 על המעריך.
    ‫מסקנה 1.10. יהי p ∈ N מספר ראשוני, לכל a ∈ Z מתקיים a p ≡ a mod p .

    לכל a ∈ Z כך ש- a זר ל- N מתקיים a ϕ ( N ) ≡ 1 mod N .
    1. ♣ משפט אוילר הוא הכללה של המשפט הקטן של פרמה.
    ‫מסקנה 1.13. לכל a ∈ Z זר ל- N ולכל i , j ∈ N , מתקיים a i ≡ a j mod N אם”ם i ≡ j mod ϕ ( N ) .
    1. ♣ המסקנה מראה לנו שממשפט אוילר נובע שכדי לחשב חזקות בחשבון מודולו N ניתן לבצע חשבון מודולו ϕ ( N ) על המעריך וזאת בתנאי שהבסיס זר ל- N .

    1.3 משפטים נוספים

    משפט 1.20: משפט וילסון (Wilson)
    6
    ערך בוויקיפדיה האנגלית: John Wilson.

    יהי p ∈ N מספר ראשוני, מתקיים: ( p - 1 ) ! ≡ - 1 mod p עבור p = 2 המשפט טריוויאלי, נראה כעת שתי הוכחות עבור p ≠ 2 .
    טענה 1.21. יהי 1 < n ∈ N , n הוא מספר ראשוני אם”ם ( n - 1 ) ! ≡ - 1 mod n .
    משפט 1.22: יהי 2 < p ∈ N מספר ראשוני, קיים x ∈ Z כך ש- x 2 ≡ - 1 mod p אם”ם p ≡ 1 mod 4 .
    טענה 1.23. קיימים אינסוף ראשוניים השקולים ל- 1 מודולו 4 , כלומר הקבוצה { p ∈ Prime ∣ p ≡ 1 mod 4 } היא קבוצה אינסופית.
    טענה 1.24. קיימים אינסוף ראשוניים השקולים ל- 3 מודולו 4 , כלומר הקבוצה { p ∈ Prime ∣ p ≡ 3 mod 4 } היא קבוצה אינסופית.
    1. ♣ הטענות בעצם אומרות שבסדרות ( 4 n + 1 ) n = 0 ∞ ו- ( 4 n + 3 ) n = 0 ∞ יש אינסוף ראשוניים ובכך הן מקרה פרטי של משפט דיריכלה שראינו בנושא הקודם: לכל a , d ∈ N הזרים זה לזה קיימים אינסוף איברים ראשונים שהם איברים בסדרה החשבונית ( a + d n ) n = 0 ∞ .
    משפט 1.25: יהיו A ∈ M n ( Z / N Z ) ו- b ∈ ( Z / N Z ) n , למערכת המשוואות הליניאריות A ⋅ x ≡ b mod N

    1. ♣ נזכיר שכדי לפתור מערכות משוואות ליניאריות (ממ”ל) מעל שדה ראינו בליניארית 1 את אלגוריתם הדירוג (דירוג מטריצות) המשתמש בשלוש פעולות שורה אלמנטריות (פש”א): החלפת שורות, כפל שורה בסקלר מהשדה והוספת כפולה של שורה אחת לשורה אחרת; כשמבצעים את האלגוריתם מעל חוג שלמים מודולרי Z / N Z יש להיזהר בשתי הפעולות האחרונות: לא לכל סקלר בחוג יש הופכי, זה הורס את האלגוריתם וכבר א”א לבצעו בצורה מכנית

    משפט 1.26: יהיו a , b , c ∈ Z , לקונגרואנציה a x 2 + b x + c ≡ 0 mod N יש פתרון אם”ם 2 a ∈ ( Z / N Z ) * ו- b 2 - 4 a c הוא שארית ריבועית מודולו N ובמקרה כזה כל d ∈ Z כך ש- d 2 ≡ b 2 - 4 a c mod N נותן פתרון שהוא: x ≡ ( - b + d ) ⋅ ( 2 a ) - 1 mod N

    משפט 1.27: הלמה של הנזל

    ערך בוויקיפדיה: הנזל קורט.

    יהי f ∈ Z [ x ] פולינום ויהיו p , e ∈ N כך ש- p ראשוני, אם קיים a ∈ Z כך ש- f ( a ) ≡ 0 mod p e ו- f ' ( a ) / ≡ 0 mod p

    אז קיים b ∈ Z כך ש- b ≡ a mod p e וגם f ( b ) ≡ 0 mod p e + 1

    ; בנוסף, אותו b הוא יחיד מודולו p e + 1 ,

    2. ♣ בוויקיפדיה מופיעה הרחבה ללמה של הנזל העוסקת במקרה שבו f ' ( a ) ≡ 0 mod p :
