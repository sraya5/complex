1 סדרת ההופכיים המודולריים מורכבת ממספר סדרות חשבוניות

    יהי 1 < n ∈ N .
    יהי n > a ∈ N כך ש- a ו- n זרים זה לזה וגם a ≠ 1 .
    נחשב את c : = n mod a ונגדיר d : = - c - 1 כאשר - c - 1 הוא הנגדי של ההופכי של c מודולו a (ואכן קיים c - 1 שהרי n זר ל- a וממילא גם n mod a ).
    תהא ( b k ) k = 0 ∞ סדרה הנתונה ע”י:
    b k = 1 + n ⋅ d a + k ⋅ d
    טענה 1. לכל k ∈ N 0 ההופכי המודולרי של a (מודולו n + a ⋅ k ) הוא b k .
    הוכחה.
    נוכיח באינדוקציה שלכל k ∈ N 0 מתקיים:
    a ⋅ b k = 1 + ( n + a ⋅ k ) ⋅ d
    תהא ( P k ) k = 0 ∞ סדרת פסוקים לוגיים הנתונה ע”י P k = " a ⋅ b k = 1 + ( n + a ⋅ k ) ⋅ d " לכל k ∈ N 0 .
    • בסיס האינדוקציה
      נשים לב שמתקיים:
      a ⋅ b 0 = a ⋅ 1 + n ⋅ d a = 1 + ( n + a ⋅ 0 ) ⋅ d
      כלומר P 0 = T r u e .
    • צעד האינדוקציה
      יהי k ∈ N 0 ונניח ש- P k = T r u e :
      a ⋅ b k = a ⋅ ( 1 + n ⋅ d a + k ⋅ d ) = 1 + ( n + a ⋅ k ) ⋅ d
      ⇒ a ⋅ b k + 1 = a ⋅ ( 1 + n ⋅ d a + ( k + 1 ) ⋅ d ) = a ⋅ ( 1 + n ⋅ d a + k ⋅ d ) + a ⋅ d = a ⋅ b k + a ⋅ d = 1 + ( n + a ⋅ k ) ⋅ d + a ⋅ d = 1 + ( n + a ⋅ [ k + 1 ] ) ⋅ d
      כלומר P k + 1 = T r u e .
    מכיוון ש- d < a נדע שלכל k ∈ N 0 מתקיים ( n + a ⋅ k ) ⋅ d < ( n + a ⋅ k ) ⋅ a ומאחר ש- 1 < n ∈ N נוכל לומר שגם:
    a ⋅ b k = 1 + ( n + a ⋅ k ) ⋅ d < ( n + a ⋅ k ) ⋅ a
    ממשפט השאריות הסיני ומיחידות ההופכי המודולרי נובע שלכל k ∈ N 0 , b k הוא ההופכי של a מודולו n + a ⋅ k .

    ‫מסקנה 2. יהי a > g ∈ N הזר ל- a נסמן m g : = a + ( - g - 1 ) (כאשר - g - 1 הוא הנגדי של ההופכי של g מודולו a ), לכל k ∈ N 0 מתקיים:
    a ⋅ 1 + m g ⋅ g a + k ⋅ g = 1 + ( m g + a ⋅ k ) ⋅ g
    כלומר לכל a > g ∈ N הזר ל- a יש סדרת הופכיים של a כך ש- g הוא ההפרש שלה כאשר כל הבסיסים (שביחס אליהם איברי הסדרה הם הופכיים של a ) שקולים זה לזה מודולו a .
    הוכחה. m g = a + ( - g - 1 ) , מכאן שגם g = - ( m g - a ) - 1 ; כעת אפשר להמשיך כמקודם כאשר g משמש בתפקיד d .
    ■
    ‫מסקנה 3. אין ל- a שתי סדרות הופכיים שההפרש שלהן זהה וגם כל הבסיסים (שביחס אליהם איברי הסדרה הם הופכיים של a ) שקולים זה לזה מודולו a .
    הוכחה. לכל a < m ∈ N יש ל- a הופכי מודולרי מודולו m אם”ם a ו- m זרים זה לזה.
    לכל 2 a < m ∈ N ההופכי המודולרי של a מודולו m כבר כלול בסדרה שאיברה הראשון הוא ההופכי המודולרי של a מודולו v ∈ { v ∈ N : a < v ≤ 2 a } .
    מכאן שקיימות בדיוק ϕ ( a ) סדרות הופכיים של a כאשר כל הבסיסים (שביחס אליהם איברי הסדרה הם הופכיים של a ) שקולים זה לזה מודולו a .
    ממסקנה 2 נובע שישנם ϕ ( a ) הפרשים שונים של סדרות הופכיים וממילא לכל סדרה יש הפרש שונה.
