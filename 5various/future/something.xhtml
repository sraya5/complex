תהא ( I n ) n = 0 ∞ סדרת קטעים המוגדרת ע”י: I n : = [ 2 π n + π 2 - 1 , 2 π n + π 2 + 1 ] ∪ [ 2 π n + 3 π 2 - 1 , 2 π n + 3 π 2 + 1 ] לכל n ∈ N 0 .
לכל n ∈ N 0 הפונקציה sin עולה בקטע [ 2 π n + π 2 - 1 , 2 π n + π 2 ] , יורדת בקטע [ 2 π n + π 2 , 2 π n + π 2 + 1 ] , יורדת בקטע [ 2 π n + 3 π 2 - 1 , 2 π n + 3 π 2 ] ועולה בקטע [ 2 π n + 3 π 2 , 2 π n + 3 π 2 + 1 ] .
לכל n ∈ N 0 קיימים לפחות ארבעה טבעיים השייכים ל- I n , וזאת משום ש- I n הוא איחוד של שני קטעים באורך 2 (ההפרש בין המקסימום של הקטע למינימום שלו הוא 2 ).
נשים לב שלכל n ∈ N 0 מתקיים: sin ( π 2 - 1 ) = sin ( 2 π n + π 2 - 1 ) = sin ( 2 π n + π 2 + 1 ) sin ( 3 π 2 - 1 ) = sin ( 2 π n + 3 π 2 - 1 ) = sin ( 2 π n + 3 π 2 + 1 ) - sin ( π 2 - 1 ) = sin ( 3 π 2 - 1 )
