3 העשרה: חזקה מרוכבת

    3.1 גבול של סדרה מרוכבת

    ‫הגדרה 3.1. התכנסות של סדרה
    תהא ( a n ) n = 1 ∞ סדרה של מספרים מרוכבים ( a n ∈ C לכל n ∈ N ); נאמר ש- ( a n ) n = 1 ∞ מתכנסת אם קיים L ∈ C כך שלכל 0 < ε ∈ R ועבור n ∈ N גדול דיו מתקיים | L - a n | < ε ,
    כלומר לכל 0 < ε ∈ R קיים N ∈ N כך שלכל N < n ∈ N מתקיים | L - a n | < ε .
    L כנ”ל יקרא גבול של ( a n ) n = 1 ∞ .
    1. ♣ נשים לב שהביטוי “ | L - a n | ” הוא ערך מוחלט של מספר מרוכב.
    טענה 3.2. תהא ( a n ) n = 1 ∞ סדרת מספרים מרוכבים אם ( a n ) n = 1 ∞ מתכנסת אז קיים L ∈ C יחיד כך ש- L הוא גבול של ( a n ) n = 1 ∞ .

    3.2 הגדרת החזקה המרוכבת

    בקורס “חשבון אינפיניטסימלי (2)“ ראינו שלכל x ∈ R מתקיים: exp ( x ) = ∑ n = 0 ∞ x n n ! כעת, מבלי להוכיח זאת, נאמר שגם לכל מספר מדומה z ∈ C הטור ∑ n = 0 ∞ z n n ! מתכנס.

    1. ♣ ומה בדבר מספר מרוכב? נזכור שכל מספר מרוכב הוא סכום של מספר ממשי ומספר מדומה ולכן מכיוון שאנחנו רוצים שהחזקה המרוכבת תקיים את חוקי החזקות היא צריכה לקיים: exp ( a + b i ) = exp ( a ) ⋅ exp ( b i )
    ‫הגדרה 3.5. יהי z ∈ C מספר מרוכב ויהיו a , b ∈ R כך ש- z = a + b i , נגדיר: exp ( z ) = exp ( a ) ⋅ exp ( b i )
    ‫הגדרה 3.6. חזקה של בסיס ממשי במעריך מרוכב
    יהיו 0 < r ∈ R ו- z ∈ C ויהיו a , b ∈ R כך ש- z = a + b i , נגדיר: r z = exp ( z ⋅ ln r )
    משפט 3.7: חוקי חזקות
    יהיו 0 < x , y ∈ R ו- z , w ∈ C , מתקיימים שלושת הפסוקים הבאים:
    1. x z ⋅ x w = x z + w
    2. ( x z ) w = x z ⋅ w
    3. ( x ⋅ y ) z = x z ⋅ y z
    צריך להוסיף הוכחה.
