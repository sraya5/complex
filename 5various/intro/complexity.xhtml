4 מחלקות סיבוכיות

    4.1 קצב גידול של פונקציות

    את הנושא הזה כבר למדנו בקורס “בדידה מתמטיקה” אך בכל זאת נחזור כאן על כמה ההגדרות וטענות.
    נכון לעכשיו הנושא הזה לא מופיע בסיכומים שלי ממתמטיקה בדידה.

    4.1.1 הגדרות

    תהיינה f , g : N → R +
    5
    נזכיר: R + : = ( 0 , ∞ ) , למעשה אין צורך בכך שהטווח יהיה R + אלא עבור n ∈ N גדול דיו מתקיים 0 < f ( n ) , g ( n ) .
    פונקציות
    6
    למעשה מדובר בסדרות ואני לא רואה שום סיבה שההגדרות לא תהיינה תקפות גם לגבי פונקציות ממשיות רגילות ( f : R + → R + ).
    .
    ‫הגדרה 4.1. נאמר ש- g היא O גדול של f ונסמן g ∈ O ( f )
    7
    בכך אנחנו למעשה מגדירים את הקבוצה O ( f ) .
    אם קיימים 0 < c ∈ R ו- N ∈ N כך שלכל N < n ∈ N מתקיים f ( n ) ≤ c ⋅ g ( n ) .
    ‫הגדרה 4.2. נאמר ש- g היא 𝛺 של f ונסמן g ∈ 𝛺 ( f ) אם קיימים 0 < c ∈ R ו- N ∈ N כך שלכל N < n ∈ N מתקיים c ⋅ g ( n ) ≤ f ( n ) .
    ‫הגדרה 4.3. נאמר ש- g היא Θ של f ונסמן g ∈ Θ ( f ) אם קיימים 0 < c 1 , c 2 ∈ R ו- N ∈ N כך שלכל N < n ∈ N מתקיים c 1 ⋅ g ( n ) ≤ f ( n ) ≤ c 2 ⋅ g ( n ) .
    1. ♣ סימון נפוץ מאד לכך ש- g היא O גדול של f הוא g = O ( f ) אבל הוא בעייתי מפני שייתכן שגם h ∈ O ( f ) ולמרות זאת g ≠ h בסתירה לטרנזיטיביות השוויון, כלומר גם מי שמשתמש בסימון השוויון אינו מתכוון באמת לכך שמה שכתוב באגף ימין הוא בדיוק מה שכתוב באגף שמאל.

    4.1.2 טענות

    טענה 4.4. תהיינה f , g , h : N → R + , מתקיימים כל הפסוקים הבאים:
    • Θ ( f ) = Θ ( g ) ⟺ g ∈ Θ ( f ) .
    • אם קיימים 0 < c ∈ R ו/או b ∈ R כך ש- f = c ⋅ g + b אז Θ ( f ) = Θ ( g ) .
    • Θ ( f ) = O ( f ) ∩ 𝛺 ( f ) .
    • f ∈ Θ ( f ) וממילא גם f ∈ O ( f ) ו- f ∈ 𝛺 ( f ) .
    • f ∈ 𝛺 ( g ) ⟺ g ∈ O ( f ) , לכן נוכל בטענות הבאות רק על O גדול וממילא הטענות תהיינה נכונות גם עבור 𝛺 .
    • g ∈ O ( f ) ⟺ O ( f ) ⊆ O ( g ) .
    • g ∈ O ( f ) ⋀ f ∈ O ( h ) ⇒ h ∈ O ( h ) .
    משפט 4.5: יהי P ∈ R [ x ] פולינום מדרגה k ∈ N 0 כך שהמקדם של החזקה ה- k חיובי, מתקיים P ∈ Θ ( n k ) .
    1. ♣ באופן כללי כאשר מחברים פונקציות הולכים לפי זו ששואפת לאינסוף מהר יותר.
    טענה 4.6. לכל k , m ∈ N 0 כך ש- k < m מתקיים O ( n k ) ⊆ O ( n m ) ובפרט n k ∈ O ( n m ) .
    משפט 4.7: מתקיים O ( 1 ) ⊆ O ( log b n ) ⊆ O ( n ) ⊆ O ( n ) ⊆ O ( n log b n ) ⊆ O ( n k ) ⊆ O ( a n ) ⊆ O ( n ! ) (לכל 1 < a , b , k ∈ N ).
    1. ♣ לעומת זמן ריצה לוגריתמי
      8
      לכל 1 < a , b ∈ R מתקיים לכל x ∈ R השוויון log a ( x ) = log a ( b ) ⋅ log b ( x ) ולכן מדובר רק בכפל בקבוע שכמובן אינו משנה את מחלקת הסיבוכיות.
      , כל זמני הריצה הפולינומיים שדרגתם שונה וכל זמני הריצה האקספוננציאליים שבסיסם שונה - שונים זה מזה.

    4.2 המחלקות

    בגלל המשפט האחרון מחלקות הסיבוכיות המעניינות אותנו הן המחלקות שלהלן (ע”פ הסדר הרצוי).
    סיבוכיות טובה:
    • O ( 1 ) - זמן ריצה קבוע שאינו תלוי בקלט
    • O ( log n ) - זמן ריצה לוגריתמי
    • O ( n ) - זמן ריצה “שורשי”
    סיבוכיות סבירה:
    • O ( n ) - זמן ריצה ליניארי
    • O ( n log n ) זמן ריצה ליניארי-לוגריתמי
    סיבוכיות גרועה:
    • O ( n k ) - זמן ריצה פולינומי
    סיבוכיות נוראית:
    • O ( a n ) - זמן ריצה מעריכי
    • O ( n ! ) - זמן ריצה “עצרתי”
    1. ♣ כשהקלט הוא יותר ממשתנה אחד אז הסיבוכיות עובדת ע”פ פונקציה בשני משתנים (כגון O ( n ⋅ m ) ).
