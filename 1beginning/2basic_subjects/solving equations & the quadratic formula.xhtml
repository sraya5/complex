    טענה 2.4. לכל שתי פונקציות f , g : A → B כך ש- A , B ⊆ R מתקיים: { x ∈ R ∣ f ( x ) = g ( x ) } = { x ∈ R ∣ f ( x ) - g ( x ) = 0 }
    טענה 2.5. לכל שני פולינומים p , q ∈ R [ x ] הדרגה של פולינום ההפרש ( p - q ) קטנה או שווה לדרגה המקסימלית מבין אלו של p ו- q , כלומר: deg ( p - q ) ≤ max { deg p , deg q }
    1. ♣ משתי הטענות ביחד נובע שכדי לפתור כל משוואה ריבועית מעל R מספיק לדעת כיצד לפתור כל משוואה מהצורה f ( x ) = 0 כאשר f היא פונקציה פולינומיאלית מעל R שדרגתה קטנה או שווה ל- 2 .

    2.3 פתרון משוואה ריבועית בשיטה ראשונה

    תהא f פונקציה פולינומיאלית מעל R שדרגתה קטנה או שווה ל- 2 ויהיו a , b , c ∈ R כך ש- f ( x ) = a x 2 + b x + c לכל x ∈ R ; מהגדרה, קבוצת הפתרונות של המשוואה f ( x ) = 0 היא: { x ∈ R ∣ f ( x ) = 0 } = { x ∈ R ∣ a x 2 + b x + c = 0 }
    טענה 2.6. נניח ש- a = 0 ונחלק למקרים:
    • אם b ≠ 0 אז מתקיים: { x ∈ R ∣ a x 2 + b x + c = 0 } = { - c b }
    • אם b = 0 וגם c = 0 אז מתקיים: { x ∈ R ∣ a x 2 + b x + c = 0 } = R
    • אם b = 0 אך c ≠ 0 אז מתקיים: { x ∈ R ∣ a x 2 + b x + c = 0 } = ∅

    2. ♣ אסטרטגיה ראשונה לפתרון משוואות: נניח שיש למשוואה פתרון ואחרי שנמצא את כל המועמדים נציב אותם אחד אחד בביטוי b x + c ונראה אם אנחנו אכן מקבלים 0 .
