1 בניית שדה המספרים המרוכבים

    1.1 התחלה

    1. ♣ ראינו בהקדמה את המוטיבציה והאינטואיציה שמאחורי המספרים המרוכבים, כעת נבנה את שדה המספרים המרוכבים באופן פורמלי.
    2. ♣ כפי שראינו בהקדמה אנחנו עומדים לקבל ש- C = { x + y ⋅ i ∣ x , y ∈ R } , מבחינה פורמלית אין לביטוי הזה שום משמעות כרגע מפני שעוד לא הגדרנו חיבור וכפל על C אך מבחינה אינטואיטיבית הוא אומר לנו המון: כל מספר מרוכב ניתן להצגה באופן יחיד ע”י שתי “קואורדינטות” ממשיות ולכן בעצם כל מספר מרוכב ניתן להצגה באופן יחיד כנקודה במישור (כשאנו מדברים עליו בהקשר של המרוכבים נקרא לו המישור המרוכב).
    סימון:
    נסמן C : = R 2 ונקרא ל- C קבוצת המספרים המרוכבים, לאחר שנגדיר עליה פעולות חיבור וכפל ונוכיח שאכן מדובר בשדה נקרא ל- C שדה המספרים המרוכבים.
    סימון:
    נסמן 0 C : = [ 0 0 ] , 1 C : = [ 1 0 ] ו- i : = [ 0 1 ] .
    1. ♣ אנחנו רוצים שיתקיים ( x , y ) = x ⋅ 1 C + y ⋅ i וכבר ראינו בהקדמה כיצד עלינו להגדיר את החיבור והכפל של מספרים מרוכבים אם רצוננו ש- C יהיה שדה ושיתקיים i 2 = - 1 C .

    3. ♣ מכיוון ש- 1 C הנ”ל הוא האיבר האדיש לכפל נוכל לכתוב מעתה [ x y ] = x + y ⋅ i לכל [ x y ] ∈ C .

    1.2 הצמוד המרוכב והערך המוחלט

    1. ♣ בקובץ ההוכחות נראה שההופכי של מספר 0 ≠ x + y i ∈ C הוא: x - y i x 2 + y 2 = x - y i ( x + y i ) ( x - y i ) עובדה זו נותנת לנו מוטיבציה לתת שם ליחס שבין x + y i ל- x - y i .
    ‫הגדרה 1.5. יהי z : = x + y i ∈ C , הצמוד המרוכב של z הוא z ¯ : = x - y i , כלומר זהו המספר המקיים Re ( z ¯ ) = Re ( z ) ו- Im ( z ¯ ) = - Im ( z ) .
    1. ♣ מבחינה גאומטרית הצמדה היא פשוט שיקוף סביב ציר ה- x במישור המרוכב.
    2. ♣ הצמוד המרוכב הוגדר גם עבור 0 : מתקיים 0 ¯ = 0 .
    ‫מסקנה 1.6. לכל z ∈ C מתקיימים ארבעת הפסוקים הבאים:
    1. z ¯ ¯ = z .
    2. z + z ¯ = 2 ⋅ Re ( z ) .
    3. z - z ¯ = 2 i ⋅ Im ( z ) .

2 ההצגה הקוטבית

    2.1 התחלה

    1. ♣ עד כה הצגנו את המספרים המרוכבים (שהם נקודות במישור המרוכב) בהצגה הקרטזית שלהם, אך בקובץ
      “על קואורדינטות ומערכות צירים” (עוד לא נכתב) ראינו שניתן לאפיין נקודות במישור גם ע”י ההצגה הקוטבית שלהן, כלומר ע”י מרחקן מראשית הצירים והזווית שהן יוצרות עם החלק החיובי של ציר ה- x , א”כ ניתן לדבר גם על ההצגה הקוטבית של המספרים המרוכבים ומכיוון שהגדרנו את הערך המוחלט כך שיהיה זהה למושג האוקלידי של מרחק נוכל לבטא את הרדיוס בהצגה הקוטבית כערך המוחלט של מהספר המרוכב.
    ‫הגדרה 2.1. יהי 0 ≠ z : = x + y i ∈ C , הצגה קוטבית (נקראת גם הצגה פולרית) של z היא ביטוי מהצורה z = r ⋅ ( cos θ + i ⋅ sin θ ) כאשר r : = | z | = x 2 + y 2 ואילו θ מוגדרת ע”י: θ : = { arctan ( y x ) + 2 π k x > 0 arctan ( y x ) ± π + 2 π k x < 0 3 π 2 + 2 π k x = 0 , y > 0 π 2 + 2 π k x = 0 , y < 0 עבור k ∈ Z כלשהו.
    ההצגה הקוטבית של 0 בשדה המרוכבים היא 0 ⋅ ( cos θ + i ⋅ sin θ ) עבור θ ∈ R כלשהי.
    1. ♣ מהגדרה r ≥ 0 .
    2. ♣ כזכור אין שום בעיה בכך שיש יותר מהצגה קוטבית אחת למספר נתון משום ש- cos ו- sin הן פונקציות מחזוריות שאורך המחזור שלהן הוא 2 π והן מוגדרות על כל הישר הממשי, וגם אם אותו מספר הוא 0 אין כאן בעיה משום ש- 0 ⋅ ( cos θ 1 + i ⋅ sin θ 1 ) = 0 = 0 ⋅ ( cos θ 2 + i ⋅ sin θ 2 ) לכל θ 1 , θ 2 ∈ R .
    3. ♣ כעת ניתן לראות בבירור שהמטריצה המתאימה לכפל במספר מרוכב היא מטריצת מתיחה וסיבוב שכן המטריצה המתאימה ל- r ⋅ ( cos θ + i ⋅ sin θ ) היא: r ⋅ [ cos θ - sin θ sin θ cos θ ] = [ r ⋅ cos θ - r ⋅ sin θ r ⋅ sin θ r ⋅ cos θ ]
    סימון:
    תהא θ ∈ R , נסמן cis ( θ ) : = cos θ + i ⋅ sin θ ; א”כ ההצגה הקוטבית של מספר מרוכב ניתנת לכתיבה בקצרה ע”י r ⋅ cis ( θ ) כאשר r הוא הערך המוחלט שלו ו- θ מוגדרת כפי שהוגדרה לעיל.
