1 התחלה

    יהי F שדה.
    ‫הגדרה 1.1. פולינום מעל F הוא ביטוי מהצורה a 0 + a 1 x + a 2 x 2 + ... + a n x n כאשר a k ∈ F לכל n ≥ k ∈ N .
    המקדם של החזקה הגדולה ביותר ( a n ) נקרא המקדם המוביל והמקדם של החזקה ה- 0 ( a 0 ) נקרא המקדם החופשי.
    נסמן את קבוצת הפולינומים מעל F ב- F [ x ] .
    1. ♣ מונום הוא פולינום בעל איבר יחיד, כלומר כל פולינום הוא חיבור של מונומים.
    2. ♣ מבחינה פורמלית פולינומים הם פשוט מחרוזת תווים, כדאי לזכור זאת בהמשך מבלי לאבד את האינטואיציה המזהה פולינום כפונקציה פולינומיאלית.
      דוגמה הממחישה את העניין: הפולינומים 1 , x 2 + x + 1 ו- x 3 + x + 1 הם פולינומים שונים מעל F 2 למרות שכפונקציות הם שווים: 1 3 + 1 + 1 = 1 = 1 2 + 1 + 1 0 3 + 0 + 1 = 1 = 0 2 + 0 + 1
    3. ♣ ה- x בסימון F [ x ] הוא משתנה סרק, באותה מידה היה יכול להופיע שם כל קשקוש עקבי אחר, יתרה מזאת - לפעמים משתמשים בסימון זה כאשר במקום x כותבים מספר כלשהו ואז פירוש הסימון הוא קבוצת כל הביטויים הפולינומיאליים כשמציבים בהם את אותו מספר, לדוגמה (גאוס של השלמים חוג): Z [ i ] : = { ∑ k = 0 n a k ⋅ i k a 0 , a 1 , … , a n ∈ Z } = { a + b i ∣ a , b ∈ Z }
    4. ♣ כל מה שנראה בקובץ זה נכון גם עבור קבוצת הפולינומים מעל חוג (שאינו בהכרח שדה), למעט נקודה אחת שעליה נעיר במפורש.
    ‫הגדרה 1.2. שוויון בין פולינומים
    יהיו P , G ∈ F [ x ] ויהיו a 0 , a 1 , … , a n , b 0 , b 1 , … , b m כך שמתקיים: P ( x ) = ∑ k = 0 n a k ⋅ x k G ( x ) = ∑ k = 0 m b k ⋅ x k נאמר ש- P = G אם a k = b k לכל min { n , m } ≥ k ∈ N 0 ובנוסף מתקיים: n ≥ m ו- a k = 0 לכל m < k ∈ N ו/או n ≤ m ו- b k = 0 לכל n < k ∈ N .
    כלומר אם ממקום מסוים ואילך המקדמים של פולינום הם אפסים, אז מקדמים אלו (אך לא אלו שלפניהם) אינם משפיעים על זהות הפולינום.
    ‫הגדרה 1.3. הדרגה או המעלה של פולינום 0 ≠ P ∈ F [ x ]
    1
    מדובר בפולינום האפס וכך גם בהמשך הקובץ.
    (מסומנת ב- deg P ) היא החזקה הגדולה ביותר של הפולינום שהמקדם שלה אינו 0
    2
    כלומר אם P ( x ) = ∑ k = 0 n a k ⋅ x k אז deg P : = max { k ∈ N 0 ∣ a k ≠ 0 } .
    .
    נסמן גם deg 0 : = - ∞ ונכתוב deg 0 < deg P לכל 0 ≠ P ∈ F [ x ] .
    1. ♣ באותה מידה היה אפשר להגדיר deg 0 : = - 1 (או כל מספר שלילי אחר) ואז אפילו לא היינו צריכים להסכים ש- deg 0 < deg P .
    2. ♣ נזהה פולינום עם סדרת המקדמים האינסופית שלו, כלומר עם סדרה כזו: ( a 0 , a 1 , a 2 , ... , a n , 0 , 0 , 0 , ... )
      3
      סדרות כאלה (שהחל ממקום מסוים כל איבריהן אפסים) נקראות סדרות נתמכות סופית.
      , כך שהפולינום 1 + x והפולינום 1 + x + 0 ⋅ x 2 (לדוגמה) יזוהו כאותו פולינום.
    3. ♣ פולינומים מדרגה 0 נקראים פולינומים קבועים משום שכפונקציות הם מהווים פונקציות קבועות.
    ‫הגדרה 1.4. יהיו P , G ∈ F [ x ] ויהיו a 0 , a 1 , … , a n , b 0 , b 1 , … , b m כך שמתקיים: P ( x ) = ∑ k = 0 n a k ⋅ x k G ( x ) = ∑ k = 0 m b k ⋅ x k
    • חיבור פולינומים יוגדר ע”י
      4
      אם n < m אז נגדיר a k : = 0 לכל k ∈ N כך ש- n < k ≤ m ואם m < n אז נגדיר b k : = 0 לכל k ∈ N כך ש- m < k ≤ n .
      : ( P + G ) ( x ) : = ∑ k = 0 max { n , m } ( a k + b k ) ⋅ x k = ( ∑ k = 0 n a k ⋅ x k ) + ( ∑ k = 0 m b k ⋅ x k )
    • כפל פולינומים יוגדר ע”י: ( P ⋅ G ) ( x ) : = ( ∑ i = 0 n a i ⋅ x i ) ⋅ ( ∑ j = 0 m b j ⋅ x j ) : = ∑ i = 0 n ( a i ⋅ x i ⋅ ( ∑ j = 0 m b j ⋅ x j ) ) : = ∑ i = 0 n ( ∑ j = 0 m ( a i ⋅ x i ) ⋅ ( b j ⋅ x j ) ) : = ∑ i = 0 n ( ∑ j = 0 m a i ⋅ b j ⋅ ( x i ⋅ x j ) ) ∑ i = 0 n ( ∑ j = 0 m a i ⋅ b j ⋅ x i + j ) = ∑ k = 0 n + m ( ( ∑ i = 0 k a i ⋅ b k - i ) ⋅ x k )
    • כפל פולינום בסקלר יוגדר ע”י (לכל c ∈ F ): ( c ⋅ P ) : = ∑ k = 0 n ( c ⋅ a k ) ⋅ x k
    1. ♣ כפל בסקלר הוא בעצם כפל בפולינום הקבוע המתאים.
    טענה 1.5. F [ x ] הוא חוג חילופי
    5
    חוג חילופי (קומוטטיבי) הוא קבוצה שעליה מוגדרות פעולות חיבור וכפל המקיימת את כל אקסיומות השדה מלבד קיום הופכי.
    ביחס לפעולות החיבור והכפל הנ”ל.
    ‫הגדרה 1.6. יהיו P , G ∈ F [ x ] , נאמר ש- G מחלק את P (או ש- P הוא כפולה של G ) אם קיים Q ∈ F [ x ] כך ש- P = Q ⋅ G
    6
    כלומר שארית החלוקה של P ב- G היא פולינום האפס.
    ובמקרה כזה נסמן G ∣ P .
    טענה 1.7. יהיו A , B , C ∈ F [ x ] , מתקיימים שלושת הפסוקים הבאים:
    1. אם A ∣ B אז A ∣ Q ⋅ B לכל Q ∈ F [ x ] .
    2. אם A ∣ B וגם A ∣ C אז A ∣ P ⋅ B + G ⋅ C לכל P , G ∈ F [ x ] .
    3. יחס החלוקה הוא טרנזיטיבי, כלומר אם A ∣ B ו- B ∣ C אז A ∣ C .
    משפט 1.8: לכל שני פולינומים P , G ∈ F [ x ] מתקיים:
    1. deg ( P + G ) ≤ max { deg P , deg , G }
      7
      בד”כ מתקיים שוויון אך אם deg P = deg Q וגם המקדמים המובילים נגדיים זה לזה (סכומם הוא 0 F ) אז יתקיים א”ש חזק.
      .
    2. deg ( P ⋅ G ) = deg P + deg G
      8
      אם אחד הפולינומים הוא פולינום האפס, נגיד P , אז - ∞ + deg G : = - ∞ = deg ( 0 ) = deg ( P ⋅ G ) .
      .
    ‫מסקנה 1.9. יהיו 0 ≠ P , Q ∈ F [ x ] , אם Q ∣ P אז deg Q ≤ deg P .
    ‫מסקנה 1.10. אם לפולינום P ∈ F [ x ] יש פולינום הופכי (כלומר קיים G ∈ F [ x ] כך ש- P ⋅ G = 1 ), אז P הוא פולינום קבוע.
    1. ♣ כלומר קבוצת האיברים ההפיכים בחוג פולינומים מעל שדה
      9
      מעל חוג (בניגוד לשדה) קבוצת הפולינומים ההפיכים תהיה קבוצת הפולינומים הקבועים כך שהקבוע המתאים להם הוא איבר הפיך בחוג.
      היא קבוצת הפולינומים הקבועים.
    ‫מסקנה 1.11. יהיו P , G ∈ F [ x ] , P ו- G מחלקים זה את זה אם”ם קיים 0 ≠ c ∈ F כך ש- P = c ⋅ G , אותו c הוא המנה של חלוקת המקדם המוביל של P במקדם המוביל של G .
    ‫מסקנה 1.12. יהיו P , G ∈ F [ x ] שני פולינומים המחלקים זה את זה ובעלי מקדמים זהים, מתקיים P = G .
    משפט 1.13: חילוק פולינומים עם שארית
    לכל שני פולינומים P , G ∈ F [ x ] כך ש- G ≠ 0 קיימים שני פולינומים Q , R ∈ F [ x ] יחידים כך ש- P = Q ⋅ G + R ו- deg R < deg G .
    1. ♣ R זה נקרא השארית של חלוקת P ב- G ו- Q נקרא המנה של חלוקה זו.
    2. ♣ בעמוד הבא מופיע אלגוריתם לחילוק פולינומים עם שארית (דוגמאות לפעולת האלגוריתם ניתן למצוא בוויקיפדיה וב-MathWorld), האלגוריתם מוכיח את קיומם של פולינום המנה ופולינום השארית, והיחידות נובעת מהשוויון P - Q ⋅ G = R ומהעובדה ש- deg R < deg G
      10
      שני הנתונים מחייבים ש- deg P = deg ( Q ⋅ G ) ושהמקדם המוביל ב- Q ⋅ G הוא הנגדי של המקדם המתאים ב- P , כלומר הדרגה של Q והמקדם המוביל שלו נקבעים ביחידות ע”י P ו- G , לאחר מכן המקדם הזה קובע ביחידות את המקדם הבא בתור וכן הלאה (ממש כפי שעובד האלגוריתם).
      .
    3. ♣ זהו אלגוריתם דומה מאד לאלגוריתם “ארוך חילוק” שלמדנו בבית הספר היסודי
      11
      למעשה אלגוריתם “חילוק ארוך” הוא מקרה פרטי של חילוק פולינומים כאשר מציבים במשתנה את 10 (או, אם תרצו, זהו חילוק פולינומים ב- R [ 10 ] ).
      .

    אלגוריתם 1 חילוק פולינומים עם שארית
    יהיו P , G ∈ F [ x ] כך ש- G ≠ 0 , נרצה למצוא שני פולינומים Q , R ∈ F [ x ] כך ש- P = Q ⋅ G + R ו- deg R < deg G .
    נסמן R 0 : = P ו- Q 0 = 0 , א”כ מתקיים P = Q 0 ⋅ G + R 0 .
    נסמן n : = deg G ויהיו a 0 , a 1 , … , a n ∈ F כך שמתקיים: G ( x ) = ∑ k = 0 n a k ⋅ x k
    נסמן i : = 0 וכל עוד deg R i ≥ deg G :
    • נסמן r : = deg R i ויהיו b 0 , b 1 , … , b m ∈ F כך שמתקיים: R i ( x ) = ∑ j = 0 r b j ⋅ x j
    • נגדיר (מהגדרה a n ≠ 0 ): q r - n : = b r a n Q i + 1 ( x ) : = Q i ( x ) + q r - n ⋅ x r - n R i + 1 ( x ) : = R i ( x ) - q r - n ⋅ x r - n ⋅ G ( x ) נשים לב לשני דברים:
      1. P ( x ) = Q i ( x ) ⋅ G ( x ) + R i ( x ) = ⋅ G ( x ) + ⋅ G ( x ) + R i + 1 ( x ) = Q i + 1 ( x ) ⋅ G ( x ) + R i + 1 ( x )
      2. המקדם המוביל של q r - n ⋅ x r - n ⋅ G ( x ) הוא b r ו- deg ( q r - n ⋅ x r - n ⋅ G ( x ) ) = ( r - n ) + deg G = r = deg R i ,
        ומשכאן ש- deg R i + 1 < deg R i .
    טענה 1.14. יהיו P , G ∈ F [ x ] כך ש- G ≠ 0 , G ∣ P אם”ם השארית של חלוקת P ב- G היא פולינום האפס.
    טענה 1.15. יהיו P , G ∈ F [ x ] , לכל 0 ≠ F ∈ F [ x ] מתקיים P ∣ G אם”ם F ⋅ P ∣ F ⋅ G .
    הוכחה. יהי 0 ≠ F ∈ F [ x ] , ההוכחה שאם P ∣ G אז F ⋅ P ∣ F ⋅ G היא טריוויאלית, נוכיח את הכיוון השני.
    נניח ש- F ⋅ P ∣ F ⋅ G ונחלק את G ב- P עם שארית: יהיו Q , R ∈ F [ x ] כך ש- G = Q ⋅ P + R ו- deg R < deg P ,
    מכאן שמתקיים F ⋅ G = Q ⋅ F ⋅ P + F ⋅ R ו- deg ( F ⋅ R ) < deg ( F ⋅ P ) .
    מיחידות השארית נובע ש- F ⋅ R היא השארית של חלוקת F ⋅ G ב- F ⋅ P , א”כ F ⋅ R = 0 ומכיוון ש- F ≠ 0 נדע ש- R = 0 ולכן P ∣ G .

2 מחלק משותף מקסימלי וכפולה משותפת מינימלית

    יהי F שדה.
    ‫הגדרה 2.1. פולינום P ∈ F [ x ] יקרא פולינום מתוקן אם המקדם המוביל שלו הוא 1 , פולינום האפס אינו נחשב מתוקן.
    משפט 2.2: יהיו P 1 , P 2 , … , P r ∈ F [ x ] , מתקיימים שני הפסוקים הבאים:
    • אם קיים r ≥ i ∈ N כך ש- P i ≠ 0 אז קיים פולינום מתוקן D ∈ F [ x ] יחיד כך ש- D ∣ P i לכל r ≥ i ∈ N ובנוסף לכל Q ∈ F [ x ] המחלק את כולם ( Q ∣ P i לכל r ≥ i ∈ N ) מתקיים Q ∣ D .
    • אם P i ≠ 0 לכל r ≥ i ∈ N אז קיים פולינום מתוקן L ∈ F [ x ] כך ש- P i ∣ L לכל r ≥ i ∈ N ובנוסף לכל M ∈ F [ x ] המתחלק בכולם ( P i ∣ M לכל r ≥ i ∈ N ) מתקיים L ∣ M .
    ‫הגדרה 2.3. מחלק משותף מקסימלי וכפולה משותפת מינימלית
    יהיו P 1 , P 2 , … , P r ∈ F [ x ] ,
    • אם קיים r ≥ i ∈ N כך ש- P i ≠ 0 אז נסמן ב- gcd ( P 1 , P 2 , … , P r ) את אותו D יחיד שמקיים את התנאים במשפט שלעיל, ונקרא לו המחלק המשותף המקסימלי של P 1 , P 2 , … , P r .
    • אם P i ≠ 0 לכל r ≥ i ∈ N אז נסמן ב- lcm ( P 1 , P 2 , … , P r ) את אותו M יחיד שמקיים את התנאים במשפט שלעיל ונקרא לו הכפולה המשותפת המינימלית של P 1 , P 2 , … , P r .
    1. ♣ הגדרות שקולות ל- gcd ול- lcm הן:
      • המחלק המשותף המקסימלי הוא הפולינום המתוקן המחלק את P 1 , P 2 , … , P r שדרגתו היא הגבוהה ביותר מבין אלה שמחלקים את כולם.
      • הכפולה המשותפת המינימלית היא הפולינום המתוקן המתחלק ב- P 1 , P 2 , … , P r שדרגתו היא הנמוכה ביותר מבין אלה שמתחלקים בכולם.
    משפט 2.4: יהיו P , G ∈ F [ x ] .
    יהיו Q 1 , Q 2 , … , Q r ∈ F [ x ] , n 1 , n 2 , … , n r , m 1 , m 2 , … , m r ∈ N 0 ו- a , b ∈ F כך שהפירוקים של P ו- Q לגורמים אי-פריקים הם
    12
    אם Q i הוא גורם של P אך אינו גורם של G אז m i = 0 , ולהיפך, אם Q i הוא גורם של G אך לא של P אז n i = 0 .
    : P = ∏ i = 1 r ( Q i ) n i G = ∏ i = 1 r ( Q i ) m i מתקיים: gcd ( P , G ) = ∏ i = 1 r ( Q i ) min { n i , m i } lcm ( P , G ) = ∏ i = 1 r ( Q i ) max { n i , m i }
    ‫מסקנה 2.5. יהיו P , G ∈ F [ x ] , מתקיים: lcm ( P , G ) = P ⋅ G gcd ( P , G )
    טענה 2.6. יהיו P , G , Q , R ∈ F [ x ] כך ש- P = Q ⋅ G + R , מתקיים gcd ( P , G ) = gcd ( G , R ) .
    טענה 2.7. יהיו P , G ∈ F [ x ] .
    1. gcd ( F , G ) = gcd ( G , F ) .
    2. לכל Q ∈ F [ x ] מתקיים Q ⋅ gcd ( F , G ) = gcd ( Q ⋅ F , Q ⋅ G ) .
    3. אם קיים Q ∈ F [ x ] כך ש- Q ∣ F ⋅ G וגם gcd ( G , Q ) = 1 אז Q ∣ F .
    ‫הגדרה 2.8. פולינומים זרים
    נאמר ששני פולינומים P , G ∈ F [ x ] זרים זה לזה אם gcd ( P , G ) = 1 , כלומר אין להם מחלק שאינו טריוויאלי.
    ‫הגדרה 2.9. פולינומים ידידים
    נאמר ששני פולינומים P , G ∈ F [ x ] הם פולינומים ידידים אם P ∣ G וגם G ∣ P .
    1. ♣ מהגדרה אם P , G ∈ F [ x ] הם פולינומים ידידים אז קיים c ∈ F [ x ] כך ש- P = c ⋅ G ולכן בהכרח deg P = deg G , אותו c הוא המנה בין שני המקדמים המובילים ולכן אם נתון בנוסך ששניהם מתוקנים אז הם שווים.
    ‫הגדרה 2.10. אידיאל
    קבוצה I ⊆ F [ x ] תקרא אידיאל אם מתקיימים שלושת התנאים הבאים:
    1. 0 ∈ I .
    2. לכל P , G ∈ I מתקיים P + G ∈ I .
    3. לכל P ∈ I ולכל Q ∈ F [ x ] מתקיים Q ⋅ P ∈ I .
    1. ♣ למעשה זהו מקרה פרטי, בכל חוג חילופי קיימים אידאלים וזוהי ההגדרה שלהם.
    משפט 2.11: יהי I ⊆ F [ x ] אידיאל, מתקיימת אחת משתי האפשרויות הבאות: I = { 0 } או שקיים פולינום מתוקן יחיד P ∈ F [ x ] כך שמתקיים I = { Q ⋅ P ∣ Q ∈ F [ x ] } .
    הוכחה. נניח ש- I ≠ { 0 } , יהי P ∈ I פולינום מתוקן מדרגה מינימלית, כלומר לכל 0 ≠ G ∈ I מתקיים deg P ≤ deg G .
    יהי G ∈ I פולינום ויהיו Q , R ∈ F [ x ] כך ש- G = Q ⋅ P + R ו- deg R < deg P , מהגדרת האידיאל נובע ש- R ∈ I ולכן מהגדרת P נובע ש- R = 0 ומכאן ש- G = Q ⋅ P .
