2 דיפרנציאביליות ונקודות קיצון

    גם בפרק זה אנחנו נעסוק רק בפונקציות מהצורה f : D → R כאשר D ⊆ R n ולכן לא אטרח לציין זאת בכל פעם.
    1. ♣ ההגדרה של נקודות קיצון (כלליות ומקומיות) מופיעה בקובץ על סדרות ופונקציות.
    ‫הגדרה 2.1. נקודות קריטיות וסינגולריות
    תהא f פונקציה, נקודה פנימית P ∈ R n בתחום ההגדרה תיקרא נקודה קריטית אם כל הנגזרות החלקיות מתאפסות בה, ותיקרא נקודה סינגולרית אם אחת הנגזרות החלקיות אינה מוגדרת בה.
    1. ♣ הרעיון הוא כמו באינפי’ 1: נקודות קריטיות ונקודות סינגולריות הן היחידות שעלולות להיות נקודות קיצון.
    ‫הגדרה 2.2. נקודת אוכף
    תהא f פונקציה, נקודה P ∈ R n בתחום ההגדרה תקרא נקודת אוכף אם כל הנגזרות החלקיות מתאפסות בה אך היא אינה נקודת קיצון.
    1. ♣ נקודת אוכף היא המקבילה של נקודת פיתול שבה הנגזרת מתאפסת, השם נובע מצורת ה”אוכף” שנוצרת במקרה כזה (להמחשה לחצו כאן).
    ‫הגדרה 2.3. נגזרות חלקיות מעורבות
    תהא f פונקציה, לכל n ≥ i , j ∈ N ולכל P ∈ R n נקודה בתחום ההגדרה נסמן D i j f ( P ) : = D i , j f ( P ) : = D i ( D j f ) ( P ) .
    1. ♣ מדובר בסימון בלבד, אף אחד לא מבטיח לנו שהביטוי מוגדר בכלל.
    2. ♣ הנגזרות הנ”ל הן נגזרות חלקיות מעורבות מסדר שני, אפשר להמשיך ולגזור גם אותן ולקבל נגזרות מעורבות מסדר שלישי: D i j k f ( P ) : = D i ( D j k f ( P ) ) = D i ( D j ( D k f ) ( P ) ) , וכן עבור נגזרות מעורבות מסדר רביעי וכו’.
