1 סדרות

    תהא ( P n ) n = 1 ∞ סדרת נקודות ב- R k ונסמן ב- x n j את הקואורדינטה ה- j של הנקודה P n (לכל k ≥ j ∈ N ולכל n ∈ N ).
    טענה 1.1. ( P n ) n = 1 ∞ היא סדרה חסומה (ב- R k ) אם”ם לכל k ≥ j ∈ N הסדרה ( x n j ) n = 1 ∞ היא סדרה חסומה (ב- R ).
    טענה 1.2. אם ( P n ) n = 1 ∞ מתכנסת אז היא גם חסומה.
    טענה 1.3. ( P n ) n = 1 ∞ היא סדרה מתכנסת (ב- R k ) אם”ם לכל k ≥ j ∈ N הסדרה ( x n , j ) n = 1 ∞ היא סדרה מתכנסת (ב- R ).
    משפט 1.4: קבוצה U ⊆ R k היא קבוצה סגורה אם”ם לכל סדרת נקודות מתכנסת ( P n ) n = 1 ∞ שכל איבריה ב- U מתקיים ש- lim n → ∞ P n ∈ U .
    משפט 1.5: קבוצה K ⊆ R k היא קבוצה קומפקטית אם”ם לכל סדרת נקודות ( P n ) n = 1 ∞ שכל איבריה ב- K קיימת תת-סדרה מתכנסת שגבולה ב- K .
    משפט 1.6: משפט בולצאנו-ויירשטראס
    אם ( P n ) n = 1 ∞ היא סדרה חסומה אז יש לה תת-סדרה מתכנסת.

2 פונקציות

    טענה 2.1. אם לפונקציה f : D → R n יש גבול בנקודה P 0 ∈ D ⊆ R m אז קיימת סביבה מנוקבת U ⊆ D של P 0 כך ש- f ( U ) היא קבוצה חסומה.
    משפט 2.2: תנאי היינה לגבול של פונקציה בנקודה
    תהא f : D → R n פונקציה המוגדרת בסביבה מנוקבת U של נקודה P 0 ∈ R m ( D ⊆ R m ).
    תנאי הכרחי ומספיק לכך ש- lim P → P 0 f ( P ) = L ∈ R n הוא שלכל סדרת נקודות ( P n ) n = 1 ∞ ב- U המתכנסת ל- P 0 מתקיים lim n → ∞ f ( P n ) = L .
    ‫מסקנה 2.3. תנאי היינה לרציפות של פונקציה בנקודה
    תהא f : D → R n פונקציה המוגדרת בסביבה מלאה U של נקודה P 0 ∈ R m ( D ⊆ R m ).
    תנאי הכרחי ומספיק לכך ש- f רציפה ב- P 0 הוא שלכל סדרת נקודות ( P n ) n = 1 ∞ ב- U המתכנסת ל- P 0 מתקיים lim n → ∞ f ( P n ) = f ( P 0 ) .
    משפט 2.4: משפט ההצבה בגבולות
    תהיינה f : D 1 → D 2 ו- g : D 2 → R k (כאשר D 1 ⊆ R m ו- D 2 ⊆ R n ) ותהיינה P 0 ∈ D 1 ו- Q 0 ∈ D 2 נקודות פנימיות (בהתאמה).
    • אם ל- g יש גבול ב- Q 0 , ל- f יש גבול ב- P 0 ו- lim P → P 0 f ( P ) = Q 0 , ובנוסף קיימת סביבה מנוקבת U של P 0 כך שלכל P ∈ U מתקיים f ( P ) ≠ Q 0 , אז ל- g ○ f יש גבול ב- P 0 ומתקיים: lim P → P 0 g ( f ( P ) ) = lim Q → Q 0 g ( Q )
    • אם f ו- g רציפות ב- P 0 וב- Q 0 (בהתאמה) אז g ○ f רציפה ב- P 0 ומתקיים g ( f ( P 0 ) ) = g ( Q 0 ) .

    2.1 מסילות

    משפט 2.5: תהא γ : I → R n מסילה ותהיינה f 1 , f 2 , … , f n ∈ R I פונקציות כך שלכל n ≥ j ∈ N ולכל t ∈ I יתקיים: γ ( t ) = ( f 1 ( t ) f 2 ( t ) ⋮ f n ( t ) )
    תהא t 0 ∈ I נקודה פנימית ב- I , γ גזירה בנקודה t 0 ∈ I אם”ם לכל n ≥ j ∈ N הפונקציה f j גזירה ב- t 0 ואז וקטור הנגזרת של γ ב- t 0 הוא: [ f 1 ' ( t 0 ) f 2 ' ( t 0 ) ⋮ f n ' ( t 0 ) ]
    משפט 2.6: יהיו a , b ∈ R כך ש- a < b ותהיינה f , g : [ a , b ] → R פונקציות רציפות בקטע [ a , b ] וגזירות בקטע ( a , b ) כך ש- g ' ( t ) ≠ 0 לכל t ∈ ( a , b ) .
    תהא גם γ : [ a , b ] → R 2 מסילה המוגדרת ע”י (לכל t ∈ [ a , b ] ): γ ( t ) : = ( g ( t ) f ( t ) ) קיימת נקודה c ∈ ( a , b ) כך שמתקיים: g ( b ) - g ( a ) g ' ( c ) ⋅ γ ' ( c ) = γ ( b ) - γ ( a )
    1. ♣ כלומר משפט הערך הממוצע של קושי אומר שקיימת נקודה c ∈ ( a , b ) שבה שיפוע הישר המשיק למסלול של γ הוא בדיוק השיפוע של הקטע המחבר בין γ ( a ) ל- γ ( b ) .
    2. ♣ כדי להוכיח את המשפט יש להשתמש במשפט הערך הממוצע של קושי, ע”פ משפט הערך הממוצע של קושי נובע קיימת c ∈ ( a , b ) כך שמתקיים: f ' ( c ) g ' ( c ) = f ( b ) - f ( a ) g ( b ) - g ( a ) וממילא גם: g ( b ) - g ( a ) g ' ( c ) ⋅ f ' ( c ) = f ( b ) - f ( a ) ומכאן שע”פ המשפט הקודם (2.5) מתקיים: g ( b ) - g ( a ) g ' ( c ) ⋅ γ ' ( t 0 ) = g ( b ) - g ( a ) g ' ( c ) ⋅ [ f ' ( c ) g ' ( c ) ] = ( f ( b ) - f ( a ) g ( b ) - g ( a ) ) = γ ( b ) - γ ( a )

    2.2 פונקציות מרובות משתנים

    כל הפונקציות שנעסוק בהן בסעיף זה הן פונקציות מהצורה f : D → R כאשר D ⊆ R n ולכן לא אטרח לציין זאת בכל פעם מחדש.
    טענה 2.7. תהא f פונקציה כך של- f יש גבול L ∈ R בנקודה P 0 ∈ R n .
    אם 0 < L אז קיימת סביבה מנוקבת U של P 0 כך שלכל P ∈ U מתקיים L 2 < f ( P ) < 3 L 2 ,
    ואם L < 0 אז קיימת סביבה מנוקבת U של P 0 כך שלכל P ∈ U מתקיים 3 L 2 < f ( P ) < L 2 .
    ‫מסקנה 2.8. תהא f פונקציה כך של- f יש גבול L ∈ R בנקודה P 0 ∈ R m .
    אם 0 < L אז קיימת סביבה מנוקבת של P 0 שבה f חיובית ואם L < 0 אז קיימת סביבה מנוקבת של P 0 שבה f שלילית.
    משפט 2.9: “האנלוג הרציף של תנאי היינה”
    תהא f פונקציה המוגדרת בסביבה מנוקבת של נקודה P 0 ∈ R n , ל- f יש גבול L ∈ R ב- P 0 אם”ם לכל מסילה γ : [ a , b ] → R m המקיימת γ ( ( a , b ] ) ⊆ D ∖ { P 0 } ו- γ ( a ) = P 0 מתקיים: lim t → a + ( f ○ γ ) ( t ) = L
    1. ♣ נזכור שמהגדרתה מסילה היא פונקציה רציפה ולכן lim x → a + γ ( t ) = γ ( a ) = P 0 .
    2. ♣ נשים לב ש- f ○ γ היא פונקציה שהתחום שלה הוא ( a , b ] והטווח שלה הוא D ∖ { P 0 } .
    משפט 2.10: אריתמטיקה של גבולות
    תהיינה f ו- g פונקציות כך שקיימים הגבולות (עבור P 0 ∈ R n ): L 1 : = lim P → P 0 f ( P ) , L 2 : = lim P → P 0 g ( P ) מתקיימים ארבעת הפסוקים הבאים:
    1. לפונקציה f + g יש גבול בנקודה P 0 ןערכו הוא L 1 + L 2 .
    2. לפונקציה f ⋅ g יש גבול בנקודה P 0 וערכו הוא L 1 ⋅ L 2 .
    3. אם L 2 ≠ 0 אז לפונקציה 1 g יש גבול בנקודה P 0 וערכו הוא 1 L 2 .
    4. אם L 2 ≠ 0 אז לפונקציה f g יש גבול בנקודה P 0 וערכו הוא L 1 L 2 .
    ‫מסקנה 2.11. אריתמטיקה של רציפות
    תהיינה f ו- g פונקציות רציפות בנקודה P 0 ∈ R n .
    מתקיימים ארבעת הפסוקים הבאים:
    1. הפונקציה f + g רציפה ב- P 0 ו- ( f + g ) ( P 0 ) = f ( P 0 ) + g ( P 0 ) .
    2. הפונקציה f ⋅ g רציפה ב- P 0 ו- ( f ⋅ g ) ( P 0 ) = f ( P 0 ) ⋅ g ( P 0 ) .
    3. אם g ( P 0 ) ≠ 0 אז הפונקציה 1 g רציפה ב- P 0 ו- ( 1 g ) ( P 0 ) = 1 g ( P 0 ) .
    4. אם g ( P 0 ) ≠ 0 אז הפונקציה 1 g רציפה ב- P 0 ו- ( f g ) ( P 0 ) = f ( P 0 ) g ( P 0 ) .
    משפט 2.12: תהיינה f ו- g פונקציות כך שקיימים הגבולות (עבור P 0 ∈ R n ): L 1 : = lim P → P 0 f ( P ) , L 2 : = lim P → P 0 g ( P )
    1. כלל חסומה ואפסה: אם L 1 = 0 ו- g חסומה בסביבה מנוקבת של P 0 אז ל- f ⋅ g יש גבול ב- P 0 והוא 0 .
    2. אם קיימת סביבה מנוקבת U של P 0 כך שלכל P ∈ U מתקיים f ( P ) ≤ g ( P ) אז L 1 ≤ L 2 .
    3. אם L 1 < L 2 אז קיימת סביבה מנוקבת U של P 0 כך שלכל P ∈ U מתקיים f ( P ) < g ( P ) .
    1. ♣ כפי שראינו באינפי’ 1 גם כאן א”ש חזק בין f ל- g בסעיף 2 לא יגרום לא”ש חזק בין הגבולות ובסעיף 3 א”א לוותר על הא”ש החזק.
    משפט 2.13: משפט הכריך
    תהיינה f , g ו- h פונקציות המוגדרות בסביבה מנוקבת U של נקודה P 0 ∈ R n .
    אם לכל P ∈ U מתקיים f ( P ) ≤ g ( P ) ≤ h ( P ) ובנוסף ל- f ול- h יש גבול בנקודה P 0 כך שמתקיים: L : = lim P → P 0 f ( P ) = lim P → P 0 h ( P ) אז גם ל- g יש גבול ב- P 0 ומתקיים: lim P → P 0 g ( P ) = L
    משפט 2.14: משפט ויירשטראס הראשון
    תהא f פונקציה המוגדרת בעל קבוצה קומפקטית ∅ ≠ K ⊆ R n , f חסומה ב- K .
    משפט 2.15: עיקרון המקסימום והמינימום של ויירשטראס (משפט ויירשטראס השני)
    תהא f פונקציה המוגדרת בעל קבוצה קומפקטית ∅ ≠ K ⊆ R n , f מקבלת מקסימום ומינימום ב- K (או במילים אחרות f ( K ) היא קטע סגור ב- R ).
