1 סדרות

    ‫הגדרה 1.1. סביבות סימטריות של נקודה
    תהא d מטריקה על R k ותהא P 0 ∈ R k , נסמן (לכל 0 < r ∈ R ): B r ( P 0 ) : = { P ∈ R k ∣ d 2 ( P , P 0 ) < r } B r ○ ( P 0 ) : = { P ∈ R k ∣ 0 < d 2 ( P , P 0 ) < r } B r ¯ ( P 0 ) : = { P ∈ R k ∣ d 2 ( P , P 0 ) ≤ r } B r ○ ¯ ( P 0 ) : = { P ∈ R k ∣ 0 < d 2 ( P , P 0 ) ≤ r }
    • B r ( P 0 ) תיקרא סביבה של P 0 (או כדור פתוח ברדיוס r שמרכזו בנקודה P 0 ) ו- B r ○ ( P 0 ) תיקרא סביבה מנוקבת של P 0 (או כדור פתוח מנוקב ברדיוס r שמרכזו בנקודה P 0 ).
    • B r ( P 0 ) ¯ תיקרא סביבה סגורה של P 0 (או כדור פתוח ברדיוס r שמרכזו בנקודה P 0 ) ו- B r ○ ( P 0 ) ¯ תיקרא סביבה סגורה מנוקבת של P 0 (או כדור פתוח מנוקב ברדיוס r שמרכזו בנקודה P 0 ).
    ‫הגדרה 1.2. קבוצה פתוחה, קבוצה סגורה וקבוצה קומפקטית
    • נאמר שקבוצה U ⊆ R k היא קבוצה חסומה אם קיימים P ∈ R k ו- 0 < r ∈ R כך ש- U ⊆ B r ( P ) .
    • נאמר שקבוצה U ⊆ R k היא קבוצה פתוחה אם לכל P ∈ U קיים 0 < r ∈ R כך ש- B r ( P ) ⊆ U .
    • נאמר שקבוצה U ⊆ R k היא קבוצה סגורה אם R k ∖ U היא קבוצה פתוחה.
    • נאמר שקבוצה K ⊆ R k היא קבוצה קומפקטית אם היא סגורה וחסומה.
    ‫הגדרה 1.3. שפה של קבוצה
    תהא D ⊆ R k קבוצה, נקודה P ∈ D תקרא נקודת שפה של D אם לכל סביבה U של P קיימות P 1 , P 2 ∈ U כך ש- P 1 ∈ D ו- P 2 ∉ D , קבוצת נקודות השפה של D תסומן ב- ∂ D .
    סימון:
    בהינתן סדרת נקודות ( P n ) n = 1 ∞ נסמן גם (לכל n ∈ N ): ( x n , 1 x n , 2 ⋮ x n , k ) : = P n
    ‫הגדרה 1.4. חסימות
    נאמר שסדרת נקודות ( P n ) n = 1 ∞ ב- R k היא סדרה חסומה אם קבוצת איבריה כזו.
    ‫הגדרה 1.5. גבול של סדרה
    תהא ( P n ) n = 1 ∞ סדרת נקודות ב- R k , נאמר שנקודה L ∈ R k היא גבול של הסדרה ( P n ) n = 1 ∞ אם לכל 0 < ε ∈ R קיים N ∈ N כך שלכל N < n ∈ N מתקיים P n ∈ B ε ( L ) .
    גבול של סדרה הוא יחיד ולכן מוצדק לדבר על הגבול של ( P n ) n = 1 ∞ ולסמן: lim n → ∞ P n : = L

2 פונקציות

    1. ♣ עד כה למדנו בקורסי אינפי’ על פונקציות שהתחום והטווח שלהן היו תתי-קבוצות של R (שהוא מ”ו מממד 1 ), בקורס זה נעסוק בעיקר בפונקציות מהצורה f : D → R n כאשר D ⊆ R m ובראש ובראשונה בפונקציות כנ”ל המקיימות n + m = 3 (יש בדיוק שתי אפשרויות) משום שאת הגרף של אלו ניתן לסרטט במרחב התלת-ממדי.
    ‫הגדרה 2.1. נאמר שנקודה P ∈ D ⊆ R n היא נקודה פנימית ב- D אם קיים 0 < r ∈ R כך ש- B r ( P ) ⊆ D .
    ‫הגדרה 2.2. נאמר שפונקציה f חסומה אם תמונתה היא קבוצה חסומה, כמו כן נאמר ש- f חסומה בקבוצה U (שמוכלת בתחום של f ) אם f ( U ) היא קבוצה חסומה.
    ‫הגדרה 2.3. גבול של פונקציה בנקודה
    תהא f : D → R n פונקציה כנ”ל המוגדרת בסביבה מנוקבת של נקודה P 0 ∈ R m , נאמר שנקודה (או וקטור) L ∈ R n היא גבול של f ב- P 0 אם לכל 0 < ε ∈ R קיימת 0 < δ ∈ R כך שלכל נקודה P ∈ B δ 0 ( P 0 ) מתקיים f ( P ) ∈ B ε ( L ) .
    גבול של פונקציה הוא יחיד ולכן מוצדק לדבר על הגבול של f ולסמן: lim P → P 0 f ( P ) : = L
    ‫הגדרה 2.4. רציפות של פונקציה
    תהא f : D → R n פונקציה כנ”ל, נאמר ש- f רציפה בנקודה P 0 ∈ D אם הגבול lim P → P 0 f ( P ) קיים ושווה ל- f ( P ) , נאמר ש- f רציפה אם f רציפה ב- P ' לכל P ' ∈ D .

    2.2 פונקציות מרובות משתנים

    ‫הגדרה 2.13. כשאנו אומרים “פונקציה מרובת משתנים” כוונתנו לפונקציה שהתחום שלה הוא קבוצת סדרות סופיות כלשהו (למשל R n ) שהרי מהגדרתה פונקציה אינה יכולה לקבל יותר ממשתנה אחד, הדרך “לעקוף” זאת היא לתת לה משתנה אחד (כגון סדרה) המכיל בתוכו יותר ממשתנה אחד; לעת עתה נעסוק בעיקר בפונקציות מהצורה f : D → R כאשר D ⊆ R 2 , הגרף של פונקציה מצורה זו נקרא יריעה.
    תהא f : D → R פונקציה כנ”ל, הגרף של f הוא תת-קבוצה של מרחב התלת-ממדי R 3
    3
    כאשר המישור הנפרש ע”י צי ה- x וציר ה- y ייצג את התחום ואילו הישר שיוצר ציר ה- z מייצג את הטווח.
    ומבחינה אינטואיטיבית הוא מהווה מפה תלת-ממדית של “פני השטח” הנוצרים ע”י הפונקציה (הפונקציה קובעת מה יהיה גובה פני הקרקע בכל נקודה במישור); אינטואיציה זו מאפשרת לנו לייצג את היריעה גם במישור כאשר בכל נקודה/אזור אנו כותבים מהו “גובה הפונקציה” בנקודה או באזור הללו, לייצוג כזה קוראים מפה טופוגרפית וכמתבקש לקבוצת הנקודות בגרף של f שכולן בגובה נתון נקרא קו גובה, כך למשל קו הגובה 3 יהיה הקבוצה { ( x , y , 3 ) ∈ R 3 ∣ 3 = f ( x , y ) } .

    ‫הגדרה 2.14. נקודות קיצון כלליות (גלובליות)
    תהא f : D → R פונקציה ותהא A ⊆ D , נאמר שנקודה P ∈ A היא נקודת מקסימום/מינימום כללית (גלובלית) של f ב- A אם לכל נקודה P ' ∈ D מתקיים f ( P ' ) ≤ f ( P ) (מקסימום) או f ( P ' ) ≥ f ( P ) (מינימום) ובמקרה כזה הערך f ( P ) יקרא הערך המקסימלי/המינימלי של f ב- A .
    ‫הגדרה 2.15. נקודות קיצון מקומיות
    תהא f פונקציה המוגדרת בסביבה מלאה של נקודה P ∈ R n , נאמר ש- P היא נקודת מקסימום/מינימום מקומית של f אם קיימת סביבה מלאה של P שבה P היא נקודת מקסימות כללית של f .
