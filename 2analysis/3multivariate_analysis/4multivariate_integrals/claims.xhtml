1 אינטגרביליות על מלבן

    1.1 תכונות בסיסיות של אינטגרביליות

    יהי R : = [ a , b ] × [ c , d ] ⊆ R 2 מלבן ותהיינה f , g : R → R שתי פונקציות אינטגרביליות ב- R .
    משפט 1.1: תכונות בסיסיות של אינטגרביליות
    1. אם f אי-שלילית אז גם 0 ≤ ∬ R f d A .
    2. אם f ( x ) ≥ g ( x ) לכל x ∈ R אז גם ∬ R f d A ≥ ∬ R g d A .
    3. הפונקציות f ± g אינטגרביליות ב- R ומתקיים: ∬ R f ± g d A = ∬ R f d A ± ∬ R g d A
    4. לכל λ ∈ R הפונקציה λ ⋅ f אינטגרבילית ב- R ומתקיים: ∬ R λ ⋅ f d A = λ ⋅ ∬ R f d A
    הפונקציה f ⋅ g אינטגרבילית ב- R .
    טענה 1.2. אם קיים 0 < m ∈ R כך ש- m ≤ | g ( x ) | לכל x ∈ R אז הפונקציה 1 g אינטגרבילית ב- R .
    ‫מסקנה 1.3. אם קיים 0 < m ∈ R כך ש- m ≤ | g ( x ) | לכל x ∈ R אז הפונקציה f g אינטגרבילית ב- R .
    טענה 1.4. אם f ( x ) ≡ 1 אז ∬ R f d A = A ( R ) .
    משפט 1.5: הפונקציה | f | אינטגרבילית ב- R ומתקיים: | ∬ R f d A | ≤ ∬ R | f | d A
    1. ♣ נשים לב לדמיון לא“ש המשולש, גם כאן הערך המוחלט של ה“סכום“ קטן או שווה לסכום של הערכים המוחלטים ומבחינה אינטואיטיבית זה קורה מאותה סיבה.
    טענה 1.6. יהי R ' ⊆ R מלבן, f אינטגרבילית ב- R ' .
    טענה 1.7. יהי γ ∈ ( a , b ) , נסמן R 1 : = [ a , γ ] × [ c , d ] ו- R 2 : = [ γ , b ] × [ c , d ] ותהא h פונקציה אינטגרבילית על R 1 ועל R 2 , h אינטגרבילית על R = R 1 ∪ R 2 ומתקיים: ∬ R h d A = ∬ R 1 h d A + ∬ R 2 h d A

    1.2 משפט פוביני ופונקציות רציפות

    יהי R : = [ a , b ] × [ c , d ] ⊆ R 2 מלבן ותהא f : R → R .
    משפט 1.8: אם f רציפה אז f אינטגרבילית ב- R .
    משפט 1.9: משפט פוביני
    1
    ערך בוויקיפדיה: פוביני גווידו.
    • אם f אינטגרבילית ב- R ולכל y ∈ [ c , d ] קיים האינטגרל ∫ a b f ( x , y ) d x אז קיים האינטגרל הנשנה ∫ c d ( ∫ a b f ( x , y ) d x ) d y ומתקיים: ∫ c d ( ∫ a b f ( x y ) d x ) d y = ∬ R f d A
    • אם f אינטגרבילית ב- R ואם לכל x ∈ [ a , b ] קיים האינטגרל ∫ c d f ( x , y ) d x אז קיים האינטגרל הנשנה ∫ a b ( ∫ c d f ( x , y ) d x ) d y ומתקיים: ∫ a b ( ∫ c d f ( x , y ) d x ) d y = ∬ R f d A
    1. ♣ הרעיון הוא שאנחנו “פורסים” את הנפח המבוקש לפרוסות דקות וסוכמים אותן.
    2. ♣ לכתוב על הקשר בין משפט פוביני למשפט שוורץ (הנוסחה היסודית).
    ‫מסקנה 1.10. אם f אינטגרבילית ב- R ובנוסף לכל y ∈ [ c , d ] קיים האינטגרל ∫ a b f ( x , y ) d x ולכל x ∈ [ a , b ] קיים האינטגרל ∫ c d f ( x , y ) d x אז מתקיים: ∫ c d ( ∫ a b f ( x y ) d x ) d y = ∬ R f d A = ∫ a b ( ∫ c d f ( x , y ) d x ) d y
    1. ♣ זה דומה מאד לעובדה שמתקיים
      2
      שהרי לא משנה באיזה סדר סוכמים איברים בטבלה.
      : ∑ i = 1 n ∑ j = 1 m x i , j = ∑ j = 1 m ∑ i = 1 n x i , j רק שכאן הסכימה רציפה ולא בדידה.
    טענה 1.11. אם f רציפה אז לכל y הפונקציה g : [ a , b ] → R המוגדרת ע”י g ( x ) : = f ( x y ) רציפה וכמו כן גם הפונקציה h : [ a , b ] → R המוגדרת ע”י h ( x ) : = f ( x y ) רציפה.
    ‫מסקנה 1.12. אם f רציפה אז מתקיים: ∫ c d ( ∫ a b f ( x y ) d x ) d y = ∬ R f d A = ∫ a b ( ∫ c d f ( x , y ) d x ) d y

3 החלפת משתנים

    משפט 3.1: הצבה
    תהא g : R → R ( R ⊆ R 2 קבוצה חסומה שאינה בהכרח מלבן), תהא D ⊆ R 2 קבוצה חסומה, תהיינה x , y : D → R ותהא f : D → R 2 המוגדרת ע”י (לכל ( s , t ) ∈ D ): f ( s t ) : = [ x ( s , t ) y ( s , t ) ] כך ש- Im ( f ) = R . מתקיים: ∬ R g d A = ∬ D ( g ○ f ) ⋅ | J f | d A
    1. ♣ הרעיון הוא בדיוק כמו במשפט ההצבה של האינטגרל המסוים באינפי’ 2: אנחנו מציבים בתוך הפונקציה שעליה מתבצעת האינטגרציה פונקציה פנימית אבל מכיוון שהאחרונה מעוותת את המרחב עלינו לבצע תיקון, וזה מתבצע ע”י כפל בדטרמיננטה של ההעתקה הליניארית המהווה את הדיפרנציאל של הפונקציה הפנימית בכל נקודה
      5
      כלומר | J f | היא בעצמה פונקציה המוגדרת ע”י הדטרמיננטה של J f בנקודה.

    2. ♣ לא הבאנו כאן את התנאים למשפט מפני שלא למדנו אותם בכיתה, בספר שנועה המליצה עליו (היא קראה לו “הספר של אנטון”) נכתב שהתנאים הם ש- f חח”ע, g רציפה, הנגזרות החלקיות של x ו- y בכל נקודה ב- D רציפות בנקודה זו (כלומר f דיפרנציאבילית ב- D ) ו-“התחומים R ו- D אינם מסובכים מדי”.
    תהא h : [ 0 , ∞ ) × [ 0 , 2 π ) → R 2 הפונקציה המעבירה מהצגה קוטבית להצגה קרטזית
