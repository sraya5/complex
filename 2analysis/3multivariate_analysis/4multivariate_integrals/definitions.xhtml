1 אינטגרביליות על מלבן

    1.1 מושגים בסיסיים

    ‫הגדרה 1.1. מלבן:
    מלבן ב- R 2 הוא קבוצה מהצורה [ a , b ] × [ c , d ] כאשר a , b , c , d ∈ R ; הצגה ברורה יותר של הקבוצה היא: R : = [ a , b ] × [ c , d ] = { ( x y ) ∈ R 2 a ≤ x ≤ b , c ≤ y ≤ d } את השטח של המלבן (שהוא ( b - a ) ( c - d ) ) נסמן ב- A ( R ) .
    ‫הגדרה 1.2. שריג
    יהי R ⊆ R 2 מלבן, שריג על R הוא זוג סדור P : = ( P 1 , P 2 ) כאשר P 1 היא חלוקה של [ a , b ] ו- P 2 היא חלוקה של [ c , d ] .
    1. ♣ שריג הוא המושג המקביל של חלוקה מאינפי’ 2.
    ‫הגדרה 1.3. עידון של שריג
    יהי R מלבן, עידון של שריג P : = ( P 1 , P 2 ) הוא כל שריג Q : = ( Q 1 , Q 2 ) על R המקיים Q 1 ⊆ P 1 ו- Q 2 ⊆ P 2 .

    1.2 סכומי דארבו

    יהי R ⊆ R 2 מלבן, יהי P שריג על R , יהיו R 1 , R 2 , … , R n ⊆ R כל תתי המלבנים שמגדירה P ותהא f : R → R פונקציה חסומה.
    ‫הגדרה 1.4. סכומי דארבו
    1
    ערך בוויקיפדיה: דארבו ז'אן גסטון.

    נסמן (לכל n ≥ i ∈ N ): M i : = sup { f ( x ) ∣ x ∈ R i } , m i : = inf { f ( x ) ∣ x ∈ R i } ונגדיר: U ( f , P ) : = ∑ i = 1 n M i ⋅ A ( R i ) , L ( f , P ) : = ∑ i = 1 n m i ⋅ A ( R i ) ל- U ( f , P ) נקרא סכום דארבו העליון של f עבור P ול- L ( f , P ) נקרא סכום דארבו התחתון של f עבור P .
    טענה 1.5. יהי Q עידון של P , מתקיים: L ( f , P ) ≤ L ( f , Q ) ≤ U ( f , Q ) ≤ U ( f , P )
    טענה 1.6. יהיו m , M ∈ R כך שלכל y ∈ Im f מתקיים m ≤ y ≤ M , לכל שני שריגים Q ו- Q ˜ על R מתקיים: m ⋅ A ( R ) ≤ L ( f , Q ) ≤ U ( f , Q ˜ ) ≤ M ⋅ A ( R )
    נסמן ב- L את קבוצת סכומי דרבו התחתונים של f עבור כל השריגים על R וב- U את קבוצת סכומי דרבו העליונים של f עבור כל השריגים על R , כפי שראינו לעיל אלו קבוצות חסומות.
    ‫הגדרה 1.7. אינטגרל עליון ואינטגרל תחתון
    • האינטגרל העליון של f ב- R הוא: ∬ R ¯ f d A : = inf ( U )
    • האינטגרל התחתון של f ב- R הוא: ∬ R ¯ f d A : = sup ( L )
    ‫מסקנה 1.8. מתקיים: ∬ R ¯ f d A ≤ ∬ R ¯ f d A
    ‫הגדרה 1.9. אינטגרביליות
    נאמר ש- f אינטגרבילית ב- R אם מתקיים: ∬ R ¯ f d A = ∬ R ¯ f d A ואז נסמן את הערך המשותף ב- ∬ R f d A ונקרא לו האינטגרל של f ב- R .
    1. ♣ למת החתכים נותנת תנאים שקולים לקיום השוויון.
    ‫הגדרה 1.10. נפח
    אם f אי-שלילית נאמר ש- ∬ R f d A הוא הנפח של הקבוצה: { ( x y z ) ∈ R 3 ( x y ) ∈ R , 0 ≤ z ≤ f ( x y ) }

2 אינטגרביליות על קבוצה כללית

    ‫הגדרה 2.1. אינטגרביליות על קבוצה שאינה מלבן
    תהיינה D ⊆ R 2 קבוצה חסומה ו- f : D → R פונקציה חסומה ויהי R ⊆ R 2 מלבן כך ש- D ⊆ R , נאמר ש- f אינטגרבילית על D אם הפונקציה המוגדרת ע”י (לכל ( x , y ) ∈ R ): g ( x y ) : = { f ( x , y ) ( x , y ) ∈ D 0 ( x , y ) ∉ D אינטגרבילית ב- R ובמקרה כזה נאמר שהאינטגרל של f ב- D הוא: ∬ D f d A = ∬ R g d A
    1. ♣ נשים לב שלכל מלבן המקיים D ⊆ R נקבל את אותו אינטגרל ולכן האינטגרל מוגדר היטב.
    ‫הגדרה 2.2. קבוצה בעלת שטח
    תהא D ⊆ R 2 קבוצה חסומה, נאמר ש- D בעלת שטח אם χ D (הפונקציה המציינת של D ) אינטגרבילית ב- D ואז נגדיר את השטח של D ע”י: A ( D ) : = ∬ D χ D d A
    ‫הגדרה 2.3. קבוצה נורמלית
    תהא D ⊆ R 2 קבוצה.
    • נאמר ש- D נורמלית ביחס לציר ה- x אם קיים קטע סגור I ⊆ R וקיימות שתי פונקציות g 1 , g 2 : I → R כך שמתקיים D = { ( x , y ) ∈ R 2 ∣ x ∈ I , g 1 ( x ) ≤ y ≤ g 2 ( x ) }
    • נאמר ש- D נורמלית ביחס לציר ה- y אם קיים קטע סגור I ⊆ R וקיימות שתי פונקציות g 1 , g 2 : I → R כך שמתקיים D = { ( x , y ) ∈ R 2 ∣ y ∈ I , g 1 ( x ) ≤ x ≤ g 2 ( x ) }
    • נאמר ש- D נורמלית בהצגה קוטבית אם קיים קטע סגור I ⊆ [ 0 , 2 π ] וקיימות שתי פונקציות g 1 , g 2 : I → R כך שמתקיים D = { ( r cos θ , r sin θ ) ∈ R 2 ∣ θ ∈ I , 0 ≤ g 1 ( x ) ≤ r ≤ g 2 ( x ) }
    1. ♣ הרעיון מאחורי ההגדרה של קבוצה נורמלית הוא שקבוצות נורמליות הן “כמעט” מלבנים
      2
      בהצגה קוטבית המלבנים הם קבוצות מהצורה { ( r cos θ , r sin θ ) ∈ R 2 ∣ a ≤ r ≤ b , c ≤ θ ≤ d } .

3 החלפת משתנים

    להוסיף כאן אינטואיציה להצבה במשתנה אחד - העיוות של הישר הממשי/המרחב ע”י הפונקציה והתיקון הנדרש.
    תהא D ⊆ R 2 קבוצה חסומה, תהיינה x , y : D → R ותהא f : D → R 2 המוגדרת ע”י (לכל ( s , t ) ∈ D ): f ( s t ) : = [ x ( s , t ) y ( s , t ) ]
    1. ♣ מטריצת יעקובי של f בנקודה פנימית P ∈ D היא המטריצה הבאה: J f ( P ) : = [ x s ' ( P ) x t ' ( P ) y s ' ( P ) y t ' ( P ) ] אם כל הנגזרות החלקיות המופיעות במטריצה רציפות ב- P אז כפי שלמדנו הדבר גורר ש- f דיפרנציאבילית ב- P ויתרה מזאת מטריצת יעקובי היא המטריצה המייצגת (בבסיס הסטנרדטי) של ההעתקה הליניארית המהווה את הדיפרנציאל של f ב- P , הדטרמיננטה של מטריצה יעקובי בנקודה P נקראת היעקוביאן של f ב- P .
