    ‫הגדרה 1.2. אוסף כל הפונקציות הקדומות של פונקציה נתונה f נקרא האינטגרל הלא מסוים של f ומסומן ב- ∫ f ( x ) d x .

2 האינטגרל המסוים (שטחים)

    2.1 אינטגרביליות לפי רימן

    יהיו a , b ∈ R כך ש- a < b ותהא f : [ a , b ] → R .
    ‫הגדרה 2.1. נאמר שקבוצה סופית P ⊆ [ a , b ] היא חלוקה של הקטע [ a , b ] אם a , b ∈ P .
    לכל חלוקה P = { x 0 , x 1 , … , x n } של [ a , b ] כך ש- a = x 0 < x 1 < … < x n = b נסמן λ ( P ) : = max { x i - x i - 1 ∣ n ≥ i ∈ N } ונכנה את λ ( P ) בשם פרמטר החלוקה של P .
    נאמר שקבוצה P * : = { t 1 , t 2 , … , t n } היא בחירת נקודות עבור חלוקה P : = { x 0 , x 1 , … , x n } ⊆ [ a , b ] אם t i ∈ [ x i - 1 , x i ] לכל n ≥ i ∈ N 0 .
    ‫הגדרה 2.2. סכום רימן
    תהא P : = { x 0 , x 1 , … , x n } חלוקה של [ a , b ] .
    סכום רימן של f עבור החלוקה P ובחירת הנקודות P * הוא כל סכום מהצורה: ∑ i = 1 n f ( t i ) ⋅ ( x i - x i - 1 ) כאשר { t 1 , t 2 , … , t n } היא בחירת נקודות עבור P .
    ‫הגדרה 2.3. אינטגרביליות לפי רימן
    נאמר ש- f אינטגרבילית רימן על הקטע [ a , b ] אם קיים I ∈ R כך שלכל 0 < ε ∈ R קיימת 0 < δ ∈ R כך שלכל חלוקה P של [ a , b ] המקיימת λ ( P ) < δ ולכל סכום רימן S של f עבור P מתקיים | S - I | < ε .
    טענה 2.4. אם f אינטגרבילית רימן על [ a , b ] אז קיים I ∈ R יחיד המקיים את ההגדרה.
    בגלל טענה זו יש משמעות לסימון (עבור אותו I ): ∫ a b f ( x ) d x : = I
    1. ♣ נשים לב כיצד ההגדרות הללו מפרמלות את מה שראינו בהקדמה האינטואיטיבית: סכום רימן הוא בעצם סכום של שטחי מלבנים שצלע אחת שלהם מונחת על ציר ה- x והצלע המקבילה לה חותכת את הגרף של f , הגדרת “אינטגרביליות רימן” אומרת שניתן לחשב את השטח שבין הגרף של f לציר ה- x
      2
      נשים לב לכך שאין מניעה ש- I יהיה שלילי.
      אם”ם כשמשאיפים את גדלי המלבנים ל- 0 מתקרבים תמיד לאותו מספר ואז אותו מספר הוא השטח.
    2. ♣ הסימון של האינטגרל מרמז על הגדרת האינטגרל המסוים של רימן: הסימן “ ∫ ” הוא כעין s (האות הראשונה של sum) ואנו סוכמים על מכפלות של ערכי הפונקציה ( f ( x ) ) באורכי הקטעים ( d x ); זוהי גם האינטואיציה מאחורי השיטה הלא פורמלית של אינטגרציה ע”י הצבה שראינו בפרק הקודם (בקובץ הטענות), שכן כפי שנראה כשנלמד על המשפט היסודי ישנו קשר הדוק בין האינטגרל המסוים לאינטגרל הלא מסוים.

    2.2 אינטגרביליות לפי דארבו

    סימון:
    נסמן ב- B [ a , b ] את קבוצת הפונקציות החסומות על קטע [ a , b ] וב- R [ a , b ] את קבוצת הפונקציות שאינטגרביליות רימן עליו.
    יהיו a , b ∈ R כך ש- a < b .
    תהיינה f ∈ B [ a , b ] ו- P : = { x 0 , x 1 , … , x n } חלוקה של [ a , b ] .
    ‫הגדרה 2.5. סכומי דארבו
    3
    ערך בוויקיפדיה: דארבו גסטון ז'אן.

    נסמן (לכל n ≥ i ∈ N ): M i ( f , P ) : = sup { f ( t ) ∣ t ∈ [ x i - 1 , x i ] } , m i ( f , P ) : = inf { f ( t ) ∣ t ∈ [ x i - 1 , x i ] } ונגדיר: U ( f , P ) : = ∑ i = 1 n M i ( f , P ) ⋅ ( x i - x i - 1 ) L ( f , P ) : = ∑ i = 1 n m i ( f , P ) ⋅ ( x i - x i - 1 ) ל- U ( f , P ) נקרא סכום דארבו העליון של f עבור P ול- L ( f , P ) נקרא סכום דארבו התחתון של f עבור P .
    1. ♣ כמובן שמהגדרה מתקיים L ( f , P ) ≤ U ( f , P ) .
    סימון:
    נסמן ב- σ ( f , P ) את קבוצת סכומי רימן של f עבור P .
    ‫למה 2.6. מתקיים U ( f , P ) = sup σ ( f , P ) וגם L ( f , P ) = inf σ ( f , P ) .
    סימון:
    נסמן ב- U ( f ) את קבוצת סכומי דארבו העליונים של f עבור חלוקות של [ a , b ] , וב- L ( f ) את קבוצת סכומי דארבו התחתונים של f עבור חלוקות של [ a , b ] .
    ‫הגדרה 2.7. אינטגרל עליון ואינטגרל תחתון
    לכל f ∈ B [ a , b ] נגדיר: I ¯ ( f ) : = inf U ( f ) I ¯ ( f ) : = sup L ( f ) ל- I ¯ ( f ) נקרא האינטגרל העליון של f ול- I ¯ ( f ) האינטגרל התחתון של f (בקטע [ a , b ] ).
    1. ♣ נשים לב לדמיון בין ההגדרות של אינטגרל עליון של פונקציה לבין זו של גבול עליון של סדרה וכמו כן בין ההגדרות של אינטגרל תחתון ושל גבול תחתון; בנוסף, כבר כעת ניתן לראות (אינטואיטיבית) שכמו שסדרה מתכנסת אם”ם הגבול העליון שלה שווה לתחתון גם פונקציה היא אינטגרבילית רימן אם”ם האינטגרל העליון שלה שווה לתחתון (נראה את המשפט הזה בהמשך).
    ‫הגדרה 2.8. אינטגרביליות לפי דארבו
    נאמר ש- f אינטגרבילית לפי דארבו על הקטע [ a , b ] אם מתקיים I ¯ ( f ) = I ¯ ( f ) , כלומר אם האינטגרל העליון שלה בקטע זה שווה לאינטגרל התחתון בקטע.

4 אינטגרלים מסוימים לא אמיתיים

    4.1 אינטגרל על קבוצה לא חסומה

    ‫הגדרה 4.1. אינטגרביליות על קרן
    תהא f פונקציה ויהי a ∈ R .
    • נאמר ש- f אינטגרבילית על הקרן [ a , ∞ ) אם f אינטגרבילית על כל תת-קטע סגור של [ a , ∞ ) ובנוסף קיים הגבול: lim b → ∞ ∫ a b f ( x ) d x במקרה כזה נסמן: ∫ a ∞ f ( x ) d x : = lim b → ∞ ∫ a b f ( x ) d x
    • נאמר ש- f אינטגרבילית על הקרן ( - ∞ , b ] אם f אינטגרבילית על כל תת-קטע סגור של ( - ∞ , b ] ובנוסף קיים הגבול: lim a → - ∞ ∫ b b f ( x ) d x במקרה כזה נסמן: ∫ - ∞ a f ( x ) d x : = lim a → - ∞ ∫ a b f ( x ) d x
    ‫למה 4.2. תהא f פונקציה. אם קיים a ∈ R כך שקיימים האינטגרלים: ∫ a ∞ f ( x ) d x , ∫ - ∞ a f ( x ) d x אז לכל c ∈ R קיימים האינטגרלים: ∫ c ∞ f ( x ) d x , ∫ - ∞ c f ( x ) d x ומתקיים: ∫ - ∞ a f ( x ) d x + ∫ a ∞ f ( x ) d x = ∫ - ∞ c f ( x ) d x + ∫ c ∞ f ( x ) d x
    ‫הגדרה 4.3. אינטגרביליות על כל הישר
    נאמר שפונקציה f אינטגרבילית על כל הישר אם קיים a ∈ R כך שקיימים האינטגרלים: ∫ - ∞ a f ( x ) d x , ∫ a ∞ f ( x ) d x ובמקרה כזה נסמן: ∫ - ∞ ∞ f ( x ) d x : = ∫ - ∞ a f ( x ) d x + ∫ a ∞ f ( x ) d x
    1. הערה: במקרה של אינטגרלים לא אמיתיים נאמר גם שהאינטגרל מתכנס/מתבדר אם הגבול המתאים קיים/לא קיים.
    ‫למה 4.4. תהא f פונקציה, יהי a ∈ R ונניח ש- f מקיימת f ∈ R [ a , b ] לכל b ∈ R ;
    אם האינטגרל ∫ a ∞ | f ( x ) | d x
    6
    ראינו ש- | f | אינטגרבילית רימן על כל קטע סגור ש- f אינטגרבילית רימן עליו.
    מתכנס אז גם האינטגרל ∫ a ∞ f ( x ) d x מתכנס,
    ואם האינטגרל ∫ - ∞ a | f ( x ) | d x מתכנס אז גם האינטגרל ∫ - ∞ a f ( x ) d x .
    ‫הגדרה 4.5. התכנסות בהחלט
    תהא f פונקציה ויהי a ∈ R .
    1. נאמר שהאינטגרל ∫ a ∞ f ( x ) d x מתכנס בהחלט אם האינטגרל ∫ a ∞ | f ( x ) | d x מתכנס, כמו כן נאמר שהאינטגרל ∫ - ∞ a f ( x ) d x מתכנס בהחלט אם האינטגרל ∫ - ∞ a | f ( x ) | d x מתכנס.
    2. אם ∫ a ∞ f ( x ) d x מתכנס אך אינו מתכנס בהחלט נאמר שהוא מתכנס בתנאי, כמו כן אם האינטגרל ∫ - ∞ a f ( x ) d x מתכנס אך אינו מתכנס בהחלט נאמר שהוא מתכנס בתנאי.

    4.2 אינטגרל של פונקציה לא חסומה

    יהיו a , b ∈ R כך ש- a < b ותהא f : [ a , b ] → R .

    נאמר שקבוצה { c 1 , c 2 , … , c n } ⊆ [ a , b ] היא קבוצת נקודות מיוחדות של f אם f אינטגרבילית רימן בכל תת-קטע סגור של [ a , b ] שאינו מכיל אף אחד מאיברי הקבוצה ובנוסף f אינה חסומה בכל סביבה של c i (לכל n ≥ ∈ N ).
    ‫הגדרה 4.7. אינטגרל של פונקציה לא חסומה
    נניח ש- a היא נקודה מיוחדת יחידה של f , נאמר שהאינטגרל (הלא אמיתי) ∫ a b f ( x ) d x מתכנס אם הגבול
    lim δ → 0 + ∫ a + δ b f ( x ) d x קיים ונסמן: ∫ a b f ( x ) d x : = lim δ → 0 + ∫ a + δ b f ( x ) d x כמו כן אם b היא נקודה מיוחדת יחידה של f נאמר שהאינטגרל ∫ a b f ( x ) d x מתכנס אם הגבול lim δ → 0 + ∫ a b - δ f ( x ) d x קיים ונסמן: ∫ a b f ( x ) d x : = lim δ → 0 + ∫ a b - δ f ( x ) d x אם a ו- b הן הנקודות המיוחדות היחידות של f אז נאמר ש- ∫ a b f ( x ) d x מתכנס אם קיים d ∈ ( a , b ) כך שהאינטגרלים ∫ d b f ( x ) d x ו- ∫ a d f ( x ) d x מתכנסים
    7
    אם קיים d אחד כזה אז כל d ∈ ( a , b ) מקיים זאת.
    ונסמן: ∫ a b f ( x ) d x : = ∫ a d f ( x ) d x + ∫ d b f ( x ) d x אם { c 1 , c 2 , … , c n } היא קבוצת נקודות מיוחדות של f המסודרת בסדר עולה והאינטגרלים ∫ a c 1 f ( x ) d x , ∫ c i c i + 1 f ( x ) d x ו- ∫ c n b f ( x ) d x מתכנסים לכל n - 1 ≥ i ∈ N אז נאמר שהאינטגרל (הלא אמיתי) ∫ a b f ( x ) d x מתכנס ונסמן: ∫ a b f ( x ) d x : = ∫ a c 1 f ( x ) d x + ∑ i = 1 n - 1 ∫ c i c i + 1 f ( x ) d x + ∫ c n b f ( x ) d x
    ‫למה 4.8. נניח של- f יש מספר סופי של נקודות מיוחדות ב- [ a , b ] , אם האינטגרל ∫ a b | f ( x ) | d x

    מתכנס אז גם האינטגרל ∫ a b f ( x ) d x מתכנס.
    ‫הגדרה 4.9. התכנסות בהחלט
    נניח של- f יש מספר סופי של נקודות מיוחדות ב- [ a , b ] .
    1. נאמר שהאינטגרל ∫ a b f ( x ) d x מתכנס בהחלט אם האינטגרל ∫ a b | f ( x ) | d x מתכנס.
    2. אם ∫ a b f ( x ) d x מתכנס אך אינו מתכנס בהחלט נאמר שהוא מתכנס בתנאי.
    1. ♣ כמובן שייתכן גם אינטגרל לא אמיתי המשלב בין הסוגים: הפונקציה לא חסומה ומוגדרת על קרן.
