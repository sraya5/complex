1 התחלה

    סימון:
    בהינתן סדרה ( a n ) n = 1 ∞ נקרא לסימון ∑ n = 1 ∞ a n (או באופן שקול: a 1 + a 2 + a 3 + ... ) “הטור האינסופי של ( a n ) n = 1 ∞ ”, לאיברי הסדרה קוראים במקרה כזה גם “איברי הטור” ולאיבר ה- n -י ( a n ) קוראים “האיבר הכללי של הטור”.
    ‫הגדרה 1.1. תהא ( a n ) n = 1 ∞ סדרת מספרים ממשיים; נסמן ב- S N את ∑ n = 1 N a n (לכל N ∈ N ) ונקרא לו “הסכום החלקי ה- N -י של הטור”, הסדרה ( S N ) N = 1 ∞ תקרא “סדרת הסכומים החלקיים של הטור”.
    נאמר שהטור ∑ n = 1 ∞ a n מתכנס אם הסדרה ( S N ) N = 1 ∞ מתכנסת ובמקרה כזה נאמר שסכום הטור ∑ n = 1 ∞ a n הוא lim N → ∞ S N ונכתוב: ∑ n = 1 ∞ a n = lim N → ∞ S N = lim N → ∞ ∑ n = 1 N a n אם הטור אינו מתכנס נאמר שהוא מתבדר.
    אם סדרת הסכומים החלקיים של ( a n ) n = 1 ∞ שואפת ל- ± ∞ נאמר שהטור שואף/מתבדר ל- ± ∞ (כזכור מהקורס הקודם שאיפה ל- ± ∞ היא סוג של התבדרות) ונכתוב: ∑ n = 1 ∞ a n = ± ∞
    1. ♣ בהינתן סדרה, הטור שלה לא מוכרח להיות קיים שהרי אין זה מוכרח שהגבול הנ”ל קיים.
    ‫הגדרה 1.2. בהינתן טור ∑ n = 1 ∞ a n נקרא לטור ∑ n = m + 1 ∞ a n ( m ∈ N 0 ) בשם ה- m זנב של ∑ n = 1 ∞ a n או גם ה- m -שארית של ∑ n = 1 ∞ a n , אם ה- m -זנב של טור מתכנס נסמן את סכומו ב- r m .

2 טורים חיוביים

    ‫הגדרה 2.1. תהא ( a n ) n = 1 ∞ סדרה,
    • אם a n אי-שלילי לכל n ∈ N נאמר שהטור ∑ n = 1 ∞ a n הוא טור חיובי.
    • אם a n חיובי לכל n ∈ N נאמר שהטור ∑ n = 1 ∞ a n הוא טור חיובי ממש.
    • אם a n אי-חיובי לכל n ∈ N נאמר שהטור ∑ n = 1 ∞ a n הוא טור שלילי.
    • אם a n שלילי לכל n ∈ N נאמר שהטור ∑ n = 1 ∞ a n הוא טור שלילי ממש.
    1. ♣ סדרת הסכומים החלקיים של טור חיובי היא סדרה מונוטונית עולה ולכן אם היא חסומה אז הטור מתכנס ואם אינה חסומה אז הטור שואף לאינסוף (אלו שתי האפשרויות היחידות כמובן), זוהי הסיבה העיקרית לעניין בטורים חיוביים.
    2. ♣ כמובן שכל מה שנאמר על טורים חיוביים נכון גם על טורים שליליים (עם השינויים הנדרשים).

3 טורים בעלי סימנים משתנים

    ‫למה 3.1. תהא ( a n ) n = 1 ∞ סדרה, אם הטור ∑ n = 1 ∞ | a n | מתכנס אז גם הטור ∑ n = 1 ∞ a n מתכנס.
    ‫הגדרה 3.2. יהי ∑ n = 1 ∞ a n טור.
    1. אם הטור ∑ n = 1 ∞ | a n | מתכנס נאמר שהטור ∑ n = 1 ∞ a n מתכנס בהחלט.
    2. אם הטור ∑ n = 1 ∞ a n מתכנס אך הטור ∑ n = 1 ∞ | a n | אינו מתכנס, נאמר שהטור ∑ n = 1 ∞ a n מתכנס בתנאי.
    1. ♣ טור הערכים המוחלטים הוא טור חיובי ולכן ניתן להשתמש במבחני ההתכנסות של טורים חיוביים כדי לבדוק אם טור נתון מתכנס בהחלט (וממילא מתכנס בעצמו).
    ‫הגדרה 3.3. נאמר שטור ∑ n = 1 ∞ a n הוא טור חסום אם סדרת הסכומים החלקיים שלו חסומה.
