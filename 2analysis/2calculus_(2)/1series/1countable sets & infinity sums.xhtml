1 קבוצות בנות מנייה

    ‫הגדרה 1.2. נאמר שקבוצה A היא אין-סופית אם קיימת פונקציה חח”ע ועל מ- A לתת-קבוצה ממש שלה (כלומר תת-קבוצה שאינה A עצמה), אחרת נאמר ש- A סופית.
    1. ♣ ניתן גם להגדיר שקבוצה A היא סופית אם קיים n ∈ N וקיימת פונקציה f : A → { 1 , 2 , 3 , … , n } שהיא חח”ע ועל, ואחרת היא אין-סופית; אנחנו לא לומדים עכשיו את תורת הקבוצות ולכן לא נעסוק בהבדל שבין שתי ההגדרות הללו.
    טענה 1.3. תהא A קבוצה בת-מנייה ותהא B ⊆ A קבוצה אין-סופית, גם B בת-מנייה.
    טענה 1.4. תהיינה A , B קבוצות בנות-מנייה, גם A ∪ B היא קבוצה בת-מנייה.
    1. ♣ כמובן שגם איחוד של קבוצה בת-מנייה עם קבוצה סופית הוא קבוצה בת-מנייה.
    ‫מסקנה 1.5. כל איחוד סופי של קבוצות בנות-מנייה הוא קבוצה בת-מנייה.
    טענה 1.6. תהא ( A n ) n = 1 ∞ סדרה של קבוצות סופיות, האיחוד של כל הקבוצות בסדרה הוא קבוצה סופית או בת-מנייה.
    משפט 1.7: תהא ( A n ) n = 1 ∞ סדרה של קבוצות בנות-מנייה, מתקיים גם: | ⋃ n = 1 ∞ A n | = ℵ 0 כלומר האיחוד האין-סופי של הקבוצות האין-סופיות בסדרה הוא קבוצה בת-מנייה.
    1. ♣ ממשפט זה נובע שלא רק שהטבעיים והשלמים הן קבוצות בנות-מנייה אלא שגם הרציונליים הם קבוצה בת-מנייה.
    הוכחה. נשים לב לכך שמתקיים: ⋃ n = 1 ∞ A n = ⋃ n = 2 ∞ ( ⋃ k = 1 k = n { ( k , n - k ) } ) = ⋃ n = 2 ∞ ( ⋃ k + j = n { ( k , j ) } ) והרי לכל n ∈ N הקבוצה ⋃ k + j = n { ( k , j ) } (שהיא האלכסון ה- n - 1 -י באינטואיציה) היא קבוצה סופית ולכן ע”פ הטענה הקודמת (1.6) האיחוד שלהן הוא קבוצה בת-מנייה.

2 סכומים אין-סופיים

    ‫הגדרה 2.1. תהא A קבוצה כלשהי (לאו דווקא תת-קבוצה של R ) ותהא f : A → [ 0 , ∞ ) פונקציה; נסמן
    4
    זהו סימון פורמלי בלבד, לא ברור שהקבוצה אכן חסומה מלעיל ולכן ייתכן שהביטוי אינו מוגדר.
    : ∑ a ∈ A f ( a ) : = sup { ∑ k = 1 n f ( a k ) : a 1 , a 2 , ... , a n ∈ A , n ∈ N } ונקרא ל- ∑ a ∈ A f ( a ) (אם הוא מוגדר) הסכום של ערכי f על פני כל איברי A .
    1. ♣ הגדרה זו אינה סותרת את הגדרת סכום של איברי קבוצה סופית אלא מכלילה אותה.
    2. ♣ למה לא הגדרנו כך (עבור A ⊆ [ 0 , ∞ ) ): ∑ a ∈ A a : = sup { ∑ k = 1 n a k a 1 , a 2 , ... , a n ∈ A , n ∈ N } בשביל מה להכניס את f לכאן?
      שלוש סיבות: כדי להשיג מקרה כללי יותר ( A אינה בהכרח תת-קבוצה של R ), כדי לאפשר חזרות ( A היא קבוצה ולא תאפשר סכימה של אותו איבר פעמיים) ומכיוון שבהמשך נדבר על טורים של סדרות (והלא סדרה היא סוג של פונקציה).
    ‫הגדרה 2.2. קבוצה של מספרים אי-שליליים A תקרא סכימה (כלומר ניתנת לסכימה, שאפשר לסכום על פני כל איבריה) אם החסם העליון הבא מוגדר: sup { ∑ k = 1 n a k a 1 , a 2 , ... , a n ∈ A , n ∈ N , ∀ i , j : i ≠ j → a i ≠ a j }
    טענה 2.3. אם קבוצה A היא היא סכימה אז לכל 0 < ε ∈ R הקבוצה { a ∈ A ∣ a ≥ ε } היא סופית.
    טענה 2.4. תהא A קבוצה ותהא f : A → [ 0 , ∞ ) פונקציה, אם הסכום של ערכי f על פני כל איברי A קיים אז הקבוצה { a ∈ A : f ( a ) ≠ 0 } היא סופית או בת-מנייה.
    הוכחה. נניח שהסכום המדובר אכן קיים, מהגדרת f לכל a ∈ A המקיים f ( a ) ≠ 0 מתקיים f ( a ) > 0 וא”כ קיים n ∈ N כך ש- f ( a ) ≥ 1 n .
    לכל n ∈ N נגדיר A n : = { a ∈ A : f ( a ) ≥ 1 n } , מהטענה הקודמת (    2.3) נובע ש- A n היא קבוצה סופית לכל n ∈ N , נשים לב לכך שמתקיים: { a ∈ A : f ( a ) ≠ 0 } = ⋃ n = 1 ∞ A n ולכן (ע”פ טענה 1.6) מדובר בקבוצה סופית או בת-מנייה.
