1 התחלה

    משפט 1.1: תהא ( a n ) n = 1 ∞ סדרה אי-שלילית, מתקיים: ∑ n ∈ N a n = ∑ n = 1 ∞ a n כלומר הטור מתכנס וסכומו שווה לסכום האינסופי של איברי ( a n ) n = 1 ∞ .
    ‫מסקנה 1.2. לכל סדרה אי-שלילית ( a n ) n = 1 ∞ סכום הטור ∑ n = 1 ∞ a n אינו תלוי בסדר האיברים בסדרה.
    משפט 1.3: תנאי קושי להתכנסות טורים
    תנאי הכרחי ומספיק לכך שטור ∑ n = 1 ∞ a n יתכנס הוא שלכל 0 < ε ∈ R קיים N ∈ N כך שלכל N < n ∈ N ולכל K ∈ N מתקיים:
    | ∑ i = n + 1 n + k a n | < ε
    1. ♣ זהו מקרה פרטי של תנאי קושי להתכנסות סדרות אותו למדנו בקורס הקודם (נזכור שההגדרה של טור היא גבול של סדרת הסכומים החלקיים).
    טענה 1.4. יהיו ∑ n = 1 ∞ a n ו- ∑ n = 1 ∞ b n טורים מתכנסים ויהי c ∈ R .
    1. ∑ n = 1 ∞ ( a n ± b n ) = ∑ n = 1 ∞ a n ± ∑ n = 1 ∞ b n .
    2. ∑ n = 1 ∞ ( c ⋅ a n ) = c ⋅ ∑ n = 1 ∞ a n .
    1. ♣ הטורים ∑ n = 1 ∞ ( a n + b n ) ו- ∑ n = 1 ∞ ( a n - b n ) יכולים להתכנס גם אם ∑ n = 1 ∞ a n ו- ∑ n = 1 ∞ b n אינם מתכנסים; למעשה, ע”פ סעיף 1 אם שניים מארבעת הטורים הללו מתכנסים אז גם שני האחרים מתכנסים.
    2. ♣ מסעיף 2 נובע שאם c ≠ 0 אז התכנסות הטור ∑ n = 1 ∞ ( c ⋅ a n ) גוררת את התכנסות הטור ∑ n = 1 ∞ a n .
    משפט 1.5: תהא ( a n ) n = 1 ∞ סדרה, אם הטור ∑ n = 1 ∞ a n מתכנס אז lim n → ∞ a n = 0 .
    טענה 1.6. אם הטור ∑ n = 1 ∞ a n מתכנס למספר S ∈ R אז r m = S - ∑ n = 1 m a n , ואם קיים m ∈ N כך שה- m -זנב של הטור ∑ n = 1 ∞ a n מתכנס ל- S אז ∑ n = 1 ∞ a n = S + ∑ n = 1 m a n .
    ‫מסקנה 1.7. יהי ∑ n = 1 ∞ a n טור.
    1. הטור ∑ n = 1 ∞ a n מתכנס אם”ם לכל m ∈ N ה- m -זנב שלו מתכנס.
    2. אם הטור ∑ n = 1 ∞ a n מתכנס אזי lim m → ∞ r m = 0 .
    3. שינוי, הוספה או גריעה של מספר סופי מאיברי הטור אינה משנה את עצם ההתכנסות/התבדרות שלו.
    טענה 1.8. יהיו ∑ n = 1 ∞ a n ו- ∑ n = 1 ∞ b n טורים מתכנסים ויהי c ∈ R .
    1. אם לכל n ∈ N מתקיים a n ≥ 0 אז ∑ n = 1 ∞ a n ≥ 0 .
    2. אם לכל n ∈ N מתקיים a n ≤ b n אז ∑ n = 1 ∞ a n ≤ ∑ n = 1 ∞ b n .

3 טורים בעלי סימנים משתנים

    משפט 3.1: משפט לייבניץ
    תהא ( a n ) n = 1 ∞ סדרה חיובית, מונוטונית יורדת ומתכנסת ל- 0 .
    1. הטור ∑ n = 1 ∞ ( ( - 1 ) n + 1 a n ) מתכנס.
    2. 0 ≤ ∑ n = 1 ∞ ( ( - 1 ) n + 1 a n ) ≤ a 1 .
    3. ה- m זנב של הטור מקיים | r m | ≤ a m + 1 (כאשר m ∈ Odd מתקיים - a m + 1 < r m ≤ 0 וכאשר m ∈ Even מתקיים 0 ≤ r m < a m + 1 ).

5 מכפלות טורים

    טענה 5.1. יהיו ∑ n = 1 ∞ a n ו- ∑ n = 1 ∞ b n טורים המתכנסים ל- A ול- B בהתאמה, מתקיים: ∑ n = 1 ∞ ∑ k = 1 ∞ a k ⋅ b n = A ⋅ B
    משפט 5.2: משפט קושי
    יהיו ∑ n = 1 ∞ a n ו- ∑ n = 1 ∞ b n טורים מתכנסים בהחלט ויהיו A , B ∈ R כך ש- A = ∑ n = 1 ∞ a n ו- B = ∑ n = 1 ∞ b n .
    כל טור המורכב מכל המכפלות מהצורה a i ⋅ b j (עבור כל i , j ∈ N ) ללא חזרות הוא טור מתכנס בהחלט וסכומו הוא A ⋅ B .
