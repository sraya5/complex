    משפט 2.20: תהא f פונקציה אנליטית בנקודה a ∈ R ורדיוס ההתכנסות של טור טיילור שלה סביב a הוא 0 < R (כולל המקרה שבו רדיוס ההתכנסות הוא ∞ ) אז f אנליטית בכל קטע ההתכנסות של הטור; בנוסף, רדיוס ההתכנסות של טור טיילור שלה R ' סביב נקודה x 0 ∈ ( a - R , a + R ) מקיים: R ' ≥ min { | ( a + R ) - x 0 | , | ( a - R ) - x 0 | }
