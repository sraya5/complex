1 התחלה

    1. ♣ אתם רוצים לחשב את sin ( 1 ) , מה עושים? פותחים מחשבון ומקישים את החישוב המבוקש ובן רגע מופיעה התשובה על מסך המחשבון בדיוק של 10 ספרות אחרי הנקודה.
      שאלתם את עצמכם איך המחשבון עושה את זה? כשההורים שלנו למדו מתמטיקה לבגרות הייתה להם טבלה ובה הערכים של sin עבור עשרות זוויות (וכמו כן עבור cos ו- tan ) אך לא יתכן שזה המצב במחשבון שהרי הוא נותן תשובה לכל זווית, טבלה גדולה כזו הייתה דורשת נפח אחסון גדול מאד, אז איך בכל זאת החשבון יודע כמה שווה sin ( 1 ) ?
      התשובה היא הנושא הבא בקורס שלנו וכאן אני רוצה לתת הבהרה חשובה: סדרות וטורים של פונקציות זה נושא לא מעניין, לפחות לא בפני עצמו, הסיבה שאנחנו בכל זאת מתעניינים בו היא שבתנאים מסוימים ניתן ליצור סדרת פונקציות פשוטות לחישוב שמתקרבת לפונקציה קשה לחישוב (כמו סינוס) כך שנדע בדיוק עבור כל טווח טעות רצוי כמה אנחנו צריכים להתקדם בסדרה ע”מ למצוא קירוב טוב מספיק. זהו, ניגש לעבודה.
    ‫הגדרה 1.1. נקודת התכנסות, תחום התכנסות ופונקציה גבולית
    תהא ( f n ) n = 1 ∞ סדרת פונקציות ונסמן ב- D את החיתוך של תחומי ההגדרה שלהן, נאמר שנקודה x 0 ∈ D היא נקודת התכנסות של ( f n ) n = 1 ∞ אם הגבול lim n → ∞ f n ( x 0 ) קיים.
    קבוצת נקודות ההתכנסות של ( f n ) n = 1 ∞ תקרא תחום ההתכנסות של ( f n ) n = 1 ∞ ונאמר גם ש- ( f n ) n = 1 ∞ מתכנסת נקודתית בקבוצה זו ובכל תת-קבוצה שלה.
    יהי D ' תחום ההתכנסות של ( f n ) n = 1 ∞ הפונקציה f : D ' → R המוגדרת ע”י (לכל x ∈ D ' ): f ( x ) : = lim n → ∞ f n ( x ) תיקרא הפונקציה הגבולית של f n .
    ‫הגדרה 1.2. התכנסות טור פונקציות
    תהא ( u n ) n = 1 ∞ סדרת פונקציות ממשיות ונסמן ב- D את החיתוך של תחומי ההגדרה שלהן.
    תהא ( S N ) N = 1 ∞ סדרת פונקציות מ- D ל- R המוגדרת ע”י (לכל N ∈ N ולכל x ∈ D ): S N ( x ) : = ∑ n = 1 N u n ( x ) נאמר שנקודה x 0 ∈ D היא נקודת התכנסות של טור הפונקציות ∑ n = 1 ∞ u n ( x ) אם הגבול lim N → ∞ S N ( x 0 ) קיים (כלומר הטור ∑ n = 1 ∞ u n ( x 0 ) מתכנס).
    קבוצת נקודות ההתכנסות של טור הפונקציות ∑ n = 1 ∞ u n ( x ) תקרא תחום ההתכנסות שלו ונאמר גם שהוא מתכנס בקבוצה זו ובכל תת-קבוצה שלה.
    יהי D ' תחום ההתכנסות של ∑ n = 1 ∞ u n ( x ) הפונקציה S : D ' → R המוגדרת ע”י (לכל x ∈ D ' ): S ( x ) : = ∑ n = 1 ∞ u n ( x ) תיקרא הפונקציה הגבולית של טור הפונקציות ∑ n = 1 ∞ u n ( x ) .
    1. ♣ נשים לב שההגדרה עבור טור של פונקציות כלולה בהגדרה של סדרת פונקציות שהרי טור הוא בסך הכל גבול של סדרה (סדרת הסכומים החלקיים).
    1. ♣ בהינתן סדרת פונקציות ( f n ) n = 1 ∞ כך ש- D הוא חיתוך תחומי ההגדרה של הפונקציות, נוכל להגדיר סדרת פונקציות ( u n ) n = 1 ∞ ע”י u n = f n - f n - 1 לכל 1 < n ∈ N ו- u 1 = f 1 ואז נקבל שלכל n ∈ N מתקיים: S N = ∑ n = 1 N u n = f N ומכאן שגם (מדובר בשוויון פורמלי בלבד): S = ∑ n = 1 ∞ u n = lim N → ∞ S N = lim N → ∞ ∑ n = 1 N u n = lim N → ∞ f N = f כלומר הסדרה ( f n ) n = 1 ∞ מתלכדת עם סדרת הסכומים החלקיים של הטור ∑ n = 1 ∞ u n ובפרט הם מתכנסים ומתבדרים ביחד, זוהי התאמה חח”ע ועל בין סדרות של פונקציות לטורי פונקציות השומרת על תכונת ההתכנסות.
    ‫הגדרה 1.3. התכנסות במידה שווה של סדרות וטורי פונקציות
    נאמר שסדרת פונקציות ( f n ) n = 1 ∞ מתכנסת במידה שווה לפונקציה גבולית f ב- D
    1
    D מוכל בחיתוך תחומי ההגדרה של הפונקציות וכן לגבי טורים ובכלל בסיכום זה.
    אם לכל 0 < ε ∈ R קיים N ∈ N כך שלכל x ∈ D ולכל N < n ∈ N מתקיים | f n ( x ) - f ( x ) | < ε .
    כמו כן נאמר שטור פונקציות ∑ n = 1 ∞ u n מתכנס במידה שווה לפונקציה גבולית S ב- D אם לכל 0 < ε ∈ R קיים N ∈ N כך שלכל x ∈ D ולכל N < n ∈ N מתקיים | S n ( x ) - S ( x ) | < ε .
    ‫הגדרה 1.4. התכנסות בהחלט של טור פונקציות בנקודה
    יהא ∑ n = 1 ∞ u n ( x ) טור פונקציות המוגדרות בתחום D , נאמר ש- ∑ n = 1 ∞ u n ( x ) מתכנס בהחלט בנקודה x 0 ∈ D אם הטור ∑ n = 1 ∞ u n ( x 0 ) מתכנס בהחלט, כמו כן נאמר ש- ∑ n = 1 ∞ u n ( x ) מתכנס בהחלט ב- D אם לכל x ∈ D הטור ∑ n = 1 ∞ u n ( x ) מתכנס בהחלט. אם ∑ n = 1 ∞ u n ( x ) מתכנס בנקודה/בקבוצת נקודות אך אינו מתכנס בהן בהחלט נאמר שהוא מתכנס בתנאי בנקודה/בקבוצת הנקודות (בהתאמה).
    ‫הגדרה 1.5. התכנסות בהחלט במידה שווה של טור פונקציות
    יהי ∑ n = 1 ∞ u n ( x ) טור פונקציות המוגדרות בתחום D , נאמר ש- ∑ n = 1 ∞ u n ( x ) מתכנס בהחלט במידה שווה ב- D אם הטור ∑ n = 1 ∞ | u n ( x 0 ) | מתכנס במ”ש ב- D .
    1. ♣ ההיררכיה היא כזו: התכנסות בהחלט במ”ש גוררת התכנסות בהחלט והתכנסות במ”ש בנפרד, והתכנסות במ”ש גוררת התכנסות נקודתית; הגרירות ההפוכות אינן נכונות בהכרח.

2 טורי חזקות

    ‫הגדרה 2.1. טור חזקות
    טור חזקות סביב נקודה x 0 ∈ R הוא טור פונקציות מהצורה: ∑ n = 0 ∞ a n ⋅ ( x - x 0 ) n עבור סדרה ממשית ( a n ) n = 0 ∞ כלשהי.
    1. ♣ נשים לב שכל סדרת פולינומי טיילור היא טור חזקות, עוד נשים לב שכל טור חזקות סביב נקודה כשלהי מתכנס באותה נקודה לפונקציית האפס.
    משפט. יהי ∑ n = 0 ∞ a n ⋅ ( x - x 0 ) n טור חזקות סביב נקודה x 0 ∈ R
    2
    ( a n ) n = 0 ∞ היא סדרה כלשהי.
    מתקיימת אחת משלוש האפשרויות הבאות:
    1. הטור מתכנס נקודתית על כל הישר.
    2. קיים 0 < R ∈ R יחיד כך שהטור מתכנס נקודתית ב- ( x 0 - R , x 0 + R ) ואולי גם ב- x 0 + R ו/או ב- x 0 - R אך לא בשום נקודה אחרת.
    3. הטור מתכנס נקודתית אך ורק ב- x 0 .
    ‫הגדרה 2.2. רדיוס התכנסות וקטע התכנסות
    במונחי המשפט שלעיל וע”פ החלוקה למקרים שבו:
    1. אם מתקיימת האפשרות הראשונה נאמר שרדיוס ההתכנסות של הטור הוא ∞ וקטע ההתכנסות הוא R = ( - ∞ , ∞ ) .
    2. אם מתקיימת האפשרות השנייה נאמר שרדיוס ההתכנסות הוא R וקטע ההתכנסות הוא ( x 0 - R , x 0 + R ) .
    3. אם מתקיימת האפשרות השלישית נאמר שרדיוס ההתכנסות הוא 0 וקטע ההתכנסות הוא ∅ = ( x 0 - 0 , x 0 + 0 ) .
    1. ♣ קטע ההתכנסות אינו שווה בהכרח לתחום ההתכנסות של הטור, אמנם ניתן לראות זאת בבירור במקרה השלישי אולם הדבר נכון גם עבור האפשרות השנייה (ראו את ניסוח המשפט עבורה).

    x ∈ ( x 0 - R , x 0 + R ) מתקיים השוויון (עבור 0 < R ∈ R

    ‫הגדרה 2.4. פונקציה אנליטית
    נאמר שפונקציה f היא פונקציה אנליטית בנקודה x 0 ∈ R אם יש ל- f פיתוח לטור חזקות סביב x 0 , נאמר ש- f היא פונקציה אנליטית בקטע פתוח ( a , b ) אם לכל x 0 ∈ ( a , b ) הפונקציה f אנליטית ב- x 0 .
