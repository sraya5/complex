1 התחלה

    1. ♣ במתמטיקה המודרנית פונקציה היא מושג מתורת הקבוצות וככזו ישנם מושגים ומשפטים רבים בנושא שאינם קשורים דווקא למספרים הממשיים (ראו בקובץ “הנאיבית הקבוצות תורת”), בקבצים אלו נעסוק אך ורק בהגדרות ובמשפטים הקשורות למספרים הממשיים.
    2. ♣ בכל הסיכומים של קורסי אינפי’ נדבר אך ורק על פונקציות שהתחום והטווח שלה הם תתי-קבוצות של R (אלא אם נאמר אחרת במפורש), כמעט תמיד יהיה מדובר בפונקציה שהתחום שלה הוא מקטע
      3
      תזכורת: מקטע הוא תת-קבוצה I ⊆ R כך שלכל a , b ∈ I מתקיים ( a , b ) ⊆ I .
      (או איחוד של מקטעים) ועל הקורא תוטל המשימה להבין מן ההקשר מתי מדובר גם בפונקציות שהתחום שלהן אינו בהכרח מקטע.
    3. ♣ ישנה הסכמה מקובלת למחצה שאם נתון רק כלל ההתאמה של פונקציה (ללא התחום ו/או הטווח) אז הטווח הוא R והתחום הוא קבוצת כל הנקודות ב- R שעבורן כלל ההתאמה מוגדר, מוסכמה זו נקראת “מוסכמת התחום המרבי” ונתקלנו בה כבר בתיכון; למרות הנוחות שבמוסכמה זו אני אשתדל שלא להסתמך עליה ולציין בכל מקום במפורש את התחום והטווח.
    תהא D ⊆ R ותהא f : D → R פונקציה.

    נקודות ביחס לקבוצות ופונקציות

    ‫הגדרה 1.1. תהא a ∈ D .
    1. נאמר ש- x 0 היא נקודה פנימית של D אם קיים קטע פתוח המכיל את x 0 ומוכל כולו ב- D .
    2. נאמר ש- x 0 היא נקודה מבודדת של D אם קיים קטע פתוח המכיל את x 0 וחיתוכו עם D הוא { x 0 } .
    3. נאמר ש- x 0 היא בקצה (או בשפה) של D אם כל קטע פתוח במכיל את x 0 מכיל נקודה נוספת מ- D (כלומר בכל מקרה שאינו כלול בשני הראשונים).
    ‫הגדרה 1.2. נקודות קיצון כלליות
    • תהא a ∈ D נקודה.
      • נאמר ש- a היא נקודת מקסימום של f אם לכל x ∈ D מתקיים f ( x ) ≤ f ( a ) .
      • נאמר ש- a היא נקודת מינימום של f אם לכל x ∈ D מתקיים f ( x ) ≤ f ( a ) .
    • תהא A ⊆ D תת-קבוצה ותהא a ' ∈ A .
      • נאמר ש- a ' היא נקודת מקסימום של f ב- A אם לכל x ∈ A מתקיים f ( x ) ≤ f ( a ) .
      • נאמר ש- a ' היא נקודת מינימום של f ב- A אם לכל x ∈ A מתקיים f ( x ) ≥ f ( a ) .
    ‫הגדרה 1.3. נקודות קיצון מקומיות
    תהא a ∈ D נקודה פנימית.
    • נאמר ש- a היא נקודת מקסימום מקומית של f אם קיימת 0 < δ ∈ R כך שלכל x ∈ B δ ( a ) מתקיים f ( x ) ≤ f ( a ) .
    • נאמר ש- a היא נקודת מקסימום מקומית של f אם קיימת 0 < δ ∈ R כך שלכל x ∈ B δ ( a ) מתקיים f ( x ) ≥ f ( a ) .

    סימטריות של קבוצות ופונקציות

    ‫הגדרה 1.4. סימטריות אפשריות בקבוצות
    1. נאמר ש- D היא סימטרית אם לכל x ∈ D מתקיים - x ∈ D .
    2. נאמר ש- D היא אינווריאנטית ביחס להזזה במספר T ∈ R אם לכל x ∈ D מתקיים x + T ∈ D .
    ‫הגדרה 1.5. סימטריות אפשריות בפונקציות
    1. נאמר ש- f היא פונקציה זוגית אם לכל x ∈ D מתקיים f ( x ) = f ( - x ) .
    2. נאמר ש- f היא פונקציה אי-זוגית אם לכל x ∈ D מתקיים - f ( x ) = f ( - x ) .
    3. נאמר ש- f היא פונקציה מחזורית בעלת אורך מחזור 0 < T ∈ R אם לכל x ∈ D מתקיים f ( x ) = f ( x + T )

      T כזה אף פעם אינו יחיד.
      , אם קיים T מינימלי כזה נקרא לו המחזור היסודי של f .
    1. ♣ f יכולה להיות זוגית או אי-זוגית רק אם D סימטרית ויכולה להיות מחזורית רק אם D אינווריאנטית ביחס להזזה באורך המחזור.

2 גבול של פונקציה בנקודה

    ‫הגדרה 2.1. גבול של פונקציה בנקודה
    תהא f פונקציה המוגדרת בסביבה מנוקבת של a ∈ R , נאמר ש- L ∈ R הוא גבול של f בנקודה a אם לכל 0 < ε ∈ R קיימת 0 < δ ∈ R כך שלכל x ∈ B δ ° ( a ) מתקיים f ( x ) ∈ B ε ( L ) .
    או בסימונים אחרים: ∀ x ∈ R : 0 < | x - a | < δ ⇒ | f ( x ) - L | < ε גבול של פונקציה בנקודה הוא יחיד (אם הוא קיים), לכן מוצדק לדבר על הגבול של f בנקודה a ולסמנו ב- lim x → a f ( x ) .

    הסכמה:
    נסכים שבכל מקום שבו נכתב ביטוי מהצורה lim x → a f ( x ) = L או L = lim x → a f ( x ) הכוונה היא שהגבול קיים ומתקיים השוויון אלא אם נאמר אחרת בפירוש; הסכמה זו תהיה תקפה בכל הסיכומים של כל הקורסים (ולגבי כל הגבולות המופיעים בהם) מכאן ואילך, בכל מקום שבו נטען שגבול מקיים טענה אנו טוענים שהגבול קיים ומקיים את הטענה.
    ‫הגדרה 2.2. גבולות חד-צדדיים
    תהא f פונקציה המוגדרת בסביבה ימנית של a ∈ R , נאמר ש- L ∈ R הוא גבול מימין של f בנקודה a אם לכל 0 < ε ∈ R קיימת 0 < δ ∈ R כך שמתקיים: x ∈ ( a , a + δ ) ⇒ f ( x ) ∈ B ε ( l ) כמו כן אם f מוגדרת בסביבה שמאלית של a ' ∈ R נאמר שמספר ממשי L ∈ R הוא גבול משמאל של f בנקודה a ' אם לכל 0 < ε ∈ R קיימת 0 < δ ∈ R כך שמתקיים: x ∈ ( a ' - δ , a ' ) ⇒ f ( x ) ∈ B ε ( l ) גם גבולות חד-צדדיים הם יחידים (אם הם קיימים), לכן מוצדק לדבר על הגבול מימין של f ב- a (אם הוא קיים) ולסמנו ב- lim x → a + f ( x ) וכמו כן מוצדק לדבר על הגבול משמאל של f ב- a ' ולסמנו ב- lim x → a - f ( x ) .
    1. ♣ אצל יורם ראינו גם את הסימונים lim x ↓ a f ( x ) עבור גבול מימין ו- lim x ↑ a ' f ( x ) עבור גבול משמאל.
    2. ♣ מהגדרות אלו נובע כמעט מיד שלפונקציה יש גבול בנקודה אם”ם הגבולות החד-צדדיים שלה בנקודה זו קיימים ושווים.

3 חסימות וסדר

    ‫הגדרה 3.1. תהא f פונקציה,
    • נאמר ש- f חסומה מלעיל אם Im f חסומה מלעיל.
    • נאמר ש- f חסומה מלרע אם Im f חסומה מלרע.
    • נאמר ש- f חסומה אם הקבוצה Im f חסומה.
    ‫הגדרה 3.2. תהא f : D → R פונקציה ותהא U ⊆ D ,
    • נאמר ש- f חסומה מלעיל ב- U אם f ( U ) חסומה מלעיל.
    • נאמר ש- f חסומה מלרע ב- U אם f ( U ) חסומה מלרע.
    • נאמר ש- f חסומה ב- U אם הקבוצה f ( U ) חסומה.
    ‫הגדרה 3.3. תהא f פונקציה המוגדרת בסביבה מנוקבת של נקודה a ∈ R , נאמר ש- f חסומה מקומית ב- a אם קיימת סביבה מנוקבת U של a כך ש- f חסומה ב- U .

4 רציפות

    ‫הגדרה 4.1. רציפות של פונקציה בנקודה
    תהא f פונקציה המוגדרת בסביבה של נקודה a ∈ R , נאמר ש- f רציפה ב- a אם מתקיים lim x → a f ( x ) = f ( a ) .
    כלומר: לכל 0 < ε ∈ R קיימת 0 < δ ∈ R כך שמתקיים: x ∈ B δ ( a ) ⇒ f ( x ) ∈ B ε ( L ) או בסימונים אחרים: ∀ x ∈ R : | x - a | < δ ⇒ | f ( x ) - L | < ε
    ‫הגדרה 4.2. תהא f פונקציה, את הנקודות שבהן f אינה רציפה (אם יש כאלה) נחלק לשלושה סוגים:
    1. אם הגבול בנקודה קיים אך אינו שווה לערך שמקבלת הפונקציה בנקודה זו (כולל המקרה שבו הפונקציה אינה מוגדרת בנקודה), נאמר שזוהי נקודת אי-רציפות סליקה.
    2. אם הגבול בנקודה אינו קיים אך שני הגבולות החד-צדדיים קיימים
      5
      מהעובדה שהגבול עצמו אינו קיים נובע שהגבולות החד-צדדיים שונים.
      , נאמר שזוהי נקודת אי-רציפות מסוג ראשון.
    3. ואם לפחות אחד משני הגבולות החד-צדדיים אינו קיים
      6
      שאז בהכרח גם הגבול אינו קיים.
      , נאמר שזוהי נקודת אי-רציפות מסוג שני.
    1. ♣ נשים לב: f אינה רציפה בנקודה a אם לא מתקיים lim x → a f ( x ) = f ( a ) מכל סיבה שהיא, כולל מהסיבה ש- f ( a ) אינו קיים (כך לדוגמה π 2 היא נקודת אי-רציפות של tan ).
    2. ♣ את ההערה הקודמת שמעתי מפיו של רז במו אוזניי אולם ניתן גם לומר שנקודה שבה הפונקציה אינה מוגדרת אינה נקודת רציפות וגם אינה נקודת אי-רציפות, היא לא שייכת לעניין בכלל; אם זכרוני אינו מטעני שמענו מיורם אמירה ברוח זו.
    ‫הגדרה 4.3. תהא f פונקציה,
    • נאמר ש- f רציפה מימין בנקודה a ∈ R אם מתקיים lim x → a + f ( x ) = f ( a ) .
    • נאמר ש- f רציפה משמאל בנקודה a ∈ R אם מתקיים lim x → a - f ( x ) = f ( a ) .
    1. ♣ כדי שפונקציה תהיה רציפה (מימין/משמאל) בנקודה היא צריכה להיות מוגדרת בסביבה (ימנית/שמאלית) מלאה שלה.
    2. ♣ מהגדרות אלו נובע כמעט מיד שפונקציה רציפה בנקודה אם”ם היא רציפה בנקודה זו מימין ומשמאל.
    ‫הגדרה 4.4. פונקציה f תקרא רציפה אם היא רציפה בכל נקודה בתחום הגדרתה, כמו כן f תקרא רציפה בקטע פתוח ( a , b ) ⊆ R אם לכל x ∈ ( a , b ) - f רציפה ב- x , ותקרא רציפה בקטע סגור [ a , b ] ⊆ R אם היא רציפה בקטע הפתוח ( a , b ) ובנוסף היא רציפה ב- a מימין וב- b משמאל.
    1. ♣ האינטואיציה לפונקציה רציפה היא פונקציה שאפשר לצייר את הגרף שלה מבלי להרים את העיפרון מהדף, וזו אכן משתקפת בהגדרה: בכל נקודה אין “קפיצות” בגרף (אחרת הגבול לא היה שווה לערך הפונקציה בנקודה) ולכן אין צורך “להרים את העיפרון” כדי לצייר אותה לאחר שציירנו את כל אלו שלפניה, ומאידך - אם הייתה נקודה שבה הגבול לא היה קיים או שלא היה שווה לערך הפונקציה בנקודה היינו מקבלים “קפיצה” בגרף ולא יכולים לצייר אותו מבלי “להרים את העיפרון”.

    ‫הגדרה 5.1. גבולות במובן הרחב

    מסודרים בסדר הנ”ל טבלה אחרי טבלה, מלמעלה למטה ומימין לשמאל.

      • נאמר ש- f שואפת למינוס אין-סוף ב- a אם לכל m ∈ R קיימת 0 < δ ∈ R כך שמתקיים x ∈ B δ ° ( a ) ⇒ f ( x ) < m .
    • תהא f פונקציה המוגדרת בסביבה ימנית של נקודה a ∈ R .
      • נאמר ש- f שואפת לאין-סוף מימין ל- a אם לכל M ∈ R קיימת 0 < δ ∈ R כך שמתקיים x ∈ ( a , a + δ ) ⇒ f ( x ) > M .
      • נאמר ש- f שואפת למינוס אין-סוף מימין ל- a אם לכל m ∈ R קיימת 0 < δ ∈ R כך שמתקיים x ∈ ( a , a + δ ) ⇒ f ( x ) < m .
    • תהא f פונקציה המוגדרת בסביבה שמאלית של נקודה a ∈ R .
      • נאמר ש- f שואפת לאין-סוף משמאל ל- a אם לכל M ∈ R קיימת 0 < δ ∈ R כך שמתקיים x ∈ ( a - δ , a ) ⇒ f ( x ) > M .
      • נאמר ש- f שואפת למינוס אין-סוף משמאל ל- a אם לכל m ∈ R קיימת 0 < δ ∈ R כך שמתקיים x ∈ ( a - δ , a ) ⇒ f ( x ) < m .
    • תהא f פונקציה המוגדרת על קרן ימנית.
      • נאמר ש- f שואפת למספר ממשי L ב- ∞ אם לכל 0 < ε ∈ R קיים K ∈ R כך שלכל K < x ∈ R מתקיים | f ( x ) - l | < ε .
      • נאמר ש- f שואפת לאין-סוף ב- ∞ אם לכל M ∈ R קיים K ∈ R כך שלכל K < x ∈ R מתקיים f ( x ) > M .
      • נאמר ש- f שואפת למינוס אין-סוף ב- ∞ אם לכל m ∈ R קיים K ∈ R כך שלכל K < x ∈ R מתקיים f ( x ) < m .
    • תהא f פונקציה המוגדרת על קרן שמאלית.
      • נאמר ש- f שואפת למספר ממשי L ב- - ∞ אם לכל 0 < ε ∈ R קיים K ∈ R כך שלכל K > x ∈ R מתקיים | f ( x ) - l | < ε .
      • נאמר ש- f שואפת לאין-סוף ב- - ∞ אם לכל M ∈ R קיים K ∈ R כך שלכל K > x ∈ R מתקיים f ( x ) > M .
      • נאמר ש- f שואפת למינוס אין-סוף ב- - ∞ אם לכל m ∈ R קיים K ∈ R כך שלכל K > x ∈ R מתקיים f ( x ) < m .
    1. ♣ כמובן שעדיף להבין את ההיגיון מאחורי ההגדרות ולא לשנן אותן כתוכי, הסיבה להבאתן של כל ההגדרות כאן היא רצוני שהכל יהיה כתוב כך שכאשר נדבר על מושגים אלו בהמשך נדבר על מושגים שאכן הגדרנו בפירוש.
    ‫הגדרה 5.2. אסימפטוטות בנקודה
    נאמר של- f יש אסימפטוטה אנכית בנקודה a ∈ R אם מתקיים לפחות אחד מהשוויונים הבאים: lim x → a + f ( x ) = ∞ , lim x → a - f ( x ) = ∞ , lim x → a + f ( x ) = - ∞ , lim x → a - f ( x ) = - ∞
    ‫הגדרה 5.3. אסימפטוטות משופעות (אסימפטוטות ב- ± ∞ )
    • נאמר שישר { ( x , a x + b ) ∣ x ∈ R } הוא אסימפטוטה משופעת של f ב- ∞ אם מתקיים: lim x → ∞ ( f ( x ) - ( a x + b ) ) = 0
    • נאמר שישר { ( x , a x + b ) ∣ x ∈ R } הוא אסימפטוטה משופעת של f ב- - ∞ אם מתקיים: lim x → - ∞ ( f ( x ) - ( a x + b ) ) = 0
    כאשר a = 0 נהוג לומר גם שזו אסימפטוטה אופקית.

6 מונוטוניות

    ‫הגדרה 6.1. פונקציות מונוטוניות
    תהא f : A → B פונקציה.
    1. נאמר ש- f מונוטונית עולה אם לכל x , y ∈ A המקיימים x < y מתקיים f ( x ) ≤ f ( y ) .
    2. נאמר ש- f עולה ממש
      8
      יש אומרים מונוטונית עולה ממש וכן לגבי מונוטונית יורדת ממש.
      אם לכל x , y ∈ A המקיימים x < y מתקיים f ( x ) < f ( y ) .
    3. נאמר ש- f מונוטונית יורדת אם לכל x , y ∈ A המקיימים x < y מתקיים f ( x ) ≥ f ( y ) .
    4. נאמר ש- f יורדת ממש אם לכל x , y ∈ A המקיימים x < y מתקיים f ( x ) > f ( y ) .
    5. נאמר ש- f מונוטונית אם היא מקיימת את אחד מארבעת הסעיפים הקודמים.
    6. נאמר ש- f מונוטונית ממש אם היא עולה ממש או יורדת ממש.
    ‫מסקנה 6.2. כל פונקציה מונוטונית ממש היא חח”ע.

8 פונקציית האקספוננט, הלוגריתם הטבעי וחזקות ממשיות

    8.1 הגדרת האקספוננט

    ‫הגדרה 8.1. כשלמדנו את נושא הסדרות ראינו שהסדרה ( ( 1 + 1 n ) n ) n = 1 ∞ מתכנסת, נסמן: e : = lim n → ∞ ( 1 + 1 n ) n
    1. ♣ על הקשר בין e לריבית דריבית (ציטוט מוויקיפדיה):
      בחישובי ריבית דריבית e מהווה קבוע לחישוב החוב כעבור זמן מסוים או כעבור מספר מחזורים מסוים. הקבוע e נתגלה בידי
      ברנולי יאקוב בעת ניתוח חישובי ריבית דריבית.
      לדוגמה, אם אדם מפקיד בבנק סכום של שקל אחד (הקרן היא שקל אחד) ומקבל ריבית של 100% המחושבת אחת לשנה, הוא יצבור סכום של שני שקלים בסוף השנה. אם הבנק יחשב את הריבית מדי חצי שנה בריבית דריבית, כלומר ריבית של 50% בחצי הראשון של השנה שלאחריו יהיה בבנק שקל וחצי, וחישוב ריבית של 50% בחצי השני של השנה, הפעם 50% על שקל וחצי, שהם 50% על הקרן - השקל, ועוד 50% על הריבית שנצברה - חצי השקל שהתקבל בחלק הראשון של השנה. בתום החצי השני של השנה יהיו ברשותו 2.25 שקלים. אם חישוב הריבית יבוצע מדי רבע שנה, יסיים עם 2.44 שקלים. אם חישובי הריבית יבוצעו במרווחי הזמן הקטנים ביותר הסכום שיתקבל יתקרב ל- e כפול הקרן. בדוגמה שלנו הקרן הוא שקל אחד, ולכן בסוף השנה המפקיד יקבל מהבנק e שקלים לפי הנוסחה: e = lim n → ∞ ( 1 + 1 n ) n שהם בקירוב 2 שקלים ו- 72 אגורות.
    יהי α ∈ R ותהא ( a n ) n = 1 ∞ סדרה המוגדרת ע”י a n : = ( 1 + α n ) n (לכל n ∈ N ), נרצה להוכיח שהגבול lim n → ∞ a n קיים.
    טענה. קיים N ∈ N כך שהסדרה ( α n ) n = N ∞ מונוטונית עולה.
    טענה. ( a n ) n = 1 ∞ חסומה מלעיל.
    מסקנה. ( a n ) n = 1 ∞ מתכנסת.
    מסקנה. הגבול lim n → ∞ ( 1 + x n ) n קיים לכל x ∈ R .
    ‫הגדרה 8.2. נגדיר את פונקציית האקספוננט (נקראת גם הפונקציה האקספוננציאלית) exp : R → ( 0 , ∞ ) ע”י (לכל x ∈ R ): exp ( x ) = lim n → ∞ ( 1 + x n ) n
    1. ♣ מהגדרה exp ( 1 ) = e ו- exp ( 0 ) = 1 .

    8.2 תכונות פונקציית האקספוננט

    משפט. exp חח”ע ועל ומכאן שהיא הפיכה.

    8.3 הלוגריתם הטבעי

    ‫הגדרה 8.3. נגדיר את פונקציית הלוגריתם הטבעי ln : ( 0 , ∞ ) → R ע”י ln : = exp - 1 .
    1. ♣ מהגדרה ln ( e ) = 1 .
    למה. לכל 0 < a ∈ R וכל q ∈ Q מתקיים ln ( a q ) = q ⋅ ln ( a ) .
    מסקנה. לכל 0 < a ∈ R ולכל q ∈ Q מתקיים exp ( q ⋅ ln ( a ) ) = a q .
    1. ♣ מסקנה זו מאפשרת לנו להגדיר חזקות במעריך ממשי בצורה שתתלכד עם הגדרת חזקות במעריך רציונלי.

    8.4 חזקות ממשיות ולוגריתמים בבסיסים שונים מ-e

    ‫הגדרה 8.4. חזקות ממשיות
    לכל 0 < a ∈ R ולכל x ∈ R נגדיר a x : = exp ( x ⋅ ln a ) .
    1. ♣ מהגדרה e x = exp ( x ) לכל x ∈ R .
    2. ♣ נעיר שניתן להגדיר חזקה עם מעריך ממשי גם בצורה שהבאנו בקובץ ההגדרות של המספרים הממשיים, אלו הגדרות שקולות מפני שפונקציית האקספוננט רציפה והסופרמום המופיע בהגדרה הנ”ל הוא הגבול של exp ( x ⋅ ln ( a ) ) בנקודה המתאימה.
    ‫מסקנה 8.5. לכל 0 < a ∈ R ולכל x ∈ R מתקיים a x > 0 .
    ‫מסקנה 8.6. לכל 0 < a ∈ R , הפונקציה f : R → ( 0 , ∞ ) היא פונקציה רציפה.
    ‫מסקנה 8.7. לכל 0 < a ∈ R ולכל x ∈ R מתקיים ln ( a x ) = x ⋅ ln ( a ) .
    1. ♣ בפרט לכל x ∈ R מתקיים 1 x = 1 .
    מסקנה. יהי 0 < a ∈ R ונגדיר את הפונקציה f : R → ( 0 , ∞ ) ע”י f ( x ) = a x (לכל x ∈ R );
    • אם 1 < a אז f עולה ממש.
    • אם a < 1 אז f יורדת ממש.
    • אם a = 1 אז f פונקציה קבועה ( f ( x ) = 1 לכל x ∈ R ).
    מכאן נסיק שאם a ≠ 1 אז f חח”ע ועל וממילא הפיכה.
    ‫הגדרה 8.8. לוגריתם בבסיס כללי
    לכל 0 < a ∈ R השונה מ- 1 נגדיר את הפונקציה log a : ( 0 , ∞ ) → R להיות הפונקציה ההופכית לפונקציה המעתיקה את x ∈ R ל- a x .
    כלומר לכל x , y ∈ R מתקיים: y = log a ( x ) ⟺ a y = x .
    1. ♣ מהגדרה לכל 0 < a ∈ R השונה מ- 1 מתקיים log a ( 1 ) = 0 .
    2. ♣ מהגדרה log e = ln .
    ‫מסקנה 8.9. לכל 0 < a ∈ R השונה מ- 1 הפונקציה לוגריתמית log a רציפה.

9 רציפות במידה שווה

    ‫הגדרה 9.1. תהא f פונקציה המוגדרת על מקטע I , נאמר ש- f רציפה במידה שווה (להלן גם: רציפה במ”ש) על I אם לכל 0 < ε ∈ R קיימת 0 < δ ∈ R כך שלכל x 1 , x 2 ∈ I המקיימים | x 1 - x 2 | < δ מתקיים | f ( x 1 ) - f ( x 2 ) | < ε .
    1. ♣ בכל פעם שאנו רוצים למצוא גבול של פונקציה עלינו למצוא לכל 0 < ε את הסביבה הדלתאית שבה כל האיברים המדוברים קרובים לגבול עד כדי ε , אך כיצד נבחר את התחום המתאים? כמובן שע”פ ה- ε הנתון (יהי 0 < ε ...), בד”כ הבחירה תהיה תלויה בנקודה שבה אנו רוצים להוכיח את קיום הגבול ותדרוש חישובים מסובכים אך לפעמים הבחירה תהיה קלה מאד: לדוגמה עבור פונקציה ליניארית יספיק תמיד לבחור ב- δ : = ε | m | כאשר m הוא שיפוע הפונקציה ולא משנה באיזו נקודה מדובר; רציפות במידה שווה אומרת בדיוק את הדבר הזה, ה- δ תלויה אך ורק ב- ε ולא בנקודה.
      הנה מאמר נהדר של אמיר חדאיו שמטיב להסביר את הרעיון.
    2. ♣ עבור פונקציה רציפה במידה שווה לכל 0 < ε ∈ R קיימת 0 < δ ∈ R כך שלכל נקודה בתחום הגרף בסביבה הדלתאית שלה נכנס למלבן באורך 2 ε וברוחב של 2 δ שמרכזו בנקודה.
