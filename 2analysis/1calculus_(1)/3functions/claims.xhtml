1 התחלה

    1. ♣ במתמטיקה המודרנית פונקציה היא מושג מתורת הקבוצות וככזו ישנם מושגים ומשפטים רבים בנושא שאינם קשורים דווקא למספרים הממשיים (ראו בקובץ “הנאיבית הקבוצות תורת”), בקבצים אלו נעסוק אך ורק בהגדרות ובמשפטים הקשורות למספרים הממשיים.
    2. ♣ בכל הסיכומים של קורסי אינפי’ נדבר אך ורק על פונקציות שהתחום והטווח שלה הם תתי-קבוצות של R (אלא אם נאמר אחרת במפורש), כמעט תמיד יהיה מדובר בפונקציה שהתחום שלה הוא מקטע
      3
      תזכורת: מקטע הוא תת-קבוצה I ⊆ R כך שלכל a , b ∈ I מתקיים ( a , b ) ⊆ I .
      (או איחוד של מקטעים) ועל הקורא תוטל המשימה להבין מן ההקשר מתי מדובר גם בפונקציות שהתחום שלהן אינו בהכרח מקטע.
    3. ♣ ישנה הסכמה מקובלת למחצה שאם נתון רק כלל ההתאמה של פונקציה (ללא התחום ו/או הטווח) אז הטווח הוא R והתחום הוא קבוצת כל הנקודות ב- R שעבורן כלל ההתאמה מוגדר, מוסכמה זו נקראת “מוסכמת התחום המרבי” ונתקלנו בה כבר בתיכון; למרות הנוחות שבמוסכמה זו אני אשתדל שלא להסתמך עליה ולציין בכל מקום במפורש את התחום והטווח.

    1.1 פונקציות זוגיות ואי-זוגיות

    תהיינה f , g : D → R שתי פונקציות.
    טענה 1.1.
    • אם f ו- g זוגיות אז גם f + g זוגית.
    • אם f ו- g אי-זוגיות אז גם f + g אי-זוגית.
    טענה 1.2.
    • אם f ו- g זוגיות אז f ⋅ g זוגית.
    • אם f ו- g אי-זוגיות אז f ⋅ g זוגית.
    • אם f זוגית ו- g אי-זוגית אז f ⋅ g אי-זוגית.
    1. ♣ בפרט הכפלה בקבוע אינה משנה את הזוגיות/אי-זוגיות של פונקציה.
    ‫למה 1.3. נניח ש- g ( x ) ≠ 0 לכל x ∈ D .
    • אם g זוגית אז גם 1 g זוגית.
    • אם g אי-זוגית אז גם 1 g אי-זוגית.
    ‫מסקנה 1.4.
    • אם f ו- g זוגיות אז f g זוגית.
    • אם f ו- g אי-זוגיות אז f g זוגית.
    • אם f זוגית ו- g אי-זוגית אז f g אי-זוגית.
    טענה 1.5. תהיינה h 1 : A → B ו- h 2 : B → C פונקציות.
    • נניח h 1 זוגית, אם h 2 זוגית או ש- h 2 אי-זוגית אז h 2 ○ h 1 זוגית.
    • נניח ש- h 2 זוגית, אם h 1 זוגית או ש- h 1 אי-זוגית אז h 2 ○ h 1 זוגית.
    • אם h 1 ו- h 2 אי-זוגיות אז גם h 2 ○ h 1 אי-זוגית.
    טענה 1.6. תהיינה h 1 , h 2 : D → R שתי פונקציות המוגדרות ע”י (לכל x ∈ D ): h 1 ( x ) = f ( x ) - f ( - x ) h 2 ( x ) = f ( x ) + f ( - x ) h 1 אי-זוגית ו- h 2 זוגית.
    טענה 1.7. קיים זוג פונקציות יחיד f Odd , f Even : D → R כך ש- f Odd אי-זוגית ו- f Even זוגית ובנוסף מתקיים f = f Odd + f Even .
    פונקציות אלו הן הפונקציות המוגדרות ע”י (לכל x ∈ D ): f Odd ( x ) : = f ( x ) - f ( - x ) 2 f Even ( x ) : = f ( x ) + f ( - x ) 2

    1.2 טענות וזהויות טריגונומטריות

    1. ♣ לא נגדיר כאן את הפונקציות הטריגונומטריות ונסתמך על היחידה מעגל ע"פ הגדרתן כפי שלמדנו בתיכון כאשר את הזוויות נמדוד ברדיאנים.
    טענה 1.8. לכל θ ∈ R מתקיים | sin θ | ≤ 1 וגם | cos θ | ≤ 1 .
    טענה 1.9. לכל θ ∈ R מתקיים | sin θ | ≤ | θ | .
    טענה 1.10. לכל α , β , θ ∈ R מתקיימות הזהויות הבאות: sin ( π 2 - θ ) = cos θ cos ( π 2 - θ ) = sin θ sin ( α ± β ) = sin α ⋅ cos β ± cos α ⋅ sin β cos ( α ± β ) = cos α ⋅ cos β ∓ sin α ⋅ sin β sin α + sin β = 2 sin ( α + β 2 ) ⋅ cos ( α - β 2 ) cos α + cos β = 2 sin ( α - β 2 ) ⋅ cos ( α + β 2 ) sin α - sin β = 2 sin ( α - β 2 ) ⋅ cos ( α + β 2 ) cos α - cos β = - 2 sin ( α + β 2 ) ⋅ sin ( α - β 2 )
    טענה 1.11. sin היא פונקציה אי-זוגית ו- cos היא פונקציה זוגית.

2 גבול של פונקציה בנקודה

    משפט 2.1: יחידות הגבול
    לפונקציה f יש לכל היותר גבול אחד בנקודה a ∈ R .
    משפט 2.2: תהא f פונקציה המוגדרת בסביבה מנוקבת של נקודה a , הגבול lim x → a f ( x ) קיים אם”ם הגבול lim x → 0 f ( a + x ) קיים ובמקרה זה מתקיים גם: lim x → a f ( x ) = lim x → 0 f ( a + x )

    משפט 2.7: תנאי קושי לגבול של פונקציה בנקודה
    תהא f פונקציה המוגדרת בסביבה מנוקבת של נקודה a ∈ R .
    תנאי הכרחי ומספיק לקיום הגבול lim x → a f ( x ) הוא שלכל 0 < ε ∈ R קיימת 0 < δ ∈ R כך שלכל x 1 , x 2 ∈ B δ ○ ( a ) מתקיים
    | f ( x 1 ) - f ( x 2 ) | < ε .
    משפט 2.8: תנאי קושי לגבול חד-צדדי של פונקציה בנקודה
    תהא f פונקציה ותהא a ∈ R כך ש- f מוגדרת בסביבה ימנית/שמאלית מנוקבת של a .
    תנאי הכרחי ומספיק לקיום הגבול lim x → a + f ( x ) הוא שלכל 0 < ε ∈ R קיימת 0 < δ ∈ R כך שלכל x 1 , x 2 ∈ ( a , a + δ ) מתקיים | f ( x 1 ) - f ( x 2 ) | < ε .
    תנאי הכרחי ומספיק לקיום הגבול lim x → a - f ( x ) הוא שלכל 0 < ε ∈ R קיימת 0 < δ ∈ R כך שלכל x 1 , x 2 ∈ ( a - δ , a ) מתקיים | f ( x 1 ) - f ( x 2 ) | < ε .

    2.2 משפטים נוספים

    משפט 2.9: אריתמטיקה של גבולות
    תהיינה f ו- g פונקציות ממשיות המוגדרות בסביבה מנוקבת של a ∈ R כך שהגבולות הבאים קיימים: l : = lim x → a f ( x ) , m : = lim x → a g ( x ) מתקיימים כל הפסוקים הבאים:
    1. lim x → a ( f + g ) ( x ) = l + m .
    2. lim x → a ( f ⋅ g ) ( x ) = l ⋅ m .
    3. אם m ≠ 0 אז lim x → a 1 g ( x ) = 1 m .
    4. אם m ≠ 0 אז lim x → a f ( x ) g ( x ) = l m .
    1. ♣ המשפט נובע ישירות מאפיון היינה לגבול של פונקציה בנקודה ומאריתמטיקה של גבולות לסדרות.
    משפט 2.10: משפט ההצבה בגבולות לפונקציות לא רציפות
    תהיינה f : A → B ו- g : B → C שתי פונקציות כך שהגבולות l : = lim x → a f ( x ) ו- m : = lim y → l g ( y ) קיימים (עבור נקודה a ∈ A כלשהי), אם נתון בנוסף שקיימת סביבה U של a כך שלכל x ∈ U מתקיים f ( x ) ≠ l אז מתקיים lim x → a ( g ○ f ) ( x ) = m .

3 חסימות וסדר

    תהיינה f , g , h פונקציות המוגדרות בסביבה מנוקבת U של a ∈ R .
    טענה 3.1. נניח שהגבולות lim x → a f ( x ) ו- lim x → a g קיימים.
    1. אם לכל x ∈ U מתקיים f ( x ) < g ( x ) אז l ≤ m .
    2. אם lim x → a f ( x ) < lim x → a g ( x ) אז קיימת 0 < δ ∈ R כך ש- f ( x ) < g ( x ) לכל x ∈ B δ ° ( a ) .
    משפט 3.2: משפט הכריך
    אם f ( x ) ≤ g ( x ) ≤ h ( x ) לכל x ∈ U וגם lim x → a f ( x ) = l = lim x → a h ( x ) אז lim x → a g ( x ) = l .
    טענה 3.3. אם הגבול lim x → a f ( x ) קיים אז f חסומה מקומית ב- U .
    טענה 3.4. הגבול lim x → a f ( x ) קיים אם”ם קיים l ∈ R כך ש- lim x → a ( f ( x ) - l ) = 0 ובמקרה כזה l = lim x → a f ( x ) .
    משפט 3.5: כלל אפסה וחסומה
    אם lim x → a f ( x ) = 0 וגם g חסומה מקומית ב- a אז lim x → a ( f ⋅ g ) ( x ) = 0 .

4 רציפות

    4.1 משפטי רציפות

    משפט 4.1: אפיון היינה לרציפות של פונקציה בנקודה
    תהא f פונקציה המוגדרת בסביבה U של נקודה a ∈ R .
    f רציפה ב- a אם”ם לכל סדרה ( x n ) n = 1 ∞ המתכנסת ל- a שכל איבריה ב- U מתקיים lim n → ∞ f ( x n ) = f ( a ) .
    ‫מסקנה 4.2. אפיון היינה לרציפות חד-צדדית של פונקציה בנקודה
    תהא f פונקציה המוגדרת בסביבה ימנית/שמאלית U של נקודה a ∈ R .
    f רציפה מימין/משמאל ב- a אם”ם לכל סדרה ( x n ) n = 1 ∞ המתכנסת ל- a שכל איבריה ב- U מתקיים lim n → ∞ f ( x n ) = f ( a ) .
    משפט 4.3: אריתמטיקה של רציפות
    תהיינה f ו- g פונקציות רציפות בנקודה a ∈ R , מתקיימים שלושת הפסוקים הבאים:
    1. f + g רציפה ב- a .
    2. גם f ⋅ g רציפה ב- a
    3. אם g ( a ) ≠ 0 אז גם 1 g רציפה ב- a .
    4. אם g ( a ) ≠ 0 אז גם f g רציפה ב- a .
    משפט 4.4: משפט ההצבה בגבולות לפונקציות רציפות
    תהיינה f : B → C ו- g : A → B שתי פונקציות כך ש- g רציפה בנקודה פנימית a ∈ A ו- f רציפה ב- g ( a ) שהיא נקודה פנימית של B , הפונקציה f ○ g רציפה ב- a .
    1. ♣ המשפט נובע כמעט באופן ישיר ממשפט ההצבה בגבולות לפונקציות לא רציפות (משפט 2.10).
    2. ♣ מכאן שאם f רציפה ב- A ו- g רציפה ב- B אז g ○ f רציפה ב- A .
    משפט 4.5: תהא f פונקציה רציפה בנקודה a ∈ R ויהי y ∈ R ,
    • אם f ( a ) > y אז קיימת סביבה U של a כך ש- f ( x ) > y לכל x ∈ U .
    • אם f ( a ) < y אז קיימת סביבה U של a כך ש- f ( x ) < y לכל x ∈ U .
    1. ♣ המשפט נכון גם עבור רציפות חד-צדדית וסביבה חד-צדדית מתאימה.
    משפט 4.6: משפט ערך הביניים
    תהא f פונקציה רציפה בקטע סגור [ a , b ] ⊆ R , לכל y בקטע הסגור שבין f ( a ) ל- f ( b )

    קיים c ∈ [ a , b ] כך ש- f ( c ) = y .

5 גבולות במובן הרחב

    1. ♣ כמעט כל המשפטים הבאים הם גרסאות של המשפטים המתאימים לגבול של פונקציה בנקודה והוכחותיהם דומות מאד לאלו שכבר ראינו.
    משפט 5.1: תהא f פונקציה, הגבול lim x → ± ∞ f ( x ) קיים במובן הרחב אם”ם הגבול lim x → ∓ ∞ f ( - x ) קיים במובן הרחב ובמקרה זה מתקיים גם: lim x → ∓ ∞ f ( - x ) = lim x → ± ∞ f ( x )

    ‫מסקנה 8.24. יהי 0 < a ∈ R ונגדיר את הפונקציה f : R → ( 0 , ∞ ) ע”י f ( x ) = a x (לכל x ∈ R );
    • אם 1 < a אז f עולה ממש.
    • אם a < 1 אז f יורדת ממש.
    • אם a = 1 אז f פונקציה קבועה ( f ( x ) = 1 לכל x ∈ R ).
    מכאן נסיק שאם a ≠ 1 אז f חח”ע ועל וממילא הפיכה.
    ‫מסקנה 8.25. יהי 0 < a ∈ R שונה מ- 1 , אם 1 < a אז log a עולה ממש ואם a < 1 אז log a יורדת ממש.
    ‫מסקנה 8.26. יהי 0 < a ∈ R שונה מ- 1 .
    • אם 1 < a אז lim x → ∞ a x = ∞ ו- lim x → - ∞ a x = 0 , וכמו כן lim x → ∞ log a ( x ) = ∞ ו- lim x → 0 + log a ( x ) = - ∞ .
    • אם a < 1 אז lim x → ∞ a x = 0 ו- lim x → - ∞ a x = - ∞ , וכמו כן lim x → ∞ log a ( x ) = - ∞ ו- lim x → 0 + log a ( x ) = ∞ .

    משפט 8.28: חוקי לוגריתמים

9 רציפות במידה שווה

    תהא f פונקציה המוגדרת על מקטע I ⊆ R .
    טענה 9.1. אם f רציפה במידה שווה על I אז f רציפה במידה שווה על כל מקטע J ⊆ I .
    משפט 9.2: אם f רציפה במידה שווה על I אז f גם רציפה ב- I .
    משפט 9.3: אם f רציפה במידה שווה על I אז לכל סדרה מתכנסת ( a n ) n = 1 ∞ שכל איבריה ב- I גם הסדרה ( f ( a n ) ) n = 1 ∞ היא סדרה מתכנסת.
    1. ♣ שימו לב: העובדה ש- ( a n ) n = 1 ∞ מתכנסת ושכל איבריה ב- I אינה אומרת שגם lim n → ∞ a n נמצא ב- I , מסיבה זו המשפט אינו נכון לכל פונקציה רציפה.
      לדוגמה: הפונקציה f ( x ) = 1 x רציפה בקטע ( 0 , 1 ] ו- a n = 1 n היא סדרה מתכנסת שכל איבריה ב- ( 0 , 1 ] אך lim n → ∞ f ( a n ) = ∞ .
    משפט 9.4: “אפיון היינה” לרציפות במידה שווה של פונקציה
    f רציפה במידה שווה על I אם”ם לכל שתי סדרות ( x n ) n = 1 ∞ ו- ( y n ) n = 1 ∞ שכל איבריהן ב- I המקיימות lim n → ∞ | x n - y n | = 0 מתקיים lim n → ∞ | f ( x n ) - f ( y n ) | = 0 .
    משפט 9.5: אריתמטיקה של רציפות במידה שווה
    נניח ש- f רציפה במידה שווה על I ותהא g גם היא פונקציה רציפה במידה שווה על I (ובפרט מוגדרת בו).
    1. f + g רציפה במידה שווה על I .
    2. אם f ו- g חסומות ב- I אז f ⋅ g רציפה במידה שווה על I .
    3. אם קיים 0 < c ∈ R כך ש- | g ( x ) | ≥ c לכל x ∈ I אז 1 g רציפה במידה שווה על I .
    4. אם f חסומה ב- I וגם קיים 0 < c ∈ R כך ש- | g ( x ) | ≥ c לכל x ∈ I אז f g רציפה במידה שווה על I .
    משפט 9.6: נניח ש- f רציפה במידה שווה על I ותהא g : Im f → R גם היא פונקציה רציפה במידה שווה על Im f
    10
    מהעובדה ש- I הוא מקטע ו- f רציפה (משפט 9.2) נובע שגם Im f הוא מקטע.
    , גם הפונקציה g ○ f רציפה במידה שווה על I .
    משפט 9.7: נניח ש- I הוא קטע פתוח ויהיו a , b ∈ R כך ש- I = ( a , b ) , אם f רציפה במידה שווה על I אז הגבולות lim x → a + f ( x ) ו- lim x → b - f ( x ) קיימים.
    ‫מסקנה 9.8. “משפט ויירשטראס” לרציפות במידה שווה
    נניח ש- I הוא קטע פתוח, אם f רציפה במידה שווה על I אז f חסומה ב- I .
    ‫מסקנה 9.9. נניח ש- f רציפה במידה שווה על I ,
    • אם I = ( a , ∞ ) (עבור a ∈ R כלשהו) אז הגבול lim x → a - f ( x ) קיים
    • אם I = ( - ∞ , a ) (עבור a ∈ R כלשהו) אז הגבול lim x → a + f ( x ) קיים.
    משפט 9.10: תהא g : R → R פונקציה רציפה, אם ל- g יש אסימפטוטות משופעות ב- ± ∞ אז g רציפה במידה שווה על כל R .
    ‫מסקנה 9.11. נניח ש- f רציפה ב- I ,
    • אם I היא קרן ימנית ול- f יש אסימפטוטה משופעת ב- ∞ אז f רציפה במידה שווה על I .
    • אם I היא קרן שמאלית ול- f יש אסימפטוטה משופעת ב- - ∞ אז f רציפה במידה שווה על I .
    משפט 9.12: משפט קנטור

    נניח ש- I הוא קטע סגור, אם f רציפה ב- I אז f רציפה במידה שווה על I .
