1 גבול של סדרה

    ‫הגדרה 1.1. תהא ( P n ) n = 1 ∞ סדרת פסוקים לוגיים.
    • נאמר ש- P n מתקיימת באופן שכיח אם לכל N ∈ N קיים N < n ∈ N כך ש- P n = True .
    • נאמר ש- P n מתקיימת ממקום מסוים ואילך/ עבור n גדול דיו/ כמעט תמיד אם קיים N ∈ N כך שלכל N < n ∈ N מתקיים P n = True .
    1. ♣ מכאן ואילך כל הסדרות שנדבר עליהן הן סדרות אינסופיות של מספרים ממשיים אלא אם נכתב אחרת.
    תזכורת:
    הגדרנו את הכדור הפתוח שרדיוסו r ∈ R מסביב לנקודה α ∈ R ע”י B r ( α ) : = ( α - r , α + r ) .
    ‫הגדרה 1.2. גבול של סדרה
    • נאמר ש- L ∈ R הוא גבול של סדרה ( a n ) n = 1 ∞ אם לכל 0 < ε ∈ R ועבור n גדול דיו מתקיים a n ∈ B ε ( L ) ,
      כלומר לכל 0 < ε ∈ R קיים N ∈ N כך שלכל N < n ∈ N מתקיים | a n - L | < ε .
    • בקובץ הטענות נראה שגבול של סדרה הוא יחיד, כלומר אם α ∈ R וגם β ∈ R הם גבולות
      1
      המשפט אינו נכון מבחינה לשונית משום שמדובר במספר אחד ולכן לא שייך לדבר עליו ברבים, אך הכוונה ברורה ולפני שהמשכנו ואמרנו שהם שווים לא ידענו שבהכרח מדובר באותו מספר (לא יכולתי להתאפק...).
      של ( a n ) n = 1 ∞ אז α = β ; לכן מוצדק לדבר על הגבול של ( a n ) n = 1 ∞ ולכתוב: lim n → ∞ a n = L , a n → n → ∞ L
    • סדרה תקרא מתכנסת אם יש לה גבול, אחרת תקרא מתבדרת.
      כלומר:
      • סדרה תקרא מתכנסת אם קיים L ∈ R כך שלכל 0 < ε ∈ R קיים N ∈ N כך שלכל N < n ∈ N מתקיים | a n - L | < ε , במקרה כזה נאמר שהסדרה מתכנסת ל- L .
      • סדרה תקרא מתבדרת אם לכל L ∈ R קיים 0 < ε ∈ R כך שלכל N ∈ N קיים N < n ∈ N כך שמתקיים | a n - L | ≥ ε .
    הסכמה:
    נסכים שבכל מקום שבו נכתב ביטוי מהצורה lim n → ∞ a n = L או L = lim n → ∞ a n הכוונה היא שהגבול קיים ומתקיים השוויון.

2 חסימות וסדר

    ‫הגדרה 2.1. סדרות חסומות מלעיל/מלרע
    • סדרה תקרא חסומה/חסומה מלעיל/חסומה מלרע אם קבוצת איבריה כזו.
    • מספר ממשי ייקרא חסם מלעיל/מלרע של סדרה אם הוא חסם מלעיל/מלרע של קבוצת איבריה.
    • מספר ממשי ייקרא חסם עליון/תחתון של סדרה אם הוא חסם מלעיל/מלרע של קבוצת איבריה.
    ‫הגדרה 2.2. סדרה ( a n ) n = 1 ∞ תקרא סדרה חיובית/שלילית/אי-שלילית/אי-חיובית אם כל איבריה חיוביים/שליליים/אי-שליליים/אי-חיוביים.

3 אריתמטיקה של גבולות

    ‫הגדרה 3.1. סדרה ( a n ) n = 0 ∞ תקרא סדרה חשבונית אם קיים d ∈ R כך שלכל n ∈ N 0 מתקיים a n + 1 = a n + d , d כזה הוא יחיד וייקרא הפרש הסדרה.
    ‫הגדרה 3.2. סדרה ( a n ) n = 0 ∞ תקרא סדרה הנדסית אם קיים q ∈ R כך שלכל n ∈ N 0 מתקיים a n + 1 = a n ⋅ q , q כזה הוא יחיד וייקרא מנת הסדרה.

4 גבולות במובן הרחב

    ‫הגדרה 4.1. שאיפה ל- ± ∞
    • סדרה ( a n ) n = 1 ∞ תקרא שואפת לאין-סוף אם לכל M ∈ R ועבור n גדול דיו מתקיים a n > M ,
      כלומר קיים N ∈ N כך שלכל N < n ∈ N מתקיים a n > M .
      נסמן: lim n → ∞ a n = ∞
    • סדרה ( a n ) n = 1 ∞ תקרא שואפת למינוס אין-סוף אם לכל m ∈ R ועבור n גדול דיו מתקיים a n < m ,
      כלומר קיים N ∈ N כך שלכל N < n ∈ N מתקיים a n < m .
      נסמן: lim n → ∞ a n = - ∞
    • אם סדרה שואפת ל- ± ∞ נאמר שהיא מתכנסת במובן הרחב וש- ± ∞ הוא גבול שלה במובן הרחב.
    1. ♣ חשוב לשים לב: סדרות השואפות לאין-סוף / למינוס אין-סוף הן סדרות מתבדרות.

5 מונוטוניות

    ‫הגדרה 5.1. סדרות מונוטוניות
    • סדרה ( a n ) n = 1 ∞ תקרא מונוטונית עולה אם לכל n ∈ N מתקיים a n + 1 ≥ a n .
    • סדרה ( a n ) n = 1 ∞ תקרא עולה ממש
      2
      יש אומרים מונוטונית עולה ממש וכן לגבי מונוטונית יורדת ממש.
      אם לכל n ∈ N מתקיים a n + 1 > a n .
    • סדרה ( a n ) n = 1 ∞ תקרא מונוטונית יורדת אם לכל n ∈ N מתקיים a n + 1 ≤ a n .
    • סדרה ( a n ) n = 1 ∞ תקרא יורדת ממש אם לכל n ∈ N מתקיים a n + 1 < a n .

6 תתי-סדרות וגבולות חלקיים

    ‫הגדרה 6.1. תהא ( a n ) n = 1 ∞ סדרה; נאמר שסדרה ( b k ) k = 1 ∞ היא תת-סדרה של ( a n ) n = 1 ∞ , אם קיימת סדרה ( n k ) k = 1 ∞ עולה ממש שכל איבריה טבעיים (סדרת אינדקסים) כך שלכל k ∈ N מתקיים b k = a n k .
    ‫הגדרה 6.2. תהא ( a n ) n = 1 ∞ סדרה.
    • נאמר ש- α ∈ R הוא גבול חלקי של ( a n ) n = 1 ∞ אם יש ל- ( a n ) n = 1 ∞ תת-סדרה ש- α הוא הגבול שלה,
      כלומר קיימת סדרה ( n k ) k = 1 ∞ עולה ממש שכל איבריה טבעיים (סדרת אינדקסים) כך ש- lim k → ∞ a n k = α .
    • כמו כן נאמר ש- ± ∞ הוא גבול חלקי של ( a n ) n = 1 ∞ אם יש ל- ( a n ) n = 1 ∞ תת-סדרה השואפת ל- ± ∞ ,
      כלומר קיימת סדרה ( n k ) k = 1 ∞ עולה ממש שכל איבריה טבעיים (סדרת אינדקסים) כך ש- lim k → ∞ a n k = ± ∞ .
    ‫הגדרה 6.3. לכל סדרה ( a n ) n = 1 ∞ , אינדקס n ∈ N יקרא אינדקס שיא אם לכל n < k ∈ N מתקיים a k ≤ a n .
