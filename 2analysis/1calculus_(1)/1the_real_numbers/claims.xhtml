    משפט 6.10: למת החתכים
    תהיינה ∅ ≠ A , B ⊆ R כך ש- a ≤ b לכל a ∈ A ולכל b ∈ B , הפסוקים הבאים שקולים:
    1. קיים M ∈ R יחיד כך ש- a ≤ M ≤ b לכל a ∈ A ולכל b ∈ B .
    2. sup A = inf B .
    3. לכל 0 < ε ∈ R קיימים a ∈ A ו- b ∈ B כך ש- b - a < ε .
    4. לכל 0 < ε ∈ R קיים a ∈ A כך ש- 0 ≤ inf B - a < ε .
    5. לכל 0 < ε ∈ R קיים b ∈ B כך ש- 0 ≤ b - sup A < ε .
    טענה 6.11. תהיינה ∅ ≠ A , B ⊆ R , מתקיים:

    • יהי 0 ≤ β ∈ R ; אם A חסומה מלעיל אז sup ( β ⋅ A ) = β ⋅ sup ( A ) , ואם A חסומה מלרע אז inf ( β ⋅ A ) = β ⋅ inf ( A ) .
    • אם A , B ⊆ [ 0 , ∞ ) אז inf ( A ⋅ B ) = inf ( A ) ⋅ inf ( B ) , ואם (בנוסף) A ו- B חסומות מלעיל אז sup ( A ⋅ B ) = sup ( A ) ⋅ sup ( B ) .
    • אם A , B ⊆ ( - ∞ , 0 ] אז inf ( A ⋅ B ) = sup ( A ) ⋅ sup ( B ) , ואם (בנוסף) A ו- B חסומות מלרע אז sup ( A ⋅ B ) = inf ( A ) ⋅ inf ( B ) .

    לכל 0 < x ∈ R ולכל n ∈ N קיים 0 < y ∈ R יחיד כך ש- y n = x .
    משפט 7.6: האי-רציונליים צפופים בממשיים
    לכל x , y ∈ R כך ש- x < y קיים γ ∈ R ∖ Q כך ש- x < γ < y .
    ‫למה 7.7. יהי 0 < x ∈ N , לכל n , k ∈ N ו- m , j ∈ Z המקיימים m n = k j מתקיים ( x m ) 1 n = ( x j ) 1 k .
    משפט 7.8: חוקי חזקות כשהמעריך רציונלי
    יהיו 0 < x , y ∈ R ו- q , r ∈ Q , מתקיימים כל הפסוקים הבאים:
    1. x q ⋅ x r = x q + r .
    2. ( x q ) r = x q ⋅ r .
    3. ( x ⋅ y ) q = x q ⋅ y q .
    4. אם 0 < x < y ו- 0 < q אז x q < y q , כמו כן אם 0 < x < y ו- 0 > q אז x q > y q .
    5. אם 1 < x ו- q < r אז x q < x r .
    6. אם 0 < x < 1 ו- q < r אז x q > x r .
    1. ♣ חוקי חזקות אלו נכונים גם כשהמעריך ממשי.
    משפט 7.9: לכל a , b ∈ F ולכל n ∈ N 0 מתקיים: a n - b n = ( a - b ) ⋅ ∑ k = 0 n - 1 a n - 1 - k ⋅ b k
    1. ♣ משפט זה הוא הכללה של נוסחת הכפל המקוצר a 2 - b 2 = ( a + b ) ( a - b ) .
