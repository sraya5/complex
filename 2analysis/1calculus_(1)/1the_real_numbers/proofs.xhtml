    משפט 3.6: הוכחה באינדוקציה
    לכל n ∈ N יהי P ( n ) פסוק לוגי, אם P ( 1 ) = True וגם לכל n ∈ N מתקיים P ( n ) → P ( n + 1 ) אז P ( n ) = True לכל n ∈ N .
    1. ♣ זהו הסוג הקלאסי של הוכחה באינדוקציה אך ישנם סוגים נוספים, ראו בערך “מתמטית אינדוקציה” בוויקיפדיה.
    הוכחה. נגדיר I : = { n ∈ N ∣ P ( n ) = True } (מהגדרה I ⊆ N ), נתון ש- P ( 1 ) = True ולכן 1 ∈ I ; יהי n ∈ I , נתון שאם P ( n ) = True אז גם P ( n + 1 ) = True , כלומר אם n ∈ I אז גם n + 1 ∈ I . מכאן ש- I היא קבוצה אינדוקטיבית ומכיוון ש- I ⊆ N נדע ש- I = N , כלומר לכל n ∈ N מתקיים P ( n ) = True .
