1 יריעות פרמטריות

    תהיינה X ⊆ R n יריעה k -ממדית ו- ϕ : U → R n פרמטריזציה של X .
    טענה 1.1. לכל העתקה ליניארית T : R k → R n כך ש- k ≤ n מתקיים V ( T ) = det ( T * ○ T )
    1
    T * היא ההעתקה הצמודה של T (הוגדרה בליניארית 2), אין שום עניין לזכור זאת כאן - מי שלא זוכר יכול להסתפק בעובדה ש- T * היא ההעתקה הליניארית המקיימת [ T * ] B C = ( [ T ] C B ) t לכל בחירת בסיסים B ו- C של R k ו- R n בהתאמה.
    , וכמו כן לכל מטריצה A ∈ M n × k ( R ) מתקיים V ( A ) = det ( A t ⋅ A ) .
    ‫מסקנה 1.2. יהי ψ : V → U דיפאומורפיזם, ותהא f : R n → R ; מתקיים: ∫ U f ( ϕ ( x ) ) ⋅ V ( D ϕ x ) d x = ∫ V f ( ϕ ( ψ ( y ) ) ) ⋅ V ( D ( ϕ ○ ψ ) y ) d y כלומר אם אחד האינטגרלים מוגדר אז גם האחר מוגדר והם שווים.
    משפט 1.3: נניח ש- ( X , ϕ ) רגולרית בנקודה x 0 ∈ U , במקרה כזה קיימים:
    1. סביבה פתוחה V ⊆ U של x 0 ( x 0 ∈ U )
    2. נקודה p ∈ R k , 0 < r ∈ R ודיפאומורפיזם ψ : B ε ( p ) → V
      2
      למעשה כל נקודה ב- R k תתאים מפני שיש דיפאומורפיזם בין B ε ( p ) ל- B ε ( q ) לכל שתי נקודות p , q ∈ R k .
    3. פונקציה חלקה h : B ε ( p ) → R n - k
    4. העתקה ליניארית אורתוגונלית A : R n → R n
    כך ש- A ( ( ϕ ○ ψ ) ( x 0 ) ) = p ו- ( x , h ( x ) ) = A ( ( ϕ ○ ψ ) ( x ) ) לכל x ∈ B ε ( p ) .
    1. ♣ כלומר X נראית כמו גרף של פונקציה חלקה בסביבת ϕ ( x ) , זהו האפיון הכי ברור לכך ש- X “חלקה” בסביבה זו.

2 יריעות
    k -ממדיות ב- R n

  1. ♣ המשפטים הבאים מראים שלמרות שההגדרה של יריעה k -ממדית ב- R n הזכירה פרמטריזציה מקומית, ההגדרה אינה תלויה בה מפני שכל הפרמטריזציות המקומיות שקולות זו לזו.
משפט 2.1: תהא M ⊆ R n , M היא יריעה חלקה k -ממדית אם”ם לכל נקודה p ∈ M קיימים:
  1. סביבה פתוחה W ⊆ M
    3
    שוב נזכיר ש- W פתוחה ב- M ולאו דווקא ב- R n , המשמעות היא שקיימת קבוצה פתוחה U ⊆ R n כך ש- W = U ∩ M .
    של p ( p ∈ W )
  2. נקודה x 0 ∈ R k , 0 < r ∈ R ופונקציה חלקה h : B ε ( x 0 ) → R n - k
  3. העתקה ליניארית אורתוגונלית A : R n → R n
כך שההעתקה α : B r ( x 0 ) → R n המוגדרת ע”י α ( x ) : = A ( x , h ( x ) ) (לכל x ∈ B r ( x 0 ) ) מקיימת α ( B r ( x 0 ) ) = W .
תהא M ⊆ R m יריעה חלקה k -ממדית, ותהא p ∈ M .
משפט 2.2: תהיינה α : U 1 → W 1 ו- β : U 2 → W 2 שתי פרמטריזציות מקומיות של M בסביבת p .
נסמן: W : = W 1 ∩ W 2 , U 1 ' : = α - 1 ( W ) ו- U 2 ' : = β - 1 ( W ) ; א”כ W היא סביבה פתוחה של p ב- M , U 1 ' ו- U 2 ' הן קבוצות פתוחות ב- R k , ו- β - 1 ○ α היא דיפאומורפיזם בין U 1 ' ל- U 2 ' . בפרט ( W , β ∣ U 2 ' ) היא רה-פרמטריזציה של ( W , α ∣ U 1 ' ) ולהפך.
‫מסקנה 2.3. תהיינה α : U 1 → W 1 ו- β : U 2 → W 2 שתי פרמטריזציות מקומיות של M בסביבת p , ונסמן x : = α - 1 ( p ) ו- y : = β - 1 ( p ) ; מתקיים: Im ( D α x ) = Im ( D β y )
‫מסקנה 2.4. תהא α : U → W פרמטריזציה מקומית של M בסביבת p . קיימות סביבה פתוחה W ˜ של p ב- R n (כאן W ˜ פתוחה ב- R m ולא ב- M ), והעתקה חלקה ψ : W ˜ → U , כך שלכל x ∈ α - 1 ( W ˜ ) מתקיים ψ ( α ( x ) ) = x (כלומר ב- α - 1 ( W ˜ ) מתקיים ψ ○ α = Id ).
תהא N ⊆ R n יריעה חלקה l -ממדית.
טענה 2.5. תהיינה f : M → N פונקציה, α : U → W פרמטריזציה מקומית של M בסביבת p , ו- β : A → B פרמטריזציה מקומית של N בסביבת f ( p ) .
f חלקה ב- p אם”ם קיימת סביבה פתוחה U ' ⊆ U של α - 1 ( p ) כך שהפונקציה β - 1 ○ f ○ α ∣ U ' היא העתקה חלקה.
  1. ♣ חשיבות של טענה זו בכך שהיא נותנת אפיון להיותה של פונקציה חלקה בין יריעות מבלי להזדקק להרחבה מקומית שלה.
טענה 2.6. תהא f : M → N פונקציה חלקה ב- p , ותהיינה g : V → R n ו- h : W → R n הרחבות מקומיות של f בסביבת p , מתקיים: D g ∣ T p ( M ) = D h ∣ T p ( M ) ובנוסף הצמצום של D g ו- D h ל- T p ( M ) הוא העתקה ליניארית מ- T p ( M ) ל- T f ( p ) ( N ) .
משפט 2.7: תהא f : M → N פונקציה חלקה, ותהא q ∈ N כך ש- f - 1 ( { q } ) ≠ ∅ (כלומר q ∈ Im f ).
אם q היא ערך רגולרי של f אז f - 1 ( { q } ) היא יריעה ( k - l ) -ממדית ב- R n .
אני מנחש שלכל יריעה d -ממדית N ' ⊆ N ⊆ R n כך שכל נקודה ב- N ' היא ערך רגולרי, נקבל ש- f - 1 ( N ' ) היא יריעה ( k - l + d ) -ממדית.
