1 התחלה

    1. ♣ בקובץ ההגדרות הגדרנו מסילה רגולרית כדי להימנע ממצב שבו מסילה נתונה היא אכן חלקה אך תמונתה אינה כזו. כמו כן הבאנו שם דוגמה למסילה חלקה שאינה רגולרית ותמונתה אינה “חלקה” (מבחינה אינטואיטיבית), אך לא הראינו שהתמונה של כל מסילה רגולרית היא אכן “חלקה”.
    משפט 1.1: תהא γ : [ a , b ] → R n מסילה חלקה, ונניח ש- γ רגולרית בנקודה t ∈ [ a , b ] , במקרה כזה קיימים:
    1. קטע סגור I ⊆ [ a , b ] כך ש- t ∈ I
    2. דיפאומורפיזם ϕ : [ c , d ] → I
    3. פונקציה חלקה h : [ c , d ] → R n - 1
    4. העתקה ליניארית אורתוגונלית A : R n → R n
    כך ש- ( x , h ( x ) ) = A ( ( γ ○ ϕ ) ( x ) ) לכל x ∈ [ c , d ] .
    1. ♣ כלומר γ * נראית כמו גרף של פונקציה חלקה בסביבת γ ( t ) , זהו האפיון הכי ברור לכך ש- γ * “חלקה” בסביבה זו.

2 האינטגרל המסילתי

    משפט 2.1: תהא γ : [ a , b ] → R n מסילה חלקה בעלת אורך, ותהא f : R n → R פונקציה רציפה. האינטגרל ∫ γ f d s קיים, ומתקיים: ∫ γ f d s = ∫ a b f ( γ ( t ) ) ⋅ ∥ γ ' ( t ) ∥ d t ובפרט: L [ γ ] = ∫ a b ∥ γ ' ( t ) ∥ d t
    ‫מסקנה 2.2. תהיינה γ : [ a , b ] → R n ו- μ : [ c , d ] → R n מסילות חלקות המתקבלות זו מזו ע”י רה-פרמטריזציה, לכל פונקציה רציפה f : R n → R מתקיים: ∫ γ f d s = ∫ μ f d s L [ γ ] = L [ μ ]
    1. ♣ כלומר הפרמטריזציה אינה משפיע על האינטגרל של פונקציה לאורך עקומה, וכן אינה משפיע על אורך העקומה; א”כ מושגים אלו הוגדרו באופן הרצוי.
    משפט 2.3: תהא γ : [ a , b ] → R n מסילה חלקה, ותהא φ : [ 0 , L [ γ ] ] → [ a , b ] פונקציה המוגדרת ע”י (לכל x ∈ [ 0 , L [ γ ] ] ): φ ( x ) : = ∫ 0 x ∥ γ ' ( t ) ∥ d t φ היא רה-פרמטריזציה של [ a , b ] , ולכל x ∈ [ 0 , L [ γ ] ] מתקיים ∥ ( γ ○ φ ) ' ( x ) ∥ = 1 .

3 האינטגרל הקווי

    משפט 3.1: תהא γ : [ a , b ] → R n מסילה חלקה בעלת אורך, ויהי X → : U → R n שדה וקטורי. האינטגרל ∫ γ X → ⋅ d ℓ → קיים, ומתקיים: ∫ γ X → ⋅ d ℓ → = ∫ a b X → ( γ ( t ) ) ⋅ γ ' ( t ) d t
    1. ♣ אם γ היא מסילה רגולרית וחח”ע, נוכל להגדיר פונקציה T → : γ * → R n כך שלכל t ∈ [ a , b ] יתקיים T → ( γ ( t ) ) = γ ' ( t ) ∥ γ ( t ) ∥ , ואם נצליח להגדיר גם פונקציה רציפה h : R n → R כך שלכל p ∈ γ * מתקיים h ( p ) = X → ( p ) ⋅ T → ( p ) , אז נקבל: ∫ γ X → ⋅ d ℓ → = ∫ a b X → ( γ ( t ) ) ⋅ γ ' ( t ) d t = ∫ a b X → ( γ ( t ) ) ⋅ γ ' ( t ) ∥ γ ' ( t ) ∥ ⋅ ∥ γ ' ( t ) ∥ d t = ∫ a b h ( γ ( t ) ) ⋅ ∥ γ ' ( t ) ∥ d t = ∫ γ h d s ולהפך: אם נתונה פונקציה רציפה f : R n → R נוכל לנסות להגדיר שדה וקטורי F → : R n → R n כך שלכל p ∈ γ * מתקיים F → ( p ) ⋅ T → ( p ) = f ( p ) , ואז נקבל: ∫ γ f d s = ∫ a b f ( γ ( t ) ) ⋅ ∥ γ ' ( t ) ∥ d t = ∫ a b F → ( γ ( t ) ) ⋅ γ ' ( t ) ∥ γ ' ( t ) ∥ ⋅ ∥ γ ' ( t ) ∥ d t = ∫ a b F → ( γ ( t ) ) ⋅ γ ' ( t ) d t = ∫ γ F → ⋅ d ℓ →
    ‫מסקנה 3.2. תהיינה γ : [ a , b ] → R n ו- μ : [ c , d ] → R n מסילות חלקות המתקבלות זו מזו ע”י רה-פרמטריזציה, לכל שדה וקטורי X → : U → R n מתקיים: ∫ γ X → ⋅ d ℓ → = ∫ μ X → ⋅ d ℓ →
    1. ♣ כלומר הפרמטריזציה אינה משפיע על האינטגרל של פונקציה לאורך עקומה, וכן אינה משפיע על אורך העקומה; א”כ מושגים אלו הוגדרו באופן הרצוי.

4 שדות משמרים

    משפט 4.1: יהי F → : U → R n שדה וקטורי, F → הוא שדה וקטורי משמר אם”ם קיימת פונקציה חלקה ϕ : U → R כך ש- F → = ∇ ϕ
    1
    כלומר F → ( x ) = ∇ ϕ ( x ) לכל x ∈ U .
    .
    1. ♣ ϕ כנ”ל נקראת פוטנציאל של F → , כמובן שאין ל- F → פוטנציאל יחיד - כל הוספת קבוע ל- ϕ תיתן פוטנציאל נוסף (בהמשך נראה שאלו כל הפוטנציאלים של F → ).
    2. ♣ במובן מסוים פוטנציאל של שדה משמר הוא פונקציה קדומה שלו.
    תזכורת:
    בהינתן פונקציה f : R k → R n הגדרנו את הפונקציות f 1 , f 2 , … , f n מ- R k ל- R כך שלכל x ∈ R k יתקיים: f ( x ) = [ f 1 ( x ) f 2 ( x ) ⋮ f n ( x ) ]
    ‫מסקנה 4.2. יהי F → : U → R n שדה וקטורי משמר, לכל n ≥ i , j ∈ N מתקיים: ∂ F i ∂ x j = ∂ F j ∂ x i
    משפט 4.3: יהי F → : U → R שדה וקטורי המקיים (לכל n ≥ i , j ∈ N ): ∂ F i ∂ x j = ∂ F j ∂ x i ותהיינה γ 0 , γ 1 : [ a , b ] → U מסילות הומוטופיות ב- U ביחס לנקודות קצה. מתקיים: ∫ γ 0 F → ⋅ d ℓ → = ∫ γ 1 F → ⋅ d ℓ →
    ‫מסקנה 4.4. יהי F → : U → R שדה וקטורי המקיים (לכל n ≥ i , j ∈ N ): ∂ F i ∂ x j = ∂ F j ∂ x i אם U היא קבוצה פשוטת קשר אז F → הוא שדה משמר, בפרט לכל נקודה p ∈ U קיימת סביבה W ⊆ U של p כך ש- F → ∣ W הוא שדה משמר.
    1. ♣ החלק האחרון של המסקנה הוא הסיבה לכך ששדה וקטורי כנ”ל נקרא משמר מקומית.
