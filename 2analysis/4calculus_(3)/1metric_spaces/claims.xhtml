1 מרחבים מטריים ומרחבים נורמיים

    1.1 מרחבים מטריים

    משפט 1.1: אי-שוויון המשולש ההפוך
    יהי ( X , d ) מרחב מטרי, לכל x , y , z ∈ X מתקיים | d ( x , z ) - d ( y , z ) | ≤ d ( x , y ) .

    1.2 מרחבים נורמיים

    טענה 1.2. יהיו ( V , ∥ ⋅ ∥ V ) ו- ( W , ∥ ⋅ ∥ W ) מרחבים נורמיים מעל אותו שדה, ותהא T : V → W העתקה ליניארית כך שהנורמה ∥ T ∥ op מוגדרת; לכל v ∈ V מתקיים ∥ T ( v ) ∥ W ≤ ∥ T ∥ op ⋅ ∥ v ∥ V .
    טענה 1.3. יהיו ( V , ∥ ⋅ ∥ V ) , ( W , ∥ ⋅ ∥ W ) ו- ( U , ∥ ⋅ ∥ U ) מרחבים נורמיים מעל אותו שדה, ותהיינה T 1 : V → W ו- T 2 : W → U העתקות ליניאריות כך ש- ∥ T 1 ∥ op ו- ∥ T 2 ∥ op מוגדרות; מתקיים ∥ T 2 ○ T 1 ∥ op ≤ ∥ T 2 ∥ op ⋅ ∥ T 1 ∥ op .

2 קבוצות במרחב מטרי

    יהי ( X , d ) מרחב מטרי.

    2.1 חסימות

    ‫למה 2.1. יהיו x , y ∈ X ויהיו 0 < r , s ∈ R , מתקיימים ארבעת הפסוקים הבאים:
    1. y ∈ B r ( x ) אם”ם x ∈ B r ( y ) .
    2. y ∈ B r ˆ ( x ) אם”ם x ∈ B r ˆ ( y ) .
    3. אם y ∈ B ε ( x ) לכל 0 < ε ∈ R אז x = y .
    4. אם d ( x , y ) + s ≤ r אז B s ( y ) ⊆ B r ( x ) ו- B s ( x ) ⊆ B r ( y ) .
    1. ♣ סעיפים 1 ו-2 נכונים גם עבור ספירות.
    טענה 2.2. תהא Y ⊆ X קבוצה חסומה, לכל x ∈ X קיים 0 < r ∈ R כך ש- Y ⊆ B r ( x ) .

    2.2 פתיחות וסגירות

    ‫למה 2.3. יהי x ∈ X , כל כדור פתוח שמרכזו ב- x מכיל כדור סגור שמרכזו ב- x .
    1. ♣ כמובן שגם כדור סגור מכיל כדור פתוח ( B r ( x ) ⊆ B r ˆ ( x ) ), החידוש הוא שגם כדור פתוח מכיל כדור סגור.
    ‫למה 2.4. תהא Y ⊆ X ויהיו y ∈ Y ו- 0 < r ∈ R , מתקיים: B r Y ( y ) = B r X ( y ) ∩ Y B r Y ˆ ( y ) = B r X ˆ ( y ) ∩ Y S r Y ( y ) = S r X ( y ) ∩ Y
    ‫מסקנה 2.5. תהא Y ⊆ X ותהא A ⊆ Y , A היא קבוצה פתוחה ב- Y (כלומר במרחב המטרי ( Y , d Y ) ) אם”ם קיימת קבוצה פתוחה U ⊆ X (פתוחה ב- X ) כך ש- A = U ∩ Y .
    1. ♣ מסקנה מיידית היא שהמשפט נכון גם עבור קבוצות סגורות: תהא Y ⊆ X ותהא A ⊆ Y , A היא קבוצה סגורה ב- Y (כלומר במרחב המטרי ( Y , d Y ) ) אם”ם קיימת קבוצה סגורה C ⊆ X (סגורה ב- X ) כך ש- A = C ∩ Y .
    טענה 2.6. כל כדור פתוח הוא קבוצה פתוחה, וכל כדור סגור הוא קבוצה סגורה.
    טענה 2.7. תהא A קבוצה של קבוצות פתוחות ב- X , האיחוד של כל הקבוצות ב- A הוא קבוצה פתוחה ב- X .
    1. ♣ נשים לב לכך שיש כאן כמה אפשרויות:
      • A יכולה להיות סופית ואז קיימות U 1 , U 2 , … , U r ⊆ X כך ש- A = { U 1 , U 2 , … , U r } , ואז האיחוד של כל הקבוצות שבה בה הוא הקבוצה: ⋃ i = 1 r U i
      • A יכולה להיות אין-סופית בת-מנייה, כלומר ניתן לסדר את איבריה בסדרה אינסופית: A = { U 1 , U 2 , … } , ואז האיחוד של כל הקבוצות שבה בה הוא הקבוצה: ⋃ i = 1 ∞ U i
      • A יכולה להיות אין-סופית שאינה בת-מנייה, כלומר א”א לסדר את איבריה בסדרה אינסופית, ואז האיחוד של כל הקבוצות שבה בה הוא הקבוצה: ⋃ U ∈ A U
      בכל מקרה האיחוד של כל הקבוצות ב- A הוא הקבוצה: { x ∈ X ∃ U ∈ A : x ∈ U }
    ‫מסקנה 2.8. כל חיתוך של קבוצות סגורות ב- X הוא קבוצה סגורה - לא משנה אם מדובר באיחוד סופי, בן-מנייה או אפילו אין-סופי שאינו בן-מנייה.
    טענה 2.9. חיתוך סופי של קבוצות פתוחות הוא קבוצה פתוחה.
    1. ♣ זה לא נכון עבור חיתוך אין-סופי, לדוגמה (ב- R עם המטריקה האוקלידית): ⋂ i = 1 ∞ ( - 1 n , 1 n ) = { 0 }
    ‫מסקנה 2.10. איחוד סופי של קבוצות סגורות הוא קבוצה סגורה.
    טענה 2.11. תהא U ⊆ X , U היא קבוצה פתוחה אם”ם ניתן להציג אותה כאיחוד של כדורים פתוחים, כלומר קיימת קבוצה A ⊆ X כך שמתקיים: U = ⋃ a ∈ A B r a ( a ) כאשר 0 < r a ∈ R לכל a ∈ A .
    טענה 2.12. יהי ( V , ∥ ⋅ ∥ ) מרחב נורמי, לכל v ∈ V ולכל 0 < r ∈ R מתקיימים שלושת הפסוקים הבאים:
    1. B r ( v ) ¯ = B r ˆ ( v ) .
    2. ( B r ˆ ( v ) ) ○ = B r ( v ) .
    3. ∂ B r ( v ) = ∂ B r ˆ ( v ) = S r ( v ) .
    1. ♣ טענה זו אינה נכונה במרחב מטרי כללי.
    את שלוש הטענות הבאות לא ראינו בכיתה.
    טענה 2.13. תהא A ⊆ X .
    • הפנים של A ( A ○ ) הוא איחוד כל הקבוצות הפתוחות ב- X שמוכלות ב- A .
    • החוץ של A הוא איחוד כל הקבוצות הפתוחות ב- X שזרות ל- A .
    • הסגור של A ( A ¯ ) הוא חיתוך כל הקבוצות הסגורות ב- X שמכילות את A .
    טענה 2.14. לכל A ⊆ X מתקיים: A ○ ⊆ A ⊆ A ¯ ∂ A = A ¯ ∖ A ○ ( A ○ ) ○ = A ○ ∂ ( A ○ ) ⊆ ∂ A ( A ¯ ) ¯ = A ¯ ∂ ( A ¯ ) ⊆ ∂ A A ○ = A ∖ ∂ A X ∖ A ¯ = ( X ∖ A ) ○ A ¯ = A ∪ ∂ A X ∖ A ○ = ( X ∖ A ) ¯
    טענה 2.15. תהא A ⊆ X , A צפופה ב- X אם”ם A ¯ = X .
    עד כאן הטענות שלא ראינו בכיתה.

    2.3 קומפקטיות

    טענה 2.16. איחוד סופי של קבוצות קומפקטיות הוא קבוצה קומפקטית.
    טענה 2.17. כל מרחב מטרי קומפקטי הוא מרחב חסום לחלוטין.

    2.4 קשירות

    מכאן ועד סוף הפרק מופיעות טענות שלא ראינו בכיתה.
    טענה 2.18. כל תת-קבוצה של X היא איחוד זר של רכיבי הקשירות שלה.
    טענה 2.19. רכיבי הקשירות של קבוצה פתוחה ב- X הם קבוצות פתוחות.
    ‫מסקנה 2.20. כל קבוצה פתוחה ב- R k ניתנת להצגה כאיחוד סופי או בן-מנייה של קבוצות פתוחות וקשירות.
    טענה 2.21. תהא A ⊆ X קבוצה קשירה ותהא B ⊆ A , אם B פתוחה וסגורה אז B = A או ש- B = ∅ .

3 סדרות

    יהי ( X , d ) מרחב מטרי.
    משפט 3.1: כל סדרה מתכנסת היא סדרה חסומה.
    למעשה לא ראינו את המשפט הזה בכיתה אך הוא טריוויאלי.
    משפט 3.2: תהא C ⊆ X תת-קבוצה, C היא קבוצה סגורה אם”ם לכל סדרה מתכנסת ( x n ) n = 1 ∞ שכל איבריה ב- C מתקיים: lim n → ∞ x n ∈ C
    1. ♣ זו הסיבה לכך שקבוצה סגורה נקראת כך - היא סגורה לגבולות.
    ‫מסקנה 3.3. תהא A ⊆ X , הסגור של A ( A ¯ ) הוא קבוצת כל הגבולות של סדרות מתכנסות שכל איבריהן ב- A .
    משפט 3.4: משפט הירושה
    תהא ( x n ) n = 1 ∞ סדרה, אם ( x n ) n = 1 ∞ מתכנסת ל- x ∈ X אז גם כל תתי-הסדרות שלה מתכנסות ל- x ; כמו כן, אם ( x n ) n = 1 ∞ חסומה אז גם כל תתי-הסדרות שלה חסומות.
    את החלק השני של המשפט לא ראינו בכיתה, אך הוא טריוויאלי.
    טענה 3.5. תהא ( x n ) n = 1 ∞ סדרת קושי; אם יש ל- ( x n ) n = 1 ∞ גבול חלקי אז היא מתכנסת.
    הטענה האחרונה לא נלמדה בכיתה, אך היא די טריוויאלית, ואזדקק לה כדי להוכיח את השקילות בין קומפקטיות לקומפקטיות סדרתית מבלי להשתמש במושג השלמות.
    משפט 3.6: יהיו ( X 1 , d 1 ) , ( X 2 , d 2 ) … , ( X k , d k ) מרחבים מטריים, ונסמן X : = X 1 × X 2 × … × X k .
    תהא ( x n ) n = 1 ∞ סדרת נקודות ב- X , ונסמן את הקואורדינטה ה- i של x n ב- x n i (לכל n ∈ N ולכל k ≥ i ∈ N ).
    ( x n ) n = 1 ∞ מתכנסת (ב- X ) אם”ם לכל k ≥ i ∈ N הסדרה ( x n i ) n = 1 ∞ מתכנסת (ב- X i ), ובמקרה כזה מתקיים: lim n → ∞ x n = [ lim n → ∞ x n 1 lim n → ∞ x n 2 ⋮ lim n → ∞ x n k ]
    1. ♣ כזכור, הסכמנו שטענות המדברות על מרחבי מכפלה מתייחסות למטריקות d , ρ : X × X → R המוגדרות ע”י (לכל x , y ∈ X ): d ( x , y ) : ∑ i = 1 k d i ( x i , y i ) ρ ( x , y ) : = max { d i ( x i , y i ) ∣ k ≥ i ∈ N }
    את המשפט האחרון למדנו בכיתה בנוסח חלש יותר: רק עבור סדרות של נקודות ב- R k , אבל ההוכחה זהה לחלוטין לכל מרחב מכפלה עם מטריקה מהצורה הנ”ל.
    טענה 3.7. כל מרחב מטרי קומפקטי סדרתית הוא מרחב חסום לחלוטין.
    משפט 3.8: מרחב מטרי הוא קומפקטי אם”ם הוא קומפקטי סדרתית.
    1. ♣ מעתה נפסיק להשתמש במונח “קומפקטי סדרתית” ובכל מקום נכתוב “קומפקטי” בלבד.
    ‫מסקנה 3.9. מרחב מכפלה של מרחבים קומפקטיים גם הוא מרחב קומפקטי.
    משפט 3.10: כל קבוצה קומפקטית K ⊆ X היא סגורה וחסומה.
    1. ♣ הכיוון ההפוך אינו נכון בהכרח.
    משפט 3.11: קבוצה K ⊆ R m היא קבוצה קומפקטית אם”ם היא סגורה וחסומה.
    טענה 3.12. אם ( X , d ) קומפקטי אז כל קבוצה סגורה C ⊆ X היא קבוצה קומפקטית.
    טענה 3.13. נניח ש- ( X , d ) קומפקטי, ותהא ( x n ) n = 1 ∞ סדרת נקודות ב- X .
    ( x n ) n = 1 ∞ מתכנסת אם”ם יש לה גבול חלקי יחיד.
    לא למדנו את הטענה בכיתה אך היא טריוויאלית, ואזדקק לה כדי להוכיח את טענה 4.5.

4 פונקציות

    יהיו ( X , d X ) ו- ( Y , d Y ) מרחבים מטריים.
    טענה 4.1. תהא f : X → Y פונקציה המוגדרת בסביבה מנוקבת של נקודה a ∈ X ; אם הגבול lim x → a f ( x ) קיים אז f חסומה מקומית ב- x .
    משפט 4.2: אפיון היינה לגבול של פונקציה בנקודה ולרציפות
    תהא f : X → Y פונקציה.
    • L ∈ Y הוא הגבול של f בנקודה x ∈ X אם”ם לכל סדרה ( x n ) n = 1 ∞ (ב- X ) המתכנסת ל- x מתקיים: lim n → ∞ f ( x n ) = L
    • f רציפה בנקודה x ∈ X אם”ם לכל סדרה ( x n ) n = 1 ∞ (ב- X ) המתכנסת ל- x מתקיים: lim n → ∞ f ( x n ) = f ( x )
    לא ראינו בכיתה את המסקנה והמשפט הבאים.
    ‫מסקנה 4.3. תהא f : X → Y פונקציה המוגדרת בסביבה מנוקבת U של נקודה x ∈ X .
    ל- f יש גבול ב- x אם”ם לכל סדרה ( x n ) n = 1 ∞ המתכנסת ל- x שכל איבריה ב- U הסדרה ( f ( x n ) ) n = 1 ∞ היא סדרה מתכנסת.
    משפט 4.4: תנאי קושי לגבול של פונקציה בנקודה
    תהא f : X → Y פונקציה המוגדרת בסביבה מנוקבת של נקודה a ∈ X . תנאי הכרחי ומספיק לקיום הגבול lim x → a f ( a ) הוא שלכל 0 < ε ∈ R קיימת 0 < δ ∈ R כך שלכל x 1 , x 2 ∈ B δ ' ( a ) מתקיים d Y ( f ( x 1 ) , f ( x 2 ) ) < ε .
    טענה 4.5. תהא f : X → Y פונקציה רציפה, לכל קבוצה A ⊆ X גם הפונקציה f ∣ A : A → Y היא פונקציה רציפה ביחס למרחב המטרי ( A , d A ) .
    משפט 4.6: יהיו ( Y 1 , d 1 ) , ( Y 2 , d 2 ) … , ( Y k , d k ) מרחבים מטריים, נסמן Y : = Y 1 × Y 2 × … × Y n ותהא f : X → Y פונקציה.
    כמו כן לכל k ≥ i ∈ N תהא f i : X → Y i פונקציה, כך שלכל x ∈ X מתקיים: f ( x ) = [ f 1 ( x ) f 2 ( x ) ⋮ f n ( x ) ] ל- f יש גבול בנקודה a ∈ X אם”ם ל- f i יש גבול ב- a לכל k ≥ i ∈ N , ובמקרה כזה מתקיים: lim x → a f ( x ) = [ lim x → a f 1 ( x ) lim x → a f 2 ( x ) ⋮ lim x → a f k ( x ) ] כמו כן f רציפה בנקודה x ∈ X אם”ם f i רציפה ב- x לכל k ≥ i ∈ N .
    1. ♣ המשפט נובע ישירות ממשפט 3.6 ואפיון היינה.
    מסיבה זו למדנו אותו בכיתה רק עבור פונקציה מהצורה f : X → R k .
    משפט 4.7: תהא f : X → Y פונקציה, f רציפה אם”ם לכל קבוצה פתוחה U ⊆ Y הקבוצה f - 1 ( U ) גם היא פתוחה.
    משפט 4.8: משפט ההצבה בגבולות
    יהיו ( X 1 , d 1 ) , ( X 2 , d 2 ) ו- ( X 3 , d 3 ) מרחבים מטריים, תהיינה f : X 1 → X 2 ו- g : X 2 → X 3 שתי פונקציות, ותהא a ∈ X 1 נקודה.
    • נניח של- f יש גבול ב- a ונסמן l : = lim x → a f ( x ) , ונניח של- g יש גבול ב- l ונסמן m : = lim y → l g ( y ) .
      אם קיימת 0 < δ ∈ R כך שלכל a ≠ x ∈ B δ ( a ) מתקיים f ( x ) ≠ l , אז ל- g ○ f יש גבול ב- a והוא m .
    • אם f רציפה ב- a ו- g רציפה ב- f ( a ) אז g ○ f רציפה ב- a .
    בכיתה ראינו רק שהרכבה של שתי פונקציות רציפות היא רציפה, ובנוסף לא קראנו למשפט הזה “משפט ההצבה בגבולות”.
    ‫מסקנה 4.9. תהיינה ( x n ) n = 1 ∞ ו- ( y n ) n = 1 ∞ סדרות מתכנסות לגבולות x ∈ X ו- y ∈ X בהתאמה, מתקיים: lim n → ∞ d ( x n , y n ) = d ( x , y )
    ‫מסקנה 4.10. אריתמטיקה של גבולות ושל רציפות
    תהיינה f , g : X → C פונקציות המוגדרות בסביבה מנוקבת של נקודה a ∈ X , כך שהגבולות הבאים קיימים: l : = lim x → a f ( x ) , m : = lim x → a g ( x ) מתקיימים כל הפסוקים הבאים:
    1. lim x → a ( f + g ) ( x ) = l + m
      1
      פסוק זה נכון גם עבור מרחבים נורמיים שאינם שדות.
      .
    2. lim x → a ( f ⋅ g ) ( x ) = l ⋅ m .
    3. אם m ≠ 0 אז lim x → a 1 g ( x ) = 1 m .
    4. אם m ≠ 0 אז lim x → a f ( x ) g ( x ) = l m .
    כמו כן אם f ו- g רציפות ב- a אז גם f + g ו- f ⋅ g רציפות ב- a , ואם (בנוסף) g ( a ) ≠ 0 אז 1 g ו- f g רציפות ב- a .
    משפט 4.11: כלל אפסה וחסומה
    תהיינה f , g : X → C פונקציות המוגדרות בסביבה מנוקבת של נקודה a ∈ X , אם lim x → a f ( x ) = 0 וגם g חסומה מקומית ב- a אז lim x → a ( f ⋅ g ) ( x ) = 0 .
    משפט 4.12: משפט ערך הביניים
    תהא A ⊆ X קבוצה קשירה מסילתית, ותהא f : A → R פונקציה רציפה; לכל a , b ∈ A כך ש- f ( a ) ≤ f ( b ) ולכל y ∈ [ f ( a ) , f ( b ) ] קיים c ∈ A כך ש- f ( c ) = y .
    משפט 4.13: עיקרון המקסימום והמינימום של ויירשטראס
    נניח ש- ( X , d X ) קומפקטי, ותהא f : X → R פונקציה; אם f רציפה אז היא מקבלת מקסימום ומינימום (כלומר ל- Im f יש מקסימום ומינימום).
    משפט 4.14: תהא f : X → Y פונקציה רציפה, לכל קבוצה קומפקטית K ⊆ X גם f ( K ) היא קבוצה קומפקטית.
    ‫מסקנה 4.15. נניח ש- ( X , d X ) קומפקטי, ותהא f : X → Y פונקציה רציפה; כל קבוצה סגורה C ⊆ X היא קומפקטית (טענה 3.12), וגם f ( C ) היא קבוצה קומפקטית.
    משפט 4.16: משפט קנטור
    2
    ערך בוויקיפדיה: קנטור גאורג.

    נניח ש- ( X , d X ) קומפקטי, ותהא f : X → Y פונקציה; אם f רציפה אז היא גם רציפה במידה שווה.
    משפט 4.17: תנאי ליפשיץ
    3
    ערך בוויקיפדיה: ליפשיץ רודולף.

    תהא f : X → Y פונקציה; אם f רציפה ליפשיץ אז היא גם רציפה במידה שווה.
    משפט 4.18: יהיו ( V , ∥ ⋅ ∥ V ) ו- ( W , ∥ ⋅ ∥ W ) מרחבים נורמיים מעל אותו שדה ( R או C ), ותהא T : V → W העתקה ליניארית.
    אם הנורמה האופרטורית של T מוגדרת (בפרט אם V נ”ס), אז T היא פונקציה רציפה ליפשיץ, ולכן גם רציפה במידה שווה.

5 שקילויות בין מרחבים מטריים

    כל הטענות הבאות נכונות גם עבור C n ומרחבים נורמיים נוצרים סופית מעל C .
    טענה 5.1. יהי ( V , ∥ ⋅ ∥ V ) מרחב נורמי נוצר סופית מעל R ונסמן n : = dim V , קיימת נורמה ∥ ⋅ ∥ : R n × R n → R כך ש- ( V , ∥ ⋅ ∥ V ) ו- ( R n , ∥ ⋅ ∥ ) איזומטריים.
    טענה 5.2. כל הנורמות על R n שקולות זו לזו.
    ‫מסקנה 5.3. כל הנורמות על מרחב וקטורי נוצר סופית מעל R שקולות זו לזו.
    ‫מסקנה 5.4. יהי ( V , ∥ ⋅ ∥ ) מרחב נורמי נוצר סופית מעל R , קבוצה K ⊆ V היא קבוצה קומפקטית אם”ם היא סגורה וחסומה.
    טענה 5.5. יהיו ( X , d X ) ו- ( Y , d Y ) מרחבים מטריים, ותהא f : X → Y פונקציה חח”ע ועל (הפיכה); אם X קומפקטי ו- f רציפה, אז f היא הומיאומורפיזם בין X ל- Y (כלומר f - 1 רציפה גם היא).
    טענה 5.6. כל הומיאומורפיזם הוא העתקה פתוחה.
    טענה 5.7. יהיו ( X , d X ) ו- ( Y , d Y ) מרחבים מטריים הומיאומורפיים, ויהי f : X → Y הומיאומורפיזם; לכל תת-קבוצה S ⊆ X מתקיים: ( f ( S ) ) ○ = f ( S ○ ) f ( S ) ¯ = f ( S ¯ ) ∂ f ( S ) = f ( ∂ S )
    בכיתה ראינו את הטענה תוך הנחה ש- S קומפקטית, כפי שניתן לראות בהוכחה (בקובץ ההוכחות) אין בזה שום צורך.

6 שלמות

    יהי ( X , d ) מרחב מטרי.

    6.1 התחלה

    משפט 6.1: מרחב מטרי הוא קומפקטי אם”ם הוא שלם וחסום לחלוטין.
    משפט 6.2: משפט החיתוך של קנטור
    ( X , d ) הוא מרחב מטרי שלם אם”ם כל סדרת קושי ב- ( X , d ) מתכנסת.
    1. ♣ ההגדרה המקובלת של מרחב שלם היא שכל סדרת קושי שבו מתכנסת, ולכן הניסוח המקובל של המשפט הוא שהמרחב שלם אם”ם מקיים את ההגדרה שנתתי אני לשלמות.
    לא ראינו את המשפט בכיתה.

    6.2 משפט ההשלמה

    טענה 6.3. אם קיימת קבוצה A ⊆ X כך ש- A צפופה ב- X ולכל סדרת קושי ב- A
    4
    כלומר סדרת קושי ב- X שכל איבריה ב- A .
    יש גבול ב- X אז ( X , d ) הוא מרחב שלם.
    נאמר ששתי סדרות ( x n ) n = 1 ∞ , ( y n ) n = 1 ∞ שקולות זו לזו אם מתקיים: lim n → ∞ d ( x n , y n ) = 0 זהו אכן יחס שקילות, נסמן ב- X ˆ את קבוצת מחלקות השקילות של יחס זה ותהא d ˆ : X ˆ × X ˆ → R פונקציה המוגדרת ע”י (לכל [ ( x n ) n = 1 ∞ ] , [ ( y n ) n = 1 ∞ ] ∈ X ˆ ): d ˆ ( [ ( x n ) n = 1 ∞ ] , [ ( y n ) n = 1 ∞ ] ) : = lim n → ∞ d ( x n , y n )
    1. ♣ יש להוכיח ש- d ˆ אכן מוגדרת היטב, כלומר הגבול lim n → ∞ d ( x n , y n ) אכן קיים לכל שתי סדרות קושי ב- X ובנוסף הוא אינו תלוי בבחירת הנציגים של מחלקות השקילות [ ( x n ) n = 1 ∞ ] ו- [ ( y n ) n = 1 ∞ ] .
    טענה 6.4. d ˆ היא מטריקה על X ˆ ולכן ( X ˆ , d ˆ ) הוא מרחב מטרי.
    טענה 6.5. תהא f : X → X ˆ פונקציה המוגדרת ע”י f ( x ) : = [ ( x ) n = 1 ∞ ] לכל x ∈ X , כלומר f מעתיקה כל נקודה ב- X למחלקת השקילות של הסדרה הקבועה המתאימה.
    לכל x , y ∈ X מתקיים: d ( x , y ) = d ˆ ( [ ( x ) n = 1 ∞ ] , [ ( y ) n = 1 ∞ ] ) = d ˆ ( f ( x ) , f ( y ) ) ובנוסף Im f היא קבוצה צפופה ב- X ˆ .
    ‫מסקנה 6.6. ( X ˆ , d ˆ ) הוא מרחב שלם ומכאן שהוא השלמה של X .
    טענה 6.7. יהי ( Y , d Y ) מרחב מטרי שלם, תהא D ⊆ X כך ש- D צפופה ב- X ותהא f : D → Y פונקציה רציפה במידה שווה.
    קיימת פונקציה יחידה f ˜ : X → Y כך ש- f ˜ רציפה ו- f ˜ ∣ D = f .
    ‫מסקנה 6.8. לכל מרחב מטרי ( Y , d Y ) המהווה השלמה של ( X , d ) קיימת איזומטריה בין Y ל- X ˆ , כלומר השלמה של מרחב מטרי היא יחידה עד כדי איזומטריה.

    6.3 משפט ההעתקה המכווצת

    טענה 6.9. כל העתקה מכווצת היא העתקה כמעט מכווצת, וכל העתקה כמעט מכווצת היא רציפה ליפשיץ, ולכן גם רציפה במידה שווה.
    טענה 6.10. תהא f : X → X העתקה כמעט מכווצת, יש ל- f לכל היותר נקודת שבת אחת (ייתכן שאין לה בכלל נקודות שבת).
    משפט 6.11: משפט ההעתקה המכווצת
    לכל העתקה מכווצת על מרחב מטרי שלם יש נקודת שבת יחידה.
    האם גם המשפט ההפוך נכון? שאם לכל העתקה מכווצת על מרחב מטרי יש נקודת שבת אז המרחב המטרי שלם?
    משפט 6.12: לכל העתקה כמעט מכווצת על מרחב מטרי קומפקטי יש נקודת שבת יחידה.
