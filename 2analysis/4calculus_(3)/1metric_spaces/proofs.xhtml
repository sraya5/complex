1 מרחבים מטריים ומרחבים נורמיים

    1.1 מטריקה

    משפט 1.1: אי-שוויון המשולש ההפוך
    יהי ( X , d ) מרחב מטרי, לכל x , y , z ∈ X מתקיים | d ( x , z ) - d ( y , z ) | ≤ d ( x , y ) .

    1.2 נורמה

    טענה 1.2. יהיו ( V , ∥ ⋅ ∥ V ) ו- ( W , ∥ ⋅ ∥ W ) מרחבים נורמיים מעל אותו שדה, ותהא T : V → W העתקה ליניארית כך שהנורמה ∥ T ∥ op מוגדרת; לכל v ∈ V מתקיים ∥ T ( v ) ∥ W ≤ ∥ T ∥ op ⋅ ∥ v ∥ V .
    טענה 1.3. יהיו ( V , ∥ ⋅ ∥ V ) , ( W , ∥ ⋅ ∥ W ) ו- ( U , ∥ ⋅ ∥ U ) מרחבים נורמיים מעל אותו שדה, ותהיינה T 1 : V → W ו- T 2 : W → U העתקות ליניאריות כך ש- ∥ T 1 ∥ op ו- ∥ T 2 ∥ op מוגדרות; מתקיים ∥ T 2 ○ T 1 ∥ op ≤ ∥ T 2 ∥ op ⋅ ∥ T 1 ∥ op .

2 קבוצות במרחב מטרי

    יהי ( X , d ) מרחב מטרי.

    2.1 חסימות

    ‫למה 2.1. יהיו x , y ∈ X ויהיו 0 < r , s ∈ R , מתקיימים ארבעת הפסוקים הבאים:
    1. y ∈ B r ( x ) אם”ם x ∈ B r ( y ) .
    2. y ∈ B r ˆ ( x ) אם”ם x ∈ B r ˆ ( y ) .
    3. אם y ∈ B ε ( x ) לכל 0 < ε ∈ R אז x = y .
    4. אם d ( x , y ) + s ≤ r אז B s ( y ) ⊆ B r ( x ) ו- B s ( x ) ⊆ B r ( y ) .
    1. ♣ סעיפים 1 ו-2 נכונים גם עבור ספירות.
    טענה 2.2. תהא Y ⊆ X קבוצה חסומה, לכל x ∈ X קיים 0 < r ∈ R כך ש- Y ⊆ B r ( x ) .
    ‫למה 2.3. יהי x ∈ X , כל כדור פתוח שמרכזו ב- x מכיל כדור סגור שמרכזו ב- x .
    1. ♣ כמובן שגם כדור סגור מכיל כדור פתוח ( B r ( x ) ⊆ B r ˆ ( x ) ), החידוש הוא שגם כדור פתוח מכיל כדור סגור.
    ‫למה 2.4. תהא Y ⊆ X ויהיו y ∈ Y ו- 0 < r ∈ R , מתקיים: B r Y ( y ) = B r X ( y ) ∩ Y B r Y ˆ ( y ) = B r X ˆ ( y ) ∩ Y S r Y ( y ) = S r X ( y ) ∩ Y
    ‫מסקנה 2.5. תהא Y ⊆ X ותהא A ⊆ Y , A היא קבוצה פתוחה ב- Y (כלומר במרחב המטרי ( Y , d Y ) ) אם”ם קיימת קבוצה פתוחה U ⊆ X (פתוחה ב- X ) כך ש- A = U ∩ Y .
    1. ♣ מסקנה מיידית היא שהמשפט נכון גם עבור קבוצות סגורות: תהא Y ⊆ X ותהא A ⊆ Y , A היא קבוצה סגורה ב- Y (כלומר במרחב המטרי ( Y , d Y ) ) אם”ם קיימת קבוצה סגורה C ⊆ X (סגורה ב- X ) כך ש- A = C ∩ Y .
    טענה 2.6. כל כדור פתוח הוא קבוצה פתוחה, וכל כדור סגור הוא קבוצה סגורה.
    טענה 2.7. תהא A קבוצה של קבוצות פתוחות ב- X , האיחוד של כל הקבוצות ב- A הוא קבוצה פתוחה ב- X .

      • A יכולה להיות סופית ואז קיימות U 1 , U 2 , … , U r ⊆ X כך ש- A = { U 1 , U 2 , … , U r } , ואז האיחוד של כל הקבוצות שבה בה הוא הקבוצה: ⋃ i = 1 r U i
      • A יכולה להיות אין-סופית בת-מנייה, כלומר ניתן לסדר את איבריה בסדרה אינסופית: A = { U 1 , U 2 , … } , ואז האיחוד של כל הקבוצות שבה בה הוא הקבוצה: ⋃ i = 1 ∞ U i

      בכל מקרה האיחוד של כל הקבוצות ב- A הוא הקבוצה: { x ∈ X ∃ U ∈ A : x ∈ U }

    משפט 4.2: אפיון היינה לגבול של פונקציה בנקודה ולרציפות
    תהא f : X → Y פונקציה.
    • L ∈ Y הוא הגבול של f בנקודה x ∈ X אם”ם לכל סדרה ( x n ) n = 1 ∞ (ב- X ) המתכנסת ל- x מתקיים: lim n → ∞ f ( x n ) = L
    • f רציפה בנקודה x ∈ X אם”ם לכל סדרה ( x n ) n = 1 ∞ (ב- X ) המתכנסת ל- x מתקיים: lim n → ∞ f ( x n ) = f ( x )
    לא ראינו בכיתה את המסקנה והמשפט הבאים.
    ‫מסקנה 4.3. תהא f : X → Y פונקציה המוגדרת בסביבה מנוקבת U של נקודה x ∈ X .
    ל- f יש גבול ב- x אם”ם לכל סדרה ( x n ) n = 1 ∞ המתכנסת ל- x שכל איבריה ב- U הסדרה ( f ( x n ) ) n = 1 ∞ היא סדרה מתכנסת.
    משפט 4.4: תנאי קושי לגבול של פונקציה בנקודה
    תהא f : X → Y פונקציה המוגדרת בסביבה מנוקבת של נקודה a ∈ X . תנאי הכרחי ומספיק לקיום הגבול lim x → a f ( a ) הוא שלכל 0 < ε ∈ R קיימת 0 < δ ∈ R כך שלכל x 1 , x 2 ∈ B δ ' ( a ) מתקיים d Y ( f ( x 1 ) , f ( x 2 ) ) < ε .
    טענה 4.5. תהא f : X → Y פונקציה רציפה, לכל קבוצה A ⊆ X גם הפונקציה f ∣ A : A → Y היא פונקציה רציפה ביחס למרחב המטרי ( A , d A ) .
    משפט 4.6: יהיו ( Y 1 , d 1 ) , ( Y 2 , d 2 ) … , ( Y k , d k ) מרחבים מטריים, נסמן Y : = Y 1 × Y 2 × … × Y n ותהא f : X → Y פונקציה.

    משפט 4.8: משפט ההצבה בגבולות
    יהיו ( X 1 , d 1 ) , ( X 2 , d 2 ) ו- ( X 3 , d 3 ) מרחבים מטריים, תהיינה f : X 1 → X 2 ו- g : X 2 → X 3 שתי פונקציות, ותהא a ∈ X 1 נקודה.
    • נניח של- f יש גבול ב- a ונסמן l : = lim x → a f ( x ) , ונניח של- g יש גבול ב- l ונסמן m : = lim y → l g ( y ) .
      אם קיימת 0 < δ ∈ R כך שלכל a ≠ x ∈ B δ ( a ) מתקיים f ( x ) ≠ l , אז ל- g ○ f יש גבול ב- a והוא m .
    • אם f רציפה ב- a ו- g רציפה ב- f ( a ) אז g ○ f רציפה ב- a .
    בכיתה ראינו רק שהרכבה של שתי פונקציות רציפות היא רציפה, ובנוסף לא קראנו למשפט הזה “משפט ההצבה בגבולות”.
    ‫מסקנה 4.9. תהיינה ( x n ) n = 1 ∞ ו- ( y n ) n = 1 ∞ סדרות מתכנסות לגבולות x ∈ X ו- y ∈ X בהתאמה, מתקיים: lim n → ∞ d ( x n , y n ) = d ( x , y )
    ‫מסקנה 4.10. אריתמטיקה של גבולות ושל רציפות
    תהיינה f , g : X → C פונקציות המוגדרות בסביבה מנוקבת של נקודה a ∈ X , כך שהגבולות הבאים קיימים: l : = lim x → a f ( x ) , m : = lim x → a g ( x ) מתקיימים כל הפסוקים הבאים:
    1. lim x → a ( f + g ) ( x ) = l + m .
    2. lim x → a ( f ⋅ g ) ( x ) = l ⋅ m .
    3. אם m ≠ 0 אז lim x → a 1 g ( x ) = 1 m .
    4. אם m ≠ 0 אז lim x → a f ( x ) g ( x ) = l m .
    כמו כן אם f ו- g רציפות ב- a אז גם f + g ו- f ⋅ g רציפות ב- a , ואם (בנוסף) g ( a ) ≠ 0 אז 1 g ו- f g רציפות ב- a .
    משפט 4.11: כלל אפסה וחסומה
    תהיינה f , g : X → C פונקציות המוגדרות בסביבה מנוקבת של נקודה a ∈ X , אם lim x → a f ( x ) = 0 וגם g חסומה מקומית ב- a אז lim x → a ( f ⋅ g ) ( x ) = 0 .
    משפט 4.12: משפט ערך הביניים
    תהא A ⊆ X קבוצה קשירה מסילתית, ותהא f : A → R פונקציה רציפה; לכל a , b ∈ A כך ש- f ( a ) ≤ f ( b ) ולכל y ∈ [ f ( a ) , f ( b ) ] קיים c ∈ A כך ש- f ( c ) = y .
    הוכחה. יהיו a , b ∈ A כך ש- f ( a ) ≤ f ( b ) , תהא γ : [ 0 , 1 ] → A מסילה כך ש- γ ( 0 ) = a ו- γ ( 1 ) = b .
    ממשפט ההצבה בגבולות לפונקציות רציפות (המסקנה האחרונה) נובע ש- f ○ γ : [ 0 , 1 ] → R היא פונקציה רציפה, וממשפט ערך הביניים (אינפי’ 1) נובע שלכל y ∈ [ f ( γ ( 0 ) ) , f ( γ ( 1 ) ) ] = [ f ( a ) , f ( b ) ] קיים t ∈ [ 0 , 1 ] כך ש- f ( γ ( t ) ) = y , כלומר קיים c ∈ A כך ש- f ( c ) = y .
