1 טורים

    1.1 התחלה

    משפט 1.1: תנאי קושי להתכנסות טורים
    תנאי הכרחי ומספיק לכך שטור ∑ n = 1 ∞ a n יתכנס הוא שלכל 0 < ε ∈ R קיים N ∈ N כך שלכל N < n ∈ N ולכל K ∈ N מתקיים:
    | ∑ i = n + 1 n + k a n | < ε
    1. ♣ זהו מקרה פרטי של תנאי קושי להתכנסות סדרות.
    טענה 1.2. יהיו ∑ n = 1 ∞ a n ו- ∑ n = 1 ∞ b n טורים מתכנסים ויהי z ∈ C .
    1. ∑ n = 1 ∞ ( a n ± b n ) = ∑ n = 1 ∞ a n ± ∑ n = 1 ∞ b n .
    2. ∑ n = 1 ∞ ( z ⋅ a n ) = z ⋅ ∑ n = 1 ∞ a n .
    1. ♣ הטורים ∑ n = 1 ∞ ( a n + b n ) ו- ∑ n = 1 ∞ ( a n - b n ) יכולים להתכנס גם אם ∑ n = 1 ∞ a n ו- ∑ n = 1 ∞ b n אינם מתכנסים; למעשה, ע”פ סעיף 1 אם שניים מארבעת הטורים הללו מתכנסים אז גם שני האחרים מתכנסים.
    2. ♣ מסעיף 2 נובע שאם z ≠ 0 אז התכנסות הטור ∑ n = 1 ∞ ( c ⋅ a n ) גוררת את התכנסות הטור ∑ n = 1 ∞ a n .
    משפט 1.3: תהא ( a n ) n = 1 ∞ סדרה, אם הטור ∑ n = 1 ∞ a n מתכנס אז lim n → ∞ a n = 0 .
    טענה 1.4. אם הטור ∑ n = 1 ∞ a n מתכנס למספר S ∈ R אז r m = S - ∑ n = 1 m a n , ואם קיים m ∈ N כך שה- m -זנב של הטור ∑ n = 1 ∞ a n מתכנס ל- S אז ∑ n = 1 ∞ a n = S + ∑ n = 1 m a n .
    ‫מסקנה 1.5. יהי ∑ n = 1 ∞ a n טור.
    1. הטור ∑ n = 1 ∞ a n מתכנס אם”ם לכל m ∈ N ה- m -זנב שלו מתכנס.
    2. אם הטור ∑ n = 1 ∞ a n מתכנס אזי lim m → ∞ r m = 0 .
    3. שינוי, הוספה או גריעה של מספר סופי מאיברי הטור אינה משנה את עצם ההתכנסות/התבדרות שלו.

    יהי ∑ n = 1 ∞ a n טור.

    1. ♣ הסעיף השני הוא טריוויאלי, הוא אומר שישנם אינסוף ערכים של n ∈ N עבורם | a n | ≥ 1 ...

    2. אם limsup n → ∞ | a n | n > 1 אז ∑ n = 1 ∞ a n אינו מתכנס בהחלט.
    1. ♣ סעיף 1 במסקנה שקול לסעיף 1 במשפט אך בסעיף 2 הדבר אינו נכון.
    משפט 1.12: מבחן המנה של ד’אלמבר
    4
    ערך בוויקיפדיה: ד'אלמבר רון לה ז'אן.
    יהי ∑ n = 1 ∞ a n טור כך ש- | a n | > 0 ממקום מסוים ואילך.
    1. אם קיים q ∈ ( 0 , 1 ) כך שמתקיים | a n + 1 a n | ≤ q ממקום מסוים ואילך אז הטור ∑ n = 1 ∞ a n מתכנס בהחלט.
    2. אם מתקיים | a n + 1 a n | ≥ 1 ממקום מסוים ואילך אז הטור ∑ n = 1 ∞ a n אינו מתכנס בהחלט.
    1. ♣ מבחן השורש של קושי חזק יותר ממבחן המנה של ד’אלמבר שכן אם קיימים N ∈ N ו- q ∈ ( 0 , 1 ) כך שלכל N < n ∈ N מתקיים | a n + 1 a n | ≤ q אז עבור אותו N מתקיים גם | a n | ≤ | a N + 1 | ⋅ q n לכל N + 1 < n ∈ N ומכאן ש- | a n | n ≤ | a N + 1 | n ⋅ q ולכן: limsup n → ∞ | a n | n ≤ limsup n → ∞ ( | a N + 1 | n ⋅ q ) = q ⋅ lim n → ∞ | a N + 1 | n = q ⋅ 1 = q < 1
    ‫מסקנה 1.13. יהי ∑ n = 1 ∞ a n טור כך ש- ∑ n = 1 ∞ | a n | חיובי ממש (כלומר a n ≠ 0 לכל n ∈ N ).
    1. אם limsup n → ∞ | a n + 1 a n | < 1 אז ∑ n = 1 ∞ a n מתכנס בהחלט.
    2. אם liminf n → ∞ | a n + 1 a n | > 1 אז ∑ n = 1 ∞ a n אינו מתכנס בהחלט.
    1. ♣ גם כאן סעיף 1 במסקנה שקול לסעיף 1 במשפט אף בסעיף 2 הדבר אינו נכון.
    משפט 1.14: מבחן ראבה (Raabe)

    יהי ∑ n = 1 ∞ a n טור כך ש- ∑ n = 1 ∞ | a n | חיובי ממש (כלומר a n ≠ 0 לכל n ∈ N ), ותהא ( r n ) n = 1 ∞ סדרה המוגדרת ע”י (לכל n ∈ N ): r n : = n ⋅ ( 1 - | a n + 1 a n | )

    2. אם מתקיים r n > 1 ממקום מסוים ואילך אז ∑ n = 1 ∞ a n מתכנס בהחלט.
    1. ♣ ניתן הציג את המשפט קצת אחרת: את הדרישה בסעיף 2 נחליף בדרישה ש- liminf n → ∞ r n > 1 (אלו דרישות שקולות) ואת הדרישה בסעיף 2 נחליף בדרישה ש- limsup n → ∞ r n < 1 (דרישה זו נובעת מהדרישה שלעיל).
    2. ♣ מבחן ראבה הוא שכלול של מבחן המנה של ד’אלמבר: הוא יצליח בכל מקום שבו מבחן המנה מצליח

    1.4 מכפלות טורים

    טענה 1.20. יהיו ∑ n = 1 ∞ a n ו- ∑ n = 1 ∞ b n טורים המתכנסים ל- A ול- B בהתאמה, מתקיים: ∑ n = 1 ∞ ∑ k = 1 ∞ a k ⋅ b n = A ⋅ B
    משפט 1.21: משפט קושי
    יהיו ∑ n = 1 ∞ a n ו- ∑ n = 1 ∞ b n טורים מתכנסים בהחלט ויהיו A , B ∈ R כך ש- A = ∑ n = 1 ∞ a n ו- B = ∑ n = 1 ∞ b n .
    כל טור המורכב מכל המכפלות מהצורה a i ⋅ b j (עבור כל i , j ∈ N ) ללא חזרות הוא טור מתכנס בהחלט וסכומו הוא A ⋅ B .
