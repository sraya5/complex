1 טורים

    1.1 התחלה

    סימון:
    בהינתן סדרת מספרים מרוכבים ( a n ) n = 1 ∞ נקרא לסימון ∑ n = 1 ∞ a n (או באופן שקול: a 1 + a 2 + a 3 + ... ) “הטור האינסופי של ( a n ) n = 1 ∞ ”, לאיברי הסדרה קוראים במקרה כזה גם “איברי הטור” ולאיבר ה- n -י ( a n ) קוראים “האיבר הכללי של הטור”.
    ‫הגדרה 1.1. תהא ( a n ) n = 1 ∞ סדרת מספרים מרוכבים; נסמן ב- S N את ∑ n = 1 N a n (לכל N ∈ N ) ונקרא לו “הסכום החלקי ה- N -י של הטור”, הסדרה ( S N ) N = 1 ∞ תקרא “סדרת הסכומים החלקיים של הטור”.
    נאמר שהטור ∑ n = 1 ∞ a n מתכנס אם הסדרה ( S N ) N = 1 ∞ מתכנסת ובמקרה כזה נאמר שסכום הטור ∑ n = 1 ∞ a n הוא lim N → ∞ S N ונכתוב: ∑ n = 1 ∞ a n = lim N → ∞ S N = lim N → ∞ ∑ n = 1 N a n אם הטור אינו מתכנס נאמר שהוא מתבדר.
    1. ♣ בהינתן סדרה, הטור שלה לא מוכרח להיות קיים שהרי אין זה מוכרח שהגבול הנ”ל קיים.
    2. ♣ כל הטורים שנדבר עליהם בקורס זה יהיו טורים של מספרים מרוכבים.
    ‫הגדרה 1.2. בהינתן טור ∑ n = 1 ∞ a n נקרא לטור ∑ n = m + 1 ∞ a n ( m ∈ N 0 ) בשם ה- m זנב של ∑ n = 1 ∞ a n או גם ה- m -שארית של ∑ n = 1 ∞ a n , אם ה- m -זנב של טור מתכנס נסמן את סכומו ב- r m .

    1.2 התכנסות בהחלט

    ‫למה 1.3. תהא ( a n ) n = 1 ∞ סדרה, אם הטור ∑ n = 1 ∞ | a n | מתכנס אז גם הטור ∑ n = 1 ∞ a n מתכנס.
    ‫הגדרה 1.4. יהי ∑ n = 1 ∞ a n טור.
    1. אם הטור ∑ n = 1 ∞ | a n | מתכנס נאמר שהטור ∑ n = 1 ∞ a n מתכנס בהחלט.
    2. אם הטור ∑ n = 1 ∞ a n מתכנס אך הטור ∑ n = 1 ∞ | a n | אינו מתכנס, נאמר שהטור ∑ n = 1 ∞ a n מתכנס בתנאי.

    1.3 הכנסת סוגריים ושינוי סדר

    ‫הגדרה 1.5. נאמר שהטור ∑ k = 1 ∞ σ k מתקבל מהטור ∑ n = 1 ∞ a n ע”י הכנסת סוגריים אם קיימת תת-סדרה ( a n k ) k = 1 ∞ של ( a n ) n = 1 ∞ כך שלכל k ∈ N מתקיים: σ k = ∑ l = n k - 1 + 1 n k a l וזאת כאשר נסמן n 0 = 0 .
    המחשה: ∑ k = 1 ∞ σ k = ( a 1 + a 2 + ... + a n 1 ) ⏟ σ 1 + ( a n 1 + 1 + a n 1 + 2 + ... + a n 2 ) ⏟ σ 2 + ... + ( a n k - 1 + 1 + a n k - 1 + 2 + ... + a n k ) ⏟ σ k + ...

    1.4 טורים הנדסיים

    ‫הגדרה 1.6. סדרת מספרים מרוכבים ( a n ) n = 0 ∞ תיקרא סדרה הנדסית אם קיים q ∈ C כך שלכל n ∈ N 0 מתקיים a n + 1 = a n ⋅ q ; ובמקרה כזה q ייקרא מנת הסדרה, והטור ∑ n = 0 ∞ a n ייקרא טור הנדסי.

    1.5 פונקציית האקספוננט, הפונקציות הטריגונומטריות וחזקות מרוכבות

2 נגזרות

    1. ♣ בכל הסיכומים של קורס זה ובקורסים שיתבססו עליו נדבר אך ורק על פונקציות שתחום ההגדרה והטווח שלהן הם תתי-קבוצות של C (אלא אם נאמר אחרת במפורש) - פונקציות אלה תיקראנה גם פונקציות מרוכבות, כמעט תמיד יהיה מדובר בפונקציות שתחום ההגדרה שלהן יהיה קבוצה פתוחה וקשירה (קבוצה כזו תיקרא גם סתם תחום) ועל הקורא תוטל המשימה להבין מן ההקשר מתי מדובר גם בפונקציות שתחום ההגדרה שלהן אינו בהכרח כזה.
    ‫הגדרה 2.1. נגזרת של פונקציה בנקודה
    תהא f פונקציה המוגדרת בסביבה של נקודה w ∈ C , נאמר ש- f גזירה ב- w אם קיים הגבול: lim z → w f ( z ) - f ( w ) z - w ובמקרה כזה נקרא לאותו הגבול הנגזרת של f ב- w ונסמן אותו ב- f ' ( w ) .
    1. ♣ כמו בגזירות ב- R , גם כאן אנו מנסים למצוא את קירוב ליניארי טוב ל- f ב- w , אלא שליניאריות ב- C פירוש סיבוב ומתיחה/כיווץ ולא מתיחה/כיווץ בלבד.
    ‫מסקנה 2.2. תהא f פונקציה המוגדרת בסביבה של נקודה w ∈ C , f גזירה ב- w אם”ם קיים הגבול: lim h → 0 f ( w + h ) - f ( w ) h
    ‫הגדרה 2.3. אנליטיות בנקודה
    תהא f פונקציה המוגדרת בסביבה של נקודה w ∈ C , נאמר ש- f אנליטית ב- w אם f גזירה בסביבה של w .
    ‫הגדרה 2.4. אנליטיות על קבוצה
    • נאמר שפונקציה f אנליטית על קבוצה פתוחה Ω ⊆ C אם f גזירה בכל נקודה ב- Ω .
    • נאמר שפונקציה f אנליטית על קבוצה A ⊆ C (לאו דווקא פתוחה) אם קיימות קבוצה פתוחה Ω ⊆ C כך ש- A ⊆ Ω ופונקציה g : Ω → C אנליטית על Ω , ובנוסף f ( z ) = g ( z ) לכל z ∈ A .
    • נאמר שפונקציה היא אנליטית אם היא אנליטית על כל תחום הגדרתה.
    ‫מסקנה 2.5. פונקציה היא אנליטית על קבוצה פתוחה אם”ם היא גזירה בכל נקודה באותה קבוצה.
    ‫הגדרה 2.6. פונקציה שלמה
    נאמר שפונקציה f : C → C היא פונקציה שלמה אם היא אנליטית על C .
    ‫הגדרה 2.7. פונקציה קדומה
    נאמר שפונקציה מרוכבת F היא פונקציה קדומה של פונקציה f על קבוצה פתוחה Ω ⊆ C אם לכל z ∈ Ω מתקיים F ' ( z ) = f ( z ) .

3 אינטגרלים

    3.1 מסילות

    ‫הגדרה 3.1. חיבור של מסילות
    יהיו c , d ∈ R כך ש- c < d , ותהיינה γ 1 : [ a , b ] → C ו- γ 2 : [ c , d ] → C מסילות כך ש- γ 1 ( b ) = γ 2 ( c ) , החיבור של γ 1 ו- γ 2 הוא המסילה γ 1 * γ 2 : [ a , b + ( d - c ) ] → C המוגדרת ע”י (לכל t ∈ [ a , b + ( d - c ) ] ): γ 1 * γ 2 ( t ) : = { γ 1 ( t ) t ∈ [ a , b ] γ 2 ( c + ( t - b ) ) t ∈ [ b , b + ( d - c ) ]
    סימון:
    לכל z , w ∈ C נסמן ב- I ( z , w ) את המסילה I ( z , w ) : [ 0 , 1 ] → C המוגדרת ע”י (לכל t ∈ [ 0 , 1 ] ): I ( z , w ) t : = z + t ⋅ ( w - z ) חיבור מספר סופי של מסילות מצורה זו ייקרא מסילה פוליגונלית.
    1. ♣ הרעיון הוא כמובן ש- I ( z , w ) “הולכת” מ- z ל- w על הקו הישר המחבר אותם בתוך “יחידת זמן” אחת, ובזמן t היא עוברת t מהמרחק.
    ‫הגדרה 3.2. המסילה ההפוכה
    תהא γ : [ a , b ] → C מסילה, המסילה ההפוכה של γ היא המסילה γ ˜ : [ a , b ] → C המוגדרת ע”י (לכל t ∈ [ a , b ] ): γ ˜ ( t ) : = γ ( b - ( t - a ) ) = γ ( b - t + a )
    תזכורת:
    נאמר שקבוצה סופית P ⊆ [ a , b ] היא חלוקה של קטע סגור [ a , b ] אם a , b ∈ P .
    ‫הגדרה 3.3. אורך של מסילה
    תהא γ : [ a , b ] → C מסילה, האורך של γ הוא: ℓ ( γ ) : = L ( γ ) : = sup { ∑ i = 1 n | γ ( x i ) - γ ( x i - 1 ) | n ≥ i ∈ N לכל x i - 1 ≤ x i כך ש- [ a , b ] היא חלוקה של { x 0 , x 1 , … , x n } }

    ‫הגדרה 3.6. מסילה גזירה ברציפות למקוטעין (קונטור)
    תהא γ : [ a , b ] → C מסילה, נאמר ש- γ גזירה ברציפות למקוטעין אם קיימת חלוקה P : = { x 0 , x 1 , … , x n } של [ a , b ] כך שלכל n ≥ i ∈ N מתקיים x i > x i - 1 וגם:

    2. γ ' ( x i - 1 - ) = lim t → x i - 1 - γ ' ( t ) ≠ 0 .
    3. γ ' ( x i + ) = lim t → x i + γ ' ( t ) ≠ 0 .
    4. γ ' ( t ) ≠ 0 לכל t ∈ ( x i - 1 , x i ) .
    מסילה שתחום ההגדרה שלה אינו קטע סגור תיקרא גזירה ברציפות למקוטעין אם הצמצום שלה לכל תת-קטע סגור של תחום ההגדרה הוא כזה.

    ‫הגדרה 3.9. אינטגרביליות על קרן
    תהא γ מסילה ויהי a ∈ R .
    • נאמר ש- γ אינטגרבילית על הקרן [ a , ∞ ) אם הגבול lim b → ∞ ∫ a b γ ( t ) d t קיים, ובמקרה כזה נסמן: ∫ a ∞ γ ( t ) d t : = lim b → ∞ ∫ a b γ ( t ) d t
    • נאמר ש- γ אינטגרבילית על הקרן ( - ∞ , b ] אם הגבול lim a → - ∞ ∫ b b γ ( t ) d t קיים, ובמקרה כזה נסמן: ∫ - ∞ a γ ( t ) d t : = lim a → - ∞ ∫ a b γ ( t ) d t
    ‫למה 3.10. תהא γ : R → C מסילה אם קיים a ∈ R כך שקיימים האינטגרלים: ∫ a ∞ γ ( t ) d t , ∫ - ∞ a γ ( t ) d t אז לכל c ∈ R קיימים האינטגרלים: ∫ c ∞ γ ( t ) d t , ∫ - ∞ c γ ( t ) d t ומתקיים: ∫ - ∞ a γ ( t ) d t + ∫ a ∞ γ ( t ) d t = ∫ - ∞ c γ ( t ) d t + ∫ c ∞ γ ( t ) d t
    ‫הגדרה 3.11. אינטגרביליות על כל הישר
    נאמר שמסילה γ אינטגרבילית על כל הישר אם קיים a ∈ R כך שקיימים האינטגרלים: ∫ - ∞ a γ ( t ) d t , ∫ a ∞ γ ( t ) d t ובמקרה כזה נסמן: ∫ - ∞ ∞ γ ( t ) d t : = ∫ - ∞ a γ ( t ) d t + ∫ a ∞ γ ( t ) d t

      ‫הגדרה 4.1. נקודת התכנסות, תחום התכנסות ופונקציה גבולית
      תהא ( f n ) n = 1 ∞ סדרת פונקציות ונסמן ב- D את החיתוך של תחומי ההגדרה שלהן, נאמר שנקודה z 0 ∈ D היא נקודת התכנסות של ( f n ) n = 1 ∞ אם הגבול lim n → ∞ f n ( z 0 ) קיים.
      קבוצת נקודות ההתכנסות של ( f n ) n = 1 ∞ תקרא תחום ההתכנסות של ( f n ) n = 1 ∞ ונאמר גם ש- ( f n ) n = 1 ∞ מתכנסת נקודתית בקבוצה זו ובכל תת-קבוצה שלה.
      יהי D ' תחום ההתכנסות של ( f n ) n = 1 ∞ הפונקציה f : D ' → R המוגדרת ע”י (לכל x ∈ D ' ): f ( z ) : = lim n → ∞ f n ( z ) תיקרא הפונקציה הגבולית של f n .
      ‫הגדרה 4.2. התכנסות טור פונקציות
      תהא ( u n ) n = 1 ∞ סדרת פונקציות ממשיות ונסמן ב- D את החיתוך של תחומי ההגדרה שלהן.
      תהא ( S N ) N = 1 ∞ סדרת פונקציות מ- D ל- C המוגדרת ע”י (לכל N ∈ N ולכל z ∈ D ): S N ( z ) : = ∑ n = 1 N u n ( z ) נאמר שנקודה z 0 ∈ D היא נקודת התכנסות של טור הפונקציות ∑ n = 1 ∞ u n ( x ) אם הגבול lim N → ∞ S N ( z 0 ) קיים (כלומר הטור ∑ n = 1 ∞ u n ( z 0 ) מתכנס).
      קבוצת נקודות ההתכנסות של טור הפונקציות ∑ n = 1 ∞ u n ( z ) תקרא תחום ההתכנסות שלו ונאמר גם שהוא מתכנס בקבוצה זו ובכל תת-קבוצה שלה.
      יהי D ' תחום ההתכנסות של ∑ n = 1 ∞ u n ( z ) הפונקציה S : D ' → R המוגדרת ע”י (לכל z ∈ D ' ): S ( z ) : = ∑ n = 1 ∞ u n ( x ) תיקרא הפונקציה הגבולית של טור הפונקציות ∑ n = 1 ∞ u n ( x ) .
      1. ♣ נשים לב שההגדרה עבור טור של פונקציות כלולה בהגדרה של סדרת פונקציות שהרי טור הוא בסך הכל גבול של סדרה (סדרת הסכומים החלקיים).
      2. ♣ בהינתן סדרת פונקציות ( f n ) n = 1 ∞ כך ש- D הוא חיתוך תחומי ההגדרה של הפונקציות, נוכל להגדיר סדרת פונקציות ( u n ) n = 1 ∞ ע”י u n = f n - f n - 1 לכל 1 < n ∈ N ו- u 1 = f 1 ואז נקבל שלכל n ∈ N מתקיים: S N = ∑ n = 1 N u n = f N , ומכאן שגם (מדובר בשוויון פורמלי בלבד): S = ∑ n = 1 ∞ u n = lim N → ∞ S N = lim N → ∞ ∑ n = 1 N u n = lim N → ∞ f N = f . כלומר הסדרה ( f n ) n = 1 ∞ מתלכדת עם סדרת הסכומים החלקיים של הטור ∑ n = 1 ∞ u n ובפרט הם מתכנסים ומתבדרים ביחד, זוהי התאמה חח”ע ועל בין סדרות של פונקציות לטורי פונקציות השומרת על תכונת ההתכנסות.
      ‫הגדרה 4.3. התכנסות במידה שווה של סדרות וטורי פונקציות
